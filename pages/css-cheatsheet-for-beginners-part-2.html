CSS CheatSheet For Beginners Part 2 ✨ Thread 🧵 https://t.co/fdWFQ9cEdx

1. Box Properties ✅️ box-sizing: border-box | content-box margin: 2px 4px 3px 7px | 0 auto - Box Sizing ✅️ margin: 2px 4px 3px 7px | 0 auto - Margin ✅️ border-style: none | hidden | dotted dashed | solid groove | ridge | inset | outset - Border Style

✅️ border-color: #6066FF - Border Color ✅️ padding: 2px 4px 9px 7px - Padding ✅️ border-width: 10px - Border Width

2. Background ✅️ background-image: url() - Background Image ✅️ background-attachment: scroll | fixed | local | intial | inherit - Background Attachment ✅️ background-repeat: repeat-x | repeat-y | repeat | space | round no-repeat - Background Repeat

✅️ background-color: #FFF2EB - Background Color ✅️ background-position: top right I bottom left | center - Background Position

3. CSS Grid ✅️ grid-column-gap: 8px - Grid Column Gap ✅️ grid-row-gap: 8px - Grid Row Gap ✅️ grid-column: 3 / span 2 - Grid Column ✅️ grid-row: 3 span 2 - Grid Row ✅️ grid-template-columns:40px 58px auto 50px - Grid Template Columns

✅ grid-template-row: 25% 100px auto - Grid Template Rows ✅️ grid-template: "aaa" 1fr/50px 20% "bbb" auto | 100px 1fr - Grid Template ✅️ justify-items: start | stretch | end | center - Justify Items ✅️ align-items: start | end | center | stretch - Align Items

✅ justify-content: flex-start | flex-end | center | space-between | space-around space-evenly - Justify Content ✅ align-content: flex-start | flex-end | center | space-between | space-around space-evenly - Align Content

✅ grid-auto-columns: 100px | max - content | min-content - Grid Auto Columns ✅️ grid-auto-rows: 100px | max-content | min-content - Grid Auto Rows ✅️ grid-auto-flow: row | column | row - dense | column-dense - Grid Auto Flow

✅ grid-column-start: 2 | areaname | span 2 span areaname | auto - Grid Column Start ✅️ grid-column-end: 2 | span 2 | areaname | areaname | span - Grid Column End ✅️ grid-row-start: 2 | areaname | span 2 | span areaname | auto - Grid Row Start

✅ grid-row-end: 2 | areaname | span 2 | span areaname | auto - Grid Row End ✅️ justify-self: start | stretch | end | center - Justify Self ✅️ align-self: start | end | center | stretch - Align Self
