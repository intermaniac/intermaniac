You might find this useful about for() loops. A loop is a control structure. What is a control structure? A control structure allows the program flow to change within a unit of code or function. #100daysofcoding #100daysofcodingchallenge #javascript30 #javascript (A thread)ü™°‚Üì

Using conditions, a control structure can determine whether or not given statements are executed - and provide the basis for the repeated execution of a block of code. #webdev #webdeveloper #CodeNewbie #webdevelopment

With all this said, a loop is a controls structure that controls how a program flow changes, determines how a statement is executed within a unit of code by a giving a condition.

Syntax for if loop ‚Üí if ( let i = 0; i <= 10; i++){} ( let i = 0; i <= 10; i++) ‚Üí is the condition here. First i is the counter so i starts at 0. Second i is the range and condition of when the iteration should stop, i <= 10 means i should not be more than 10.

Third i is the iteration, i++ is same as i = i + 1. This means i will move once till the condition is met.

We also have the break and continue control flow in loops. Explanation below ‚Üì

The break control flow stops the iteration once the condition has been met. Example: let text = ''; for (let i = 0; i < 10; i++) { if (i === 3) { continue; } text = text + i; } console.log(text); // expected output: "012" because the iteration stopped immediately at 3.

The continue control flow breaks one iteration (in the loop) if a specified condition occurs, and continues with the next iteration in the loop.

Example: let text = ''; for (let i = 0; i < 10; i++) { if (i === 3) { continue; } text = text + i; } console.log(text); // expected output: "012456789" ( 3 is missing cause the iteration breaks 3 one i === 3 and continues to the other values).
