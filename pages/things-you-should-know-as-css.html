CSS Selectors It's the first thing you should learn in css because it helps in selection of element(s) you want to style is very important. These are the types of selectors: > Simple selectors > Combinator > Pseudo-class selector > Pseudo-element selector > Attribute selectors

CSS Box Model: It is the simplest yet most useful topic to master in CSS. It has these four parts: > Content > Padding > Border > Margin

CSS Layouts: By layout, I mean the structure of your design. In other words, how the elements are arranged. There are an infinite number of layouts, but all of them are primarily created using these three models: > Using the position property > Using flexbox > Using Grid
