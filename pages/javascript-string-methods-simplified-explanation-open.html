concat() : To connect two or more strings, use the concat() function. The returned new string contains the text of the connected strings and does not alter the original strings in any way. example: https://t.co/IZsWRmQD3p

indexOf(): The position of the first occurrence of a given value in a string is returned by the indexOf() method. Note : index starts from 0 {twitter: devjoehere} If the value to search for never happens, this method returns -1. example: here,"world" -> first occurrence is 6 https://t.co/yxUybm7gvy

lastIndexOf(): The last instance of a value supplied in a string is returned by the lastIndexOf() method. Note : index starts from 0 {twitter: devjoehere} The string is searched from beginning to end, but returns the index starting at position 0 at the beginning. example: https://t.co/InNG1M8e57

endsWith(): The endsWith() method checks whether a string contains the characters from a given string at the end. If the string ends with the characters, this method returns true; otherwise, it returns false. Syntax: string.endsWith(searchvalue, length) , length -> optional https://t.co/7g5sHd4X7u

includes(): By using the includes() method, you can check if a string contains the characters from a given string. example: https://t.co/bDj6L8FG3G

charAt(): The character at the given index in a string is returned by the charAt() function. The first character's index is 0, the second character's index is 1, and so on. example: https://t.co/GdMV1uAYqK

charCodeAt(): The charCodeAt() method returns the character's Unicode from a string at the specified index. here h is at index 0 and unicode->104 example: https://t.co/ZMq20CLZr7

fromCharCode(): Unicode values are converted into characters using the fromCharCode() function. The syntax of this static method of the String object is always String. fromCharCode() example: https://t.co/5RI3xzj8K1
