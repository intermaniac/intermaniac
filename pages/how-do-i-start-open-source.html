"How do I start open source contribution as a beginner ?" I have compiled 11 github repositories/resources/communities to answer the question. A thread ↓ https://t.co/oUHMSO2Yxh

1. First Contributions This is one of the best open source projects to get started as beginner. It has simple steps to follow and at the end you will learn about how contribution works and you would already made your first contribution. https://t.co/lXa3wlRmi2

2. Awesome for beginners This is also a good place to start as beginners It has list of languages you can choose from and then contribute according to your skills. [https://t.co/4YlYDRwkpQ](https://t.co/4YlYDRwkpQ)

3. Good first issue Good First Issue curates easy pickings from popular open-source projects, and helps you make your first contribution to open-source. [https://t.co/U2OR8ktBRw](https://t.co/U2OR8ktBRw)

4. Code Triage code triage is the place where you can find all the repositories where you can contribute. It has language filter option where you can filter by language of your choice and can get started. [https://t.co/v3k6bRYInZ](https://t.co/v3k6bRYInZ)

5. Gauger (dot)io Beginner friendly open source JavaScript projects to contribute to. You can also select other languages as well. [https://t.co/bDUl7rcxaS](https://t.co/bDUl7rcxaS)

6. Pandas Pandas is a python package which is used for data analysis in python. It also has beginner friendly issues which you can contribute to. [Issues · pandas-dev/pandas](https://t.co/B5gRiEsObo)

7. Firefox dev tools Here you can find all the good first bugs for beginner for various firefox tools and contribute to it. [https://t.co/1V1EVQqsa8)

8. VS code If you want to contribute to your favorite editor, you can contribute to VS code. It has more than 5000+ good first issues that you can contribute to [https://t.co/hrz7bGG76H first issue](https://t.co/MGAfSUiQa7)

9. EddieHub community You can join the EddieHub community and become an open source ninja. Join the community and start your journey with Eddie hub under the guidance of Eddie (@Eddiejaoude) and team. [https://t.co/XMeJ1Ghkc8](https://t.co/XMeJ1Ghkc8)

@eddiejaoude 10. Community classroom Join the comm classroom to learn about open source contribution, dev ops and much more. This community is managed by Kunal (@kunalstwt) and team and you can learn a lot of things and get your journey started with open source. https://t.co/5uUVpZm2cy

@eddiejaoude @kunalstwt 11. Free Web3 resource. This repo contains the resources to get started with Web3. Francesco(@FrancescoCiull4) has been very consistent and posts regular updates regarding Web3. You can use this repo to contribute and also join his community as well. https://t.co/6S6hZ537gr
