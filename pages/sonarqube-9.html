SonarQube 9.7 is LIVE! ✅ 🚀 Fast @JavaScript / @typescript PR, COBOL analysis ✅ @github security reporting ✅ @owasp ASVS reports🔐 ✅ #Python rules for tests, @awscloud CDK, path-sensitive bug detection 🎉 ✅ Easier SAML config, PII deletion Full thread below 🧵⬇️ https://t.co/phvYE5yh8y

✅ Faster @JavaScript / @typescript PR, COBOL analysis 🚀 Tired of waiting for long processing times whenever you make minor file changes for PR analysis? SonarQube 9.7’s analysis cache speeds the process by focusing ONLY on the changed files for faster results 🚀😎 https://t.co/sonMLeSNcF

✅ #github Security Reporting Users are now able to report on all vulnerabilities (taint and otherwise) directly in their Github Security projects! #vulnerability https://t.co/PnfsaJ2oKC

✅ @owasp ASVS reports🔐 Expanding on the topic of reporting, users can now measure their compliance against all 3 levels of the OWASP Application Security Verification Standard. This new report is both integrated within the UI and included as part of the Security Reports PDF https://t.co/uHF3cdTT6d

@owasp @awscloud ✅ Easier SAML config, PII deletion We've improved SAML configuration with the ability to test configurations and enhanced documentation of integration revolving Azure AD, Keycloak and Okta. We've also included the ability to remove PII from a user record. https://t.co/HU0gxsEL5o
