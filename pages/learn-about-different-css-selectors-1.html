Learn about different CSS Selectors? 1) Element Selectors 2) Id Selectors 3) Class Selectors 4) Universal Selectors (asterisk symbol *) 5) Group Selectors 6) Descendant Combinator Selectors 7) Child Combinator Selectors 8) Pseudo Class Selectors Thread:

Selectors are used to instructing browsers to behave in a certain way. It is important that it is used in the correct way.

The element selector is a way to select all the elements with a given tag name in a document, and apply the same styles to each element with the tag name. e.g body { background-color: green; } See the example from Khan Academy: https://t.co/2q2KqEaD9s

The id selector is a way to select only the element with the specified id, and apply styles to that element. The selector must start with a pound sign (#) and then the value of the id attribute. https://t.co/tfi5r67WVo.

"In CSS, a class selector is formatted as a period (.) character followed by the name of the class. It selects all elements with that class attribute. https://t.co/qj4XKCn6XT.

These are three simple CSS selectors and the most popular choice of developers. Remember: element selectors > id selectors > class selector

Universal Selectors - used to target all elements. * { color: red; } https://t.co/1V0FK3xT3J

Group Selector: The CSS grouping selector is used to select multiple elements and style them together. This reduces the code and extra effort to declare common styles for each element. To group selectors, each selector is separated by a space. https://t.co/XzUbAh4w60

Descendant Combinator Selector: - combines two or more selectors. <div id="first"> <div class="second"> childe of first </div> </div> Now apply CSS: #first .second { color: green; } https://t.co/i2zpLw4enz

Child Combinator Selector: - Combines two or more selectors like a descendant - It only targets immediate child. <div id="first"> <div class="second"> childe of first </div> </div> Now apply CSS #first>.second { color: green; } https://t.co/M1uBAdHALe
