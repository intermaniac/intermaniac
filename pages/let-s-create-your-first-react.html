At this point, our app looks good but there is no functionality. Clicking buttons doesn't do anything. So let's start adding functionality to the app. It should be obvious to you that we need a variable to store the count value and to show that on the screen (interface).

State - In React, the variables which are used to render the interface are either something called props or state. These two are simple JavaScript objects props are passed to components like function parameters. State are variables we declare and manage within our component.

I'll be using state var to store our count value When the state or prop of a component changes, React re-renders the component, & hence our interface is updated. So, somehow we need to tell React that we are going to use a state variable. For that React gives us useState hook

Hooks are simply functions provided by React. This useState is used to manage the state of a component. It takes one argument, the initial value for state var, and returns an array of 2 values. 1st being the state var and 2nd a function to update that state. see the codeüëá https://t.co/Ta221d1GeG
