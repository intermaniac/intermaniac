Smash MongoDB in 5 Minutes ✨ Thread 🧵 https://t.co/HBM17XtXjK

✅ MongoDB is a NO-SQL database program that uses JSON like documents with optional schemas to store data. ✅ A NoSQL database is a Key-Value database. ✅ A Collection is a grouping of MongoDB documents. - There are three ways to install and use MongoDB :

✅ Community Server ✅️ VSCode Extension ✅️ MongoDB Atlas

1. Documents - A document in MongoDB is a set of Key - Value pairs. 🔥 Every document in MongoDB has a unique value via Key "_id". 🔥 Documents have flexible and dynamic schemas. 🔥 Document schema is user defined and not fixed or static.

🔥 Documents can hold any data as long as they are valid data types in MongoDB. 🔥 Documents within a collection are related.

2. Data Types in MongoDB. ✅ BSON ✅ JSON ✅ Integer ✅️ Boolean ✅️ Double ✅️ Arrays ✅️ Object ✅️ Null ✅️ Object ✅️ Null ✅️ Timestamp ✅️ Object Id

6. Queries in MongoDB - A query is simply a request made to the database. - We can use multiple operations inside the find() method. - The various conditions that can be used are : 🔥 Eq 🔥 lt - less than 🔥 lte - less than equal 🔥 gt - greater than

🔥 gte - greater than equal 🔥 $ and 🔥 $ or Example : https://t.co/Jo5uTLJhty

7. Aggregation - Aggregate is very similar to the find command, where you can provide the criteria for your query in the form of JSON documents. - The Key element in Aggregation is called Pipeline. ✅ A Pipeline is simply a sequence of data aggregation operations.

- Valid aggregate stages include : 🔥 $ count 🔥 $ group 🔥 $ limit 🔥 $ lookup 🔥 $ match 🔥 $ merge 🔥 $ sort 🔥 $ project 🔥 $ unwind 🔥 $ unset
