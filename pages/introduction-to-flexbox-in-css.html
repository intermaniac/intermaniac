→ Flexbox is a one-dimensional layout method for arranging items in rows or columns. → It is the way to design the layout for the elements in our web pages. → With the Flexbox we can easily design and arrange elements as per our need.

➡️ Uses → We can avoid using Float and Position properties. → We can arrange and design all the elements at once in the container. → All the items adopt same height and properties to look better. → No need to styling every element which saves lot of time in designing.

→ All the child elements should be inserted inside the one-parent container. → The properties which are applied to the parent container are called " Flex container properties ". → The properties which are applied to the child elements are called " Flex item properties ".
