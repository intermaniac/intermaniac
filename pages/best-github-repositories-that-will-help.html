1. Free Programming Books It's a GitHub repo that lists over 4,000 free programming books, 2,000 free courses, and numerous additional programming resources in 43 spoken languages. https://t.co/k3w7oCIOoO

2. Project Based Learning It's a collection of programming lessons that teach aspiring software developers how to create an application from start. Tutorials involve multiple programming languages and technologies. https://t.co/eLu7HIpPhm

3. Coding Interview University It's a multiple months study plan for you to become a software engineer at a large tech company. If you want to work for companies like Facebook, Google, Amazon, etc., this is your guide. https://t.co/3adu9E5JXw

4. Build your own X It's a collection of well-written, detailed instructions that can be used to recreate our favorite technologies from scratch. https://t.co/631xOSIu63

5. Public APIs It's a free collection of APIs for use in software and web development. Over 1425 APIs are listed in 51 categories in this repository. https://t.co/tX2PmcAnFC

6. Professional Programming It's a collection of learning resources for curious software engineers. The repo contains resources in the form of books, articles, videos, etc. https://t.co/ncUuEkc5if

7. 30 seconds of code It's a collection of short JavaScript code snippets for all your development needs. https://t.co/S468R7lgWA

8. The System Design Primer System design is a broad topic and there are numerous resources scattered all over the internet on system design principles. It's an organized collection of great resources that will teach you how to build systems at scale. https://t.co/Sa4C3LDeEZ

9. Every Programmer Should Know Tech is a broad field and there is always so much to know. This repo is a collection of (mostly) technical things that you should know as a software developer. https://t.co/RZhhudUI80
