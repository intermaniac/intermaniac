HTML tags are like keywords define that how a web browser will format and display the content. With the help of tags, a web browser can distinguish between HTML content and simple content. Let's take a look at some:

1. <kbd> - It's called the keyboard input element. It is used to represent a piece of text that denotes a textual input received from a keyboard, voice input, or any other text entry device. It is used in combination with <samp> tag to represent inputs and outputs on the web.

2. <hgroup> - This tag is used to group up the heading and its related content together. It can contain any one of the heading (h1-h6) tags and a few <p> tags that act as subheadings or taglines. The tag has no impact on the document outline of a web page.

4. <summary> - This HTML tag specifies a summary for <details> element's disclosure box. Upon clicking, it toggles the state of the parent <details> element to open and closed. Its content can be any heading, text or paragraph.

5. <portal> - This tag in HTML facilitates the embedding of another HTML page inside the current page for easier and smoother navigation to new pages. It is similar to a <iframe> that allows for the embedding of a separate browsing context.

6. <video> - This tag is used to play a video file in a web application with an embedded media player. With it, adding videos to a webpage becomes as easy as adding an image because it does not require any separate plugin and is supported across all modern browsers.
