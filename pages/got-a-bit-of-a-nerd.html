Got a bit of a nerd rant today. ðŸ§µ The old timers in the Linux world can be really fuckin insufferable sometimes. Just got done watching a video about how Linux is great the way it is, doesn't need to grow market share, and shouldn't try to appeal to Windows users.

This is stupidly insular and short-sighted. Monopolies are terrible for everyone, not just the people stuck on Windows. Microsoft treats it's users as a resource to be exploited and because Linux can be so obtuse, the users have no choice but to be used.

As a long time Linux user, you might be content with the state of Linux but there are tons of Windows users that would love to switch if only the barriers to entry were more manageable. Yes, its better then it used to be, but its no where near where it NEEDS to be.

Nobody LIKES Windows. PC's are in dire need of an alternative. But because literally everyone thats used a PC in the last 30 years knows exactly how to use Windows and expects PC's to behave similarly to how they do with Windows means Linux needs to get with the program.

Make Linux that alternative. Make it easy, make it powerful, make it enough like Windows that anyone familiar with Windows can use it well. Think of all the incredible software that could come to Linux if it had 20% market share.

Just because something works well for your use-case, doesn't mean thats the best it can be, or even should be. Linux is the perfect example of that. /nerd rant over
