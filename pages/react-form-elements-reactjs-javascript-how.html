How to work with React form elements? we can make form elements interactive by setting a callback to the onChange prop. Form components listen to changes and they are fired when, value of <input> /<textarea> changes Checked state of input changes Selected state of input change

we should maintain a state for every form field updating the value in response to user interaction, the onChange prop can be used On change of the input field, the value can be fetched using https://t.co/bwFeHYYZ1w.value and the corresponding state is updated.

You can create two types of forms in React: 1Ô∏è‚É£ Controlled component 2Ô∏è‚É£ Uncontrolled component 1Ô∏è‚É£Controlled component: In controlled components the input values are set to corresponding state variables and when the value changes, the state is updated accordingly.

Uncontrolled component: In an uncontrolled component, the form elements data is handled by the DOM and not by the state. <input type="text" ref={input} /> The input does not have value attribute and an event handler, instead DOM ref is used to get the value of the form field

'ref' is used to access the React elements, and helps in getting the value of input field from the DOM. To create refs we can use useRef hook

useRef hook syntax: let countRef = useRef(0) useRef hook accepts the initial value and returns a ref object The countRef ref object returned, is mutable and will persist for the full lifetime of the component

we can assign the ref instance created using useRef hook to the input field using ref attribute, as shown below: <input type="text" ref={countRef} /> we can access the value of the input field as shown below: countRef.current.value
