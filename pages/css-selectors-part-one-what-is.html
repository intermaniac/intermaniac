CSS Selectors PART ONE What is CSS Selector? Thread üëá üëá üëá

CSS selectors are used to "control" (or select) the HTML elements you want to style. We can divide CSS selectors into five categories: 1. Simple selectors (select elements based on name, id, class)

2. Combinator selectors (select elements based on a specific relationship between them) 3. Pseudo-class selectors (select elements based on a certain state) 4. Pseudo-elements selectors (select and style a part of an element)

5. Attribute selectors (select elements based on an attribute or attribute value)

Now let's deal with Simple Selectors. In no particular order, it comprises of Element(div), Id(#) , Class(.), Universal(*) and Grouping(p, h1, h2)

1. The CSS element Selector denotes with tag name(div) The element selector is a way to select all the elements with a given tag name in a document, and apply the same styles to each element with the tag name.

2. The CSS id Selector denotes with hash(#) The class selector is a way to select a unique element with the specified identity name, and apply styles to matching element. https://t.co/WvZFacowMQ

3. The CSS class Selector denotes with dot(.) The class selector is a way to select all of the elements with the specified class name, and apply styles to each of the matching elements. https://t.co/ZXCXwkrCRp

We can also specify that only specific HTML elements should be affected by a class. https://t.co/5BimgJu5Xe

4. The CSS Universal Selector denotes with asterisk (*) The universal selector provided by CSS helps in choosing all elements within the HTML page. It goes with a single element and uses the asterisk (i.e., "") symbol used for denoting the selector as a universal selector. https://t.co/2oJYxJkkbF

5. The CSS Grouping Selector denotes with comma(,) The CSS grouping selector is used to select multiple elements and style them together. This reduces the code and extra effort to declare common styles for each element. https://t.co/8A3wYGQjdn
