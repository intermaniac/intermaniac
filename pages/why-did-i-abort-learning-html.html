Why did I abort learning HTML, CSS and JavaScript ‚ùì üßµ üëá

So let's jump back to 2020 when COVID-19 was at its peak. I used to play a lot of games at that time. In one of the online game, I got a friend who told me that he was making a game same as the one we were playing. At that time, I thought why don't I? So I learnt a game engine.

Most of the programmers nowadays start their journey with HTML and CSS. But for me it was a game engine. Making UI in a game engine is not a huge task and it just requires some drags and drops.

Then I met @MaxProgramming1 who inspired me very much for web development and I started learning HTML and CSS. I knew some HTML before but CSS was an alien to me. I tried to learn a lot but websites didn't came up as I expected them to be.

Here I discovered flutter. Flutter is a framework to create frontends for mobile and desktop apps. I purchased a course for it and practised it very much. I created many projects and that's how I kinda mastered flutter. React native was again a choice but its styling was like css

As time passed, I became less interested in HTML and CSS and more interested in app development. Then I bought a MacBook at this point. I downloaded Xcode and started SwiftUI. And I loved it. I didn't learn it and made a project. It is very easy for flutter developers

That's how I left web development and started app development. I hope you liked my story (if you are an app dev üòÖüòÇ). Make sure to like and share !
