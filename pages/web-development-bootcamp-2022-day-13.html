#Web_Development_Bootcamp_2022 #Day-13 of learning: Thread 🧵 . 𝐂𝐒𝐒 𝐬𝐞𝐥𝐞𝐜𝐭𝐨𝐫𝐬: CSS selectors define the elements to which a set of CSS rules apply. CSS selectors are:

𝟏. 𝐔𝐧𝐢𝐯𝐞𝐫𝐬𝐚𝐥 𝐬𝐞𝐥𝐞𝐜𝐭𝐨𝐫 (*): It is used to select everything in the document like make everything black, make everything a certain font size, etc. Exp: *{ Color:black; } https://t.co/wc4J1KnukG

𝟐. 𝐄𝐥𝐞𝐦𝐞𝐧𝐭 𝐬𝐞𝐥𝐞𝐜𝐭𝐨𝐫: It selects an element like we want to select images or buttons, it selects all the images or all the buttons and selects what we want to. Exp: img{ Width: 10px } https://t.co/ZiPCRmNhTo

𝟑. 𝐒𝐞𝐥𝐞𝐜𝐭𝐨𝐫 𝐥𝐢𝐬𝐭 (,): By selector list, we can select or combine multiple selectors using comma (,). Exp: it selects all h1 and all h2 h1 , h2 { Font-size: 10px; } https://t.co/A31F92kvCc

𝟒. 𝐈𝐃 𝐬𝐞𝐥𝐞𝐜𝐭𝐨𝐫 (#): Add the “Id” attribute to any element in HTML to which you want to change then use this id in CSS to select that element. It applies to a single element. Exp: In HTML <button id=”signup”>signup</button> In CSS: #signup{ Color: black; } https://t.co/02bFxQ6imK

𝟓. 𝐂𝐥𝐚𝐬𝐬 𝐬𝐞𝐥𝐞𝐜𝐭𝐨𝐫 (.): Same as id but it applies to multiple elements. Add “the class” attribute to all elements in HTML to which you want to change then use this class name with a dot(.) in CSS to select these elements.

Exp: In HTML: <button class=”signup”>signup</button> In CSS: .signup{ Color: black; } 𝐓𝐨 𝐛𝐞 𝐜𝐨𝐧𝐭𝐢𝐧𝐮𝐞𝐝... #learning #development #css #html #javascript #webdesign #webdevelopment #frontend #backend #webdeveloper #internet #frontenddeveloper #backenddeveloper https://t.co/cb4LR8fjyI
