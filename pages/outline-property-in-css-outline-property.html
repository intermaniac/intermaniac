Outline property in CSS 🚀 Thread 🧵

Outline property • used to draw a line around the elements or outside of the borders of elements • outline is not a part of elements dimensions • there are four different categories for outline properties : ① outline-style ② outline-width ③ outline-color ④ outline-offset

Let's discuss each one by one 👇 ① outline-style • outline-style property declared the style of the outline • outline-style values : auto, none, dotted, dashed, solid, double, groove, hidden, inset, ridge etc. Example 👇 https://t.co/MVqlZ1lXyb

② outline-width • outline-width property declared the thickness of the outline • outline-width values : thin, medium, thick and a specific size etc. Example 👇 https://t.co/ouCFTNzESS

③ outline-color • outline-color property declared the color of the outline • outline-color values : name, hex, rgba, hsl etc. Example 👇 https://t.co/ROs6rBQCUC

④ outline-offset • used to adds space between border or edge and outline of an elements • space between an element and its outline is transparent • negative value places the outline inside the element Example 👇 https://t.co/6OhPFraZpd

Outline shorthand • you can set only style(required), width and color • unable to set outline-offset value • syntax : outline : width style color; Example 👇 https://t.co/flCrp1M4QH

Difference between outline and border property 👇 • Outline elements are not part of dimensions. However, they are part of the border element. • Outline has no effect on element Size. However, border affects the element Size. • Outline cannot be rounded. End thread 🧵
