So programming languages are three types:---- 1. Procedural 2. Functional 3. Object-Oriented

Before going to any types of programming languages we need to know about what is "what to do & how to do" in programming world

------:WHAT TO DO:------ In programming world when programming language mainly focus on computation task result instead of operation of getting computation task result which is known as what to do

-----:HOW TO DO:----- In programming world when programming language focus on operation of getting any computation task which is known as how to do

Now come on first type of programming language that is procedural language

-------:PROCEDURAL LANGUAGE:----------- Procedural languages follow procedure or steps for performing task. This type programming languages focus on what to do as well as how to do.

Means in other word it focus on main task as well as operation of achieving to those task. Procedural languages follow well-structured steps and procedures for achieving task. It contains a systematic order of statements functions and commands to complete a task. EX.- Java,C++

-------:FUNCTIONAL LANGUAGE:-------- Functional languages or non-procedural languages do not follow well-structured steps for performing any task

Functional languages mainly focus on what to do instead of how to do it, which means functional language focus on task, not operation.

Funtional languages used in a situation where we have to perform lots of different operations on the same set of data, like ML

-----:OBJECT ORIENTED:------- Object-oriented languages perform any computation task based on objects,it revolves around objects. Here ,object means, in programming, it is just code with data. Data in the form of attributes and code in the form of procedures EX.- JAVA

It makes it easier to develop, debug, reuse and maintain software
