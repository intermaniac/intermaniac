⛏️️ SQL is Most IMP and inevitable skill in all Data Roles, for both interviews & on the job responsibility. 👉 Here's the list of 10 websites where you can practice SQL concepts and also SQL Problems asked by Top Companies (including FAANG) 🧵 #SQL #data #interview #faang

✅ @StrataScratch (https://t.co/nvJVGuvWsx) 🔶 Great platform to practice SQL problems asked in top companies. They also have guide videos and discussion section. 🔷 Also, has YouTube channel (StrataScratch) where they put SQL tips for interview and go through solutions.

🔶 You can also practice Non-coding problems like System Design, Business Case, Modeling on StrataScratch. 🔷 It also provides take-home assignment Data Projects to practice for your next interview.

✅ @DataLemurHQ (https://t.co/GMges92ilE) 🔶 Great platform developed by @NickSinghTech , provides rich interface to solve SQL problems. 🔷 @DataLemurHQ provides 50+ SQL questions for FREE to practice asked in Top companies to ace your SQL interview.

✅ @LeetCode 🔶 Leetcode provides wide range of SQL problems to practice with rich interface, including discussion section. 🔷 Leetcode is great platform to go through all kind of (coding, SQL, System Design) interview questions asked in Top companies.

✅ @ConfettiAi (https://t.co/uYfOwEawoP) 🔶 Confetti AI is a leetcode alternative to practice and crack your Data Science interviews. 🔷 Specifically on SQL, it provides problems on different SQL concepts, even optimizations problems.

✅ @hackerrank 🔶 HackerRank is similar to leetcode platform to practice interview questions asked in companies. 🔷 Also allows you to narrow the challenges according to the difficulty level, subject areas and subdomains.

✅ @interview_query (https://t.co/hxxzIKKDoi) 🔶 Interview Query platform provides rich sets of SQL questions asked in Data Science interviews by Top Companies. 🔷 It also provides challenges, take-homes assignments. I can't forget learning paths curated for different topics.

🔶 Great Community support like Slack, Blogs and articles is cherry on top of the cake. 😅 🔷 Wait, Did I mention about Mentorship ? 🔶 Super useful platform to start your Data Science Interview Prep.

✅ SQLZoo (https://t.co/IldTSRQPlX) 🔶 SQL Zoo provides ready to query datasets with questions on all the IMPORTANT SQL concepts. 🔷 Super helpful platform for beginners to practice SQL concepts after learning theory. 🔶 Best thing is SQL Zoo is all FREE to use.

✅ @SQLBolt (https://t.co/n6Id0KFBHb) 🔶 I included SQL Bolt in this list because it not only covers SQL concepts but right after explanation, you will have Dataset and Query Editor to practice those concepts. 🔷 If I'm a beginner in SQL, I would definitely try SQL Bolt.

✅ SQL-Easy (https://t.co/uDguitdSKP) 🔶 Learned SQL basics, but don't have database, tables (with data) set up? Well, SQL-Easy is there for your rescue. 🔷 Select the CONCEPT you want to practice on, and it will give you sample dataset, query editor & questions to solve.

✅ SQLTest (https://t.co/q5nTdMRkaa) 🔶 SQL Test is great tool to practice SQL with respect to different databases like, @MySQL , @OracleDatabase , @SQLServer etc. 🔷 It also populate create, insert queries for you in query editor. Just awesome, ready to query tool, for FREE.
