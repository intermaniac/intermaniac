ðŸ§µ The unknown #HTML Element: <base /> #1 Definition from w3: The base element allows authors to specify the document base URL for the purposes of resolving relative URLs, and the name of the default browsing context for the purposes of following hyperlinks

#2 "The base element allows authors to specify the document base URL for the purposes of resolving relative URLs" - It defines the base URL for relative links on the website. - Once it is defined, all the links on your page, will use the href attribute of <base /> as base URL.

#3 Let's look at an example: The link shown in this image will redirect the user to https://t.co/Hxn4oCqgoR, instead of https://t.co/fNg50Khnt5 Notice how even if we are using `/` in the link's href, it is NOT using the domain as the base URL. https://t.co/b4ATqkx18n

#4 "It allows the authors to define the default browsing context for the purposes of following hyperlinks" - It defines the default browser context for all the links on your page - Browser Context means where to open links on your website (Ex. Open a link on new page)

#5 The <a> tag in the example will open the link into a new tab. Notice that we didn't define the target attribute on the <a> tag, but it still opens it in a new tab. We are noticing this because all the <a> tags inherit `target` attributes from <base /> element (if defined) https://t.co/KCGWJPOLyI

#6.1 REAL-WORLD USE-CASES: - If you have version-based URLs (https://t.co/tgPf9MzTcU), you can define the <base> element in your code to easily define base URLs for all of your links to use the latest version URLs (Cont.)

#6.2 - If most of your links open in a new tab (ex links on Hackernews), you can use this element to define the default context for links, instead of adding `target` to all links. - In the past, it was used on frameset pages to make links open in a particular frame.

#7 RESTRICTIONS: - There can be only one base element - It's a self-closing - It must come before other elements with URL attributes like <link> - Open Graph tags don't follow <base /> - If multiple <base> are used, only the first href and first target are obeyed

#8 OTHER INFO: - Permitted parents: A <head> that doesn't contain another <base> element - A document's used base URL can be accessed by scripts with Node.baseURI - Supported by all the browsers - DOM Interface (shown in the image) - Supported attributes: `href`, `target` https://t.co/miU30iIvbH
