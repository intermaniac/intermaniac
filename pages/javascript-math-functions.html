#JavaScript Math functions. A thread üëá #100DaysOfCode

#JavaScript provides several built-in Math functions that allow you to perform mathematical operations on numbers. Here is a list of those Math functions in #javascript,

1. Math.abs: This function returns the absolute value of a number. The absolute value of a number is the number without its sign (positive or negative). For example: console.log(Math.abs(-5)); // 5 console.log(Math.abs(5)); // 5 #100DaysOfCode

2. Math.ceil: This function rounds a number up to the nearest integer. For example: console.log(Math.ceil(5.2)); // 6 console.log(Math.ceil(-5.2)); // -5 #ReactJS

3. Math.floor: This function rounds a number down to the nearest integer. For example: console.log(Math.floor(5.8)); // 5 console.log(Math.floor(-5.8)); // -6 #NodeJS

4. Math.max: This function returns the maximum of two or more numbers. For example: console.log(Math.max(1, 2, 3, 4, 5)); // 5 console.log(Math.max(-1, -2, -3, -4, -5)); // -1 #Angular

5. Math.min: This function returns the minimum of two or more numbers. For example: console.log(Math.min(1, 2, 3, 4, 5)); // 1 console.log(Math.min(-1, -2, -3, -4, -5)); // -5 #JavaScript

6. Math.pow: This function returns the result of a number raised to a specified power. The function takes two arguments: the base and the exponent. For example: console.log(Math.pow(2, 3)); // 8 #100daysofcodechallenge

7. Math.random: This function returns a random number between 0 (inclusive) and 1 (exclusive). The Math.random function does not take any arguments. For example: console.log(Math.random()); // a random number between 0 and 1 (exclusive) #javascript

8. Math.trunc: This function returns the integer part of a number by removing any fractional digits. The Math.trunc function takes a single argument: the number to be truncated. For example: console.log(Math.trunc(5.6)); // 5 console.log(Math.trunc(-5.6)); // -5 #nodejs #js
