1. <video> - It's used to play a video file in a web application with an embedded media player. Only MP4, WebM, and OGG files are supported. Audio files can also be played.

2. <defs> - It's used to store the definitions of the graphical objects for further usage inside an SVG image. It helps to group the SVG shapes together, allowing them to be reused as per requirements. It improves the page's accessibility and readability of the code.

3. <samp> - It's used to store sample output from any program. The browsers render the content of this tag with a monospaced font like Courier. Its main purpose is to show steps or output to a user. It accepts only global attributes and is supported by all modern browsers.

4. <colgroup> - You can group one or more similar sorts of columns using this. It may contain <col> as a child element. It has a span attribute which can take a number value. It is allowed to be set if there is no content inside.

5. <mark> - It's used to highlight specific text in an HTML document that is of particular interest or relevance. If the purpose is just highlighting some text which has no importance, it is recommended to use <span> with appropriate CSS.

6. <var> - It represents the name of the variable that is defined in a mathematical or programming context, and accepts all global attributes. The text or content enclosed inside it is rendered in italics. Tags such as <code>, <kbd> and <samp> can also be used.
