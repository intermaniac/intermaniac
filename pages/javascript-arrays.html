Javascript Arrays. A thread. 1. In #javascript, an array is a data type that represents a list of values. Arrays are ordered collections of values, and each value in an array is called an element. #100DaysOfCode

2. You can create an array in #JavaScript using the Array constructor or by enclosing a list of values in square brackets. For example: // Using the Array constructor let arr1 = new Array(1, 2, 3, 4, 5); // Using square brackets let arr2 = [1, 2, 3, 4, 5]; #100daysofcoding

3. Each element in an array has a corresponding index, which is a numerical value that starts from 0 and increases by 1 for each element. For Eg., in the array [1, 2, 3, 4, 5], the elements have the following indices: Index Element 0 1 1 2 2 3 3 4 4 5

4. You can access the elements of an array using their indices. For example: let arr = [1, 2, 3, 4, 5]; console.log(arr[0]); // 1 console.log(arr[1]); // 2 console.log(arr[2]); // 3 #NodeJS #Nodejs

5. #JS arrays are dynamic, which means that you can add/ remove elements as you wish. The push method adds elements to the end of an array and the pop method removes the last element. For Eg: let arr = [1, 2, 3]; arr.push(4); // [1, 2, 3, 4] arr.pop(); // [1, 2, 3] #programming

6. You can also use the shift method to remove the first element of an array, and the unshift method to add an element to the beginning of an array. For example: let arr = [1, 2, 3]; arr.unshift(0); // [0, 1, 2, 3] arr.shift(); // [1, 2, 3] #programmingmemes #CodeNewbie

7. You can edit the array values directly as well. For Example: let arr = [2, 2, 3]; arr[0] = 1; // [1, 2, 3] #reactnative #ReactJS
