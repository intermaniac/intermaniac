The course has 16 branches which I'd like to keep a clean history from. 01-connect-to-neo4j branches from main and contains the code for adding the driver - 02-movie-lists branches from 01-connect-to-neo4j and contains the code which pulls paginated movie lists from neo4j etc https://t.co/GKk1zto909

The Unique Email Addresses lesson on branch 04 modifies the changes made in branch 03 by moving the code into a try/catch block which seems to cause merge conflicts in every subsequent branch when trying to rebase on the previous branch https://t.co/x3eAU4YWMb

This is where rerere comes in, after enabling with: $ git config --global rerere.enabled true Git will replay the previous conflict resolution for every merge.
