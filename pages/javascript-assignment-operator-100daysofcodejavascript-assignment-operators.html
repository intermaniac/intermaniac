JavaScript assignment operators - JavaScript assignment operator assigns value to its left operand based on the value of its right operand.

- There are 7 basic types of assignment operators: Assignment ( = ) Addition Assignment ( += ) Subtraction Assignment ( -= ) Multiplication Assignment ( *= ) Division Assignment ( /= ) Remainder Assignment ( %= ) Exponential Assignment ( **= )

1. Assignment ( = ): - The simple assignment operator or the assignment operator assigns a value to a variable. https://t.co/PvKIHAcCZw

2. Addition Assignment ( += ): - The addition assignment operator adds the value of the left and right operand then assigns the sum to the left operand. - It can also be used to concatenate or add strings together. https://t.co/GNxxZjfmvO

3. Subtraction Assignment ( -= ): - The subtraction assignment operator subtracts the value of the right operand from the left operand and then assigns the difference to the variable. https://t.co/s1TcxQ21FC

4. Multiplication Assignment ( *= ): - The multiplication assignment operator multiplies the value of the left and right operand then assigns the product to the variable. https://t.co/BGoKHBaH0E

5. Division Assignment ( /= ): - The division assignment operator divides a variable by the value of the left operand by the right operand and then assigns the quotient to the variable. https://t.co/1mbKjSAh7M

6. Remainder Assignment ( %= ): - The Remainder Assignment Operator assigns a remainder to a variable. https://t.co/9PhTmTDCUu

7. Exponential Assignment ( **= ): - The exponentiation assignment operator raises a variable to the power of the operand. https://t.co/Ht17HzeF46
