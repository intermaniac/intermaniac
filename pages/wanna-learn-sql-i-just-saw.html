Wanna learn SQL? I just saw a tweet about the top *15* things you need to know. That's too much. Here are the only 3 you need...ðŸ§µ

* Select/Field * From/Table * Where Let's go through with a products example

* Select/Field This is how you kick off your sql query. You tell your query to *select* (literally go and get something) a field. Think of a *field* as a column in an excel spreadsheet Here I select 2 fields I'm interested in ----- select product_type, product_name -----

* From/Table Thing of a SQL table as a single sheet/tab on an excel spreadsheet. You can cycle through them and see different data. Same thing in SQL. From = where you want to select Here I tell my sql which table to pull from ----- from my_products_table -----

* Where Where = What you want to filter for. In plain english this reads, "where this thing is true, return the rows for me" So if I only wanted the rows *where* my field > 2 I would write "where field > 2" Here I tell my sql which rows to return. --- where field > 2 ---

Let's put it all together with a product example select product_type, product_name from my_products_table where product_type = "saas" Enjoy and good luck. If you want to try this yourself, head over to sql-practice .com
