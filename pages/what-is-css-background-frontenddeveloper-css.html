The background attributes are 1. background-color 2. background-image 3. background-repeat 4. background-attachment 5. background-position

Background Color The background-color attribute is what used to change an element's background color. https://t.co/FhoACxrAav

The following formats are most frequently used to specify color values in CSS: a color name - like "blue" a HEX value - like "#0000ff" an RGB value - like "rgb(0, 0, 255)"

Background Image The background-image attribute causes us to use an image as an HTML element's background. https://t.co/hpBYHVMnJY

Take note that we should make sure the background image you chose for an element doesn't interfere with the text's readability before applying it.

Background Repeat The background-repeat attribute permits us to decide how an element's background is repeated or tiled. A background image can be configured to repeat either vertically (y-axis), horizontally (x-axis), both directions, or neither.

The background-image can be repeated horizontally along the x-axis by using the value repeat-x. https://t.co/1nNtTdGTI4

The background-image can also be repeated vertically along the y-axis by using the value repeat-y. https://t.co/oxzDERJ2wO

The background-image been repeated can be avoided entirely by using the value no-repeat. https://t.co/Xb1XASyndm

Background Position The background-position property is employed to modify the background image's position. https://t.co/KH0mLxiZKQ

The background picture is positioned at the element's default top-left position if no background position has been given, which is at (0,0) with the above we can notice that, so let's see if we modify it to top-right.

Please note that, the first value indicates the horizontal position, and the second value represents the vertical position, if the background-position property is given with two values. When just one value is given, it is presumed that the second value is center.

In addition to keywords, you can also use percentage or length values for this attribute, such px or em.

Background Attachment The background-attachment attribute controls whether the background image scrolls with the contained block or is fixed with respect to the viewport. https://t.co/FS6WOZXMnE

The Background Shorthand Property There are various properties to take into account while working with the backgrounds, as we can see in the samples above. To cut down on typing or reduce the code, it is also feasible to declare all of these properties in a single property.

An example of a shorthand property is this. The background property is a shorthand property for setting all the individual background properties, i.e., background-color background-image background-repeat background-attachment background-position https://t.co/HOxu2xlFri

When utilizing the shorthand notation, if a background property's value is absent or not supplied, the property's default value, if any, will be used in its place.

The background properties do not inherit like the color property, but the parent element's background will be visible by default due to the background-color CSS property's initial or default transparent value.

By default, a browser fills the full area of an element with a background picture that is repeated or tiled both horizontally and vertically. With the background-repeat property, you may manage this.
