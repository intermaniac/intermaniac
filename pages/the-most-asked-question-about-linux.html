This article will solve all your confusion regarding choosing distros as a beginner. Let's start with what Linux Distributions are, Often shortened to "Linux distro" -- a version of the open-source Linux operating system that is packaged with other components.

There are 4 major Distributions of Linux · Red Hat · Slackware · Debian · Arch

Linux Distributions can also be classified as distros with Desktop environments & non-desktop Environments. The non-desktop ones are usually CLI-based and Light Weight but POWERFUL.

Distro with Desktop Environment: · Ubuntu · Debian · Fedora · Linux Mint

Distro with non-Desktop Environment: · Ubuntu Server · Debian Server · Arch · Alpine

👉 Slackware is a favorite among Advance Linux Users and has pkgtools Package Management System. 👉 Debian used APT (Advance Package Tools) and Ubuntu, Linux Mint & Kali are part of Debian Linux.

👉 Red Hat Linux is the commercial Distribution that uses RPM (Red hat Package Manager). 👉 Arch Linux is the Light Weight and Simple Distribution of Linux which uses Packman Package Manager.

Here is the bottom line. Which Distro you should start with, well if you are in an organization where the team members are using Red Hat Enterprise Linux the best option is to start with that.

However, if you are an individual Ubuntu is the safest option to start with. It doesn't matter which distro you choose but you should start with one and later once you feel comfortable you can explore other distros too.
