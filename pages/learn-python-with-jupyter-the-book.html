Learn Python With Jupyter, the book I am writing, is now open and completely free! You can find it at https://t.co/rLhaULdaFv. Here are its main characteristics (a thread): 1. The code of each chapter is in a Jupyter Notebook, which you can download to play with the code

2. Each chapter contains both theory and coding exercises. The solutions are available in a GitHub repository (find the link in the book) 3. The progression of topics is meant to develop critical thinking. Like when we learn a foreign language, we do not learn first all the verbs

then all the adjectives, etc, here we do not learn first all the datatypes, then the loops, etc. But we progress from commands with concrete names (e.g. .remove()) to more and more abstract thinking

4. For each line of code, there is a suggestions on how to pronounce it. Although code pronunciation is not standardized, we pronounce code continuously in our minds or while working with others. In addition, it is important not to pronounce the symbol = as "equal" because

it's not an equal! (The proposed pronunciation is "is assigned") 5. The "in more depth" sections contain trick on how to code or approach coding. The goal is to remove frustration in new coders. Learning coding does not need to be frustrating, neither fake fun. It's honest

learning, and anybody can do it 6. The book will contain 38 chapters in total, and so far I have published 15. Right now, I am able to publish a chapter every 4-6 weeks (I write the book in the morning on the train while going to work)

7. The content of the book is the material I developed while teaching Python privately for 2 years to student of any age and any background. They all happily code now, and according to them, it works!

Feel free to retweet if you find my book useful. I am putting my heart in it, and I really hope it will help more and more people learn coding #Python #JupyterNotebook #learningcoding
