What is JavaScript?🤔 🧵 👇 https://t.co/1GPKGFbiNL

🟣 Simple Summary JavaScript is the Programming Language for the Web. JavaScript can update and change both HTML and CSS. JavaScript can calculate, manipulate and validate data.

🟣 Why it is called JavaScript? JavaScript was initial name is "LiveScript" but at that time Java is very popular that's why it was named Javascript a younger brother of Java. But as it evolved javascript became fully independent with its own spacefication called ECMAScript.

Today Javascript not only runs in the browser but it runs on the server as well or on any device that has a special program a javascript engine.

🟣 What is Javascript Engine? A JavaScript engine is a software component that executes JavaScript code. It sounds like a complicated thing but it is quite easy to understand that is how the Javascript engine works. 👇

🟣 How do engines work? Engines are complicated. But the basics are easy. • The engine (embedded if it’s a browser) reads (“parses”) the script. • Then it converts (“compiles”) the script to machine code. • And then the machine code runs, pretty fast.

🟣 What can Javascript do? Modern JavaScript is a “safe” programming language. It does not provide low-level access to memory or the CPU, because it was initially created for browsers that do not require it. Add new HTML to the page, change the existing content, and modify it.

• Add new HTML to the page, change the existing content, and modify styles. • React to user actions, run on mouse clicks, pointer movements, and key presses. • Send requests over the network to remote servers, download and upload files (so-called AJAXand COMET technologies).

• Get and set cookies, ask questions to the visitor, and show messages. • Remember the data on the client-side (“local storage”).

🟣 What Makes Javascript Unique? 🤩 Some Awesome things about javascript are • Full integration with HTML/CSS. • Simple things are done simply. • Supported by all major browsers and enabled by default.

🟣 What are some Languages built over Javascript? The syntax of JavaScript does not suit everyone’s needs. Different people want different features. Recently a number of Languages appeared in the industry which were Compiled into javascript before running in a browser.

🟣 Some Examples of Languages built over javascript are: CoffeeScript TypeScript Flow Dart Kotlin Brython
