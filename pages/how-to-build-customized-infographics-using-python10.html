10. THE CODE The first thing we need to do is import the Python libraries that we will use import pandas as pd import matplotlib.pyplot as plt We will create Pandas dataframes from the OWID data and use both Pandas and Matplotlib to create the charts and other visuals

11. GET THE DATA The first dataset that we will use includes various climate indicators for a number of countries and regions (entities) We will be using the CO2 Concentrations data for the World as a whole so we will filter the data accordingly

12. First, we read the data into the dataframe co2 and this will, of course, contain all of the data Next, we filter the data by only considering it where the Entity name is ‘World’ and store it in the dataframe co2world

13. # original source: # https://t.co/7KVEGz7lHK co2 = https://t.co/kUBTx4lElU_csv('data/climate-change.csv') co2world = co2[co2['Entity']=='World'] co2world https://t.co/APhTNAaG8B

14. There are several other columns in the data but we are only interested in the CO2 concentrations data We can plot this data like this: co2world.plot(y = "CO2 concentrations", x = 'Year', grid = True, figsize = (20,10)) https://t.co/17JUrJOSpW

15. The result is a fairly shocking graph that shows that, for the last 800,000 years, the CO2 concentrations in the Earth’s atmosphere have been below 300 ppm (parts per million) but that over the last few decades it has shot up to over 400 ppm

16. However, unless you are familiar with the data, it’s not very easy to interpret the graph So let’s do our readers a favor and make it a little more obvious what we are looking at

17. The x-axis represents time but most of the years are negative values These values are relative to the beginning of the Common Era and so, for example, -800,000 means 800,000 BC, 0 is where we start counting centuries in our current system and the final year is 2022

18. We’ll set the ticks and labels so that it is more readable and, because the labels are quite long, we’ll rotate them so they don’t overlap

19. plt.xticks([-800000, -600000, -400000, -200000, 0], ['800,000 BCE', '600,000 BCE', '400,000 BCE', '200,000 BCE','0'], rotation=20)

20. We also define the x- and y-axis labels remove the legend (it’s redundant) and add a title plt.ylabel("carbon dioxide level (parts per million)") plt.xlabel("years before today (0 = 1950)") plt.legend("") plt.title("CO2 Concentrations")

21. Then we save the resulting figure as an image plt.savefig('images/co2world.png')
