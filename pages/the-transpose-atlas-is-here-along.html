0/ The Transpose Atlas is here, along with some other SQL Playground updates ðŸŽ‰ðŸ§µ @TransposeData https://t.co/sKkxOdMat9

1/ The Atlas is a collection of SQL queries curated by the @TransposeData team and other Transpose SQL users. Anyone can publish their queries to the Atlas, and users can search through them to find useful ones in their next projects!

2/ Another quality of life upgrade, shortened shareable URLs! Published queries can be shared by copying the link from the Atlas. For example, heres a query by @michaeljcalvey to get the top 20 NFT sales a wallet has made: https://t.co/ZygO8Lf1X4

3/ We've also allowed for Google and GitHub sign-in. No more copy-pasting your API key! https://t.co/iTRFQr6Jmz

4/ Since we've added Google and GitHub sign-in through your Transpose account, we've made it possible to select which API key is going to be used to make the SQL request. To change your key, use the menu on the SQL Playground window. https://t.co/LiUuXD3BYj

5/ That's all for now! If you want to check out the playground, head to https://t.co/9rmVEXCZIp. Stay tuned for more updates :) https://t.co/FPcfGj8SOi
