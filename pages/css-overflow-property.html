What is CSS overflow property ? CSS overflow controls the content if it cannot fit into defined area. This happens when height of the element is fixed. It's very important for Front-end programmers to know this property. If it is missed, it could lead to overflowing content.

Values of CSS overflow property: 1. visible 2. hidden 3. scroll 4. auto

1. overflow: visible This is default overflow property value. In this case, you will be content outside the boundaries of element box. https://t.co/4RxCzImEBx

2. overflow: hidden This will clip the overflow of content outside element box. It won't allow users to see clipped part. https://t.co/TqwAfjQPGk

3. overflow: scroll This will clip the overflow, and will add scrollbar to see the overflowing content. By default, scrollbars are hidden. It will show when scroll is done. https://t.co/AHbncLKVN5

overflow: auto This is similar to overflow: scroll, however, it will add scrollbar only if necessary. https://t.co/yPjw5SgWmf
