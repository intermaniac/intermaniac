What is EDA? EDA is a data analysis technique to derive patterns and insights from datasets and summarize their main characteristics, often with visual methods. (2/6) https://t.co/4pJhh0JVY5

Based on the nature of data and requirements, EDA can be classified as: 1. Univariate Analysis 2. Bivariate Analysis 3. Multivariate Analysis (3/6) https://t.co/6wZ2meeor1

1. Univariate Analysis: This refers to the analysis involving a single variable only, can be numerical or categorical. Includes plotting of histograms, distributions, frequency tables, bar & pie charts, boxplots etc. (4/6) https://t.co/PwTZxJx8Et

2. Bivariate Analysis This refers to EDA between two variables, again variables can be either numerical or categorical. Includes scatter plots, chi-squared, regression analysis and correlation coefficients etc based on nature of data. (5/6) https://t.co/tvX6yQNPWI

3. Multivariate Analysis Refers to statistical analysis involving more than two variables, be it either numerical or categorical. Includes applications in clustering, feature selection, root cause analysis, hypothesis testing, dimensionality reduction etc. (6/6) https://t.co/0Q1PXsb12Y
