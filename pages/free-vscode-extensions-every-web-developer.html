1. Prettier Prettier is an opinionated code formatter. It enforces a consistent style by parsing your code and re-printing it with its own rules that take the maximum line length into account, wrapping code when necessary.

2. Highlight Matching Tag This extension highlights matching opening and/or closing tags. Optionally it also shows the path to tag in the status bar.

3. Auto Close Tag Automatically add the closing tag when you type in the closing bracket of the opening tag.

4. Auto Rename Tag When you rename one HTML/XML tag, automatically rename the paired HTML/XML tag.

5. Color Highlight This extension styles CSS/web colors found in your document.

6. Better Comments The Better Comments extension will help you create more human-friendly comments in your code.

7. Bracket Pair Colorizer: This extension allows matching brackets to be identified with colors. The user can define which characters to match, and which colours to use.

8. ESLint Integrates ESLint JavaScript into VS Code. It statically analyzes your code to quickly find problems.

9. Live Server Launch a development local Server with live reload feature for static & dynamic pages.

10. Peacock Subtly change the workspace color of your workspace. Ideal when you have multiple VS Code instances and you want to quickly identify which is which.
