The CSS position property sets the type of positioning method, or you can say it defines the position of the elements. There are 5 different position values. 1. Static 2. Relative 3. Absolute 4. Fixed 5. Sticky

In CSS elements are positioned with the help of properties like top, right, bottom, left. - However, this elements will not work unless there is position property is not present. - They will work differently depending upon the position values.

Before diving into the position property, let's learn about top, right, bottom, left .

Top : - top property defines the verticle position of the positioned element. - it has no effect if the element is not positoned - the effect of the top property depends upon the what position property is given to the perticular element.

Right : - right property defines the horizontal position of the positioned element. - It has no effect if the element is not positioned.

Bottom : - bottom property defines the verticle position of the positioned element. - It has no effect if the element is not positioned. - The effect of the bottom property depends upon the what position property is given to the perticular element.

Left - left property defines the horizontal position of the positioned element. - it has no effect if the element is not positioned.
