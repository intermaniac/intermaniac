#Python has several Easter eggs built in! 🐍🥚🐣🐇 An Easter egg is a fun little surprise in a programming language, video game or movie. Let's talk about some of Python's cool Easter eggs! 🧵🐍👇

🥚1️⃣ - The most famous #Python Easter egg is probably The Zen of Python, which you will see if you "import this" https://t.co/Q9c5uBPtMR

🥚2️⃣ - What you might not know is that Zen of Python's source is really interesting too: https://t.co/nRsIGBZApx https://t.co/OMypYJoWW3

🥚3️⃣ - The next most well-known Easter egg in #Python is probably "import antigravity" If you type that in your Python REPL, you will be taken to this XKCD comic: https://t.co/znMLm2WGEz

🥚4️⃣ - Create "Hello World" with an import in #Python! https://t.co/yh32LXKt3A

🥚5️⃣ - Let's try and import braces in #Python! Nope! That Easter Egg won't let us! https://t.co/FdJeoPMc1j

🥚6️⃣ - This Easter Egg is kind of tricky. It's for the math nerds in #Python If you hash infinity in Python, you get back a representation of Pi If you hash NAN, you will get zero https://t.co/GfoDkdnkjn

🥚7️⃣ - Barry Warsaw, a longtime #Python core developer, was nominated "Friendly Language Uncle For Life" or FLUFL If you do this special import, the inequality operator "!=" is replaced with "<>" https://t.co/25Gg07QK4F

There are a few other Easter eggs mentioned in this curated list of Easter Eggs in Python on GitHub: https://t.co/VDWGzbS44d
