1. Promises: Promises are a pattern for handling asynchronous code that makes it easier to work with async code and avoid callback hell.

2. Async/await: Async/await is a syntax that makes it easier to work with asynchronous code, allowing you to write async code that looks and behaves like sync code.

3. Closures: Closures are a way to create a function that has access to variables in an outer scope, even after the outer function has returned.

4. Prototypal inheritance: JavaScript has a prototype-based model of inheritance, which is different from the class-based inheritance found in languages like Java and C#. Understanding how prototypal inheritance works is important for understanding how objects work in JavaScript.

5. Currying: Currying is a technique for creating a new function from an existing function by fixing some of its arguments. This can be useful for creating functions that are partially applied, or for creating functions with a consistent API.

6. Higher-order functions: Higher-order functions are functions that take other functions as arguments or return them as output. Understanding higher-order functions is important for understanding functional programming in JavaScript.

7. Generator functions: Generator functions are functions that can be paused and resumed, allowing you to write async code that looks synced.

8. WeakMaps and WeakSets: These are 2 collections that allow you to store keys and values (in the case of WeakMaps) or values (in the case of WeakSets), but the keys and values are held weakly, meaning that they can be garbage collected if there are no other references to them

9. Proxies: Proxies are objects that you can use to intercept and modify operations on other objects. Proxies are useful for creating objects with custom behavior, such as objects that are read-only or that validate their inputs.

10. Reflect API: The Reflect API is a built-in object that provides methods for intercepting and modifying operations on objects. The Reflect API is often used in conjunction with proxies.
