In 1994, Eich was an early employee at the Netscape corporation - a company founded by technology legends Jim Clark and Marc Andreessen. Netscape wanted to modernize the internet browser. After all, the internet was still in its infancy in the 90â€™s and looked like this https://t.co/ZhfotBM7L8

That year, Netscape released the Netscape Navigator - an internet browser designed for non-technical people. It was a polished version of Mosaic, the first user-friendly web browser released a year earlier in 1993. https://t.co/80nRnwcrjk

Before the Netscape Navigator, web pages were all static. This meant that a website would look the same for every user, and users would not be able to interact with the content (like posting a comment). Websites like Twitter could not exist back then.

Netscapeâ€™s management soon realized they would have to develop a new programming language that would allow for personalized and interactive experiences on the web. Their idea for this new language was to have similar syntax to Java and less like other scripting languages. https://t.co/om6mRFMwAe

How did Brendan Eich manage to create JavaScript - such a powerful language - in just 10 days? He revealed on the Lex Friedman Podcast that as a physics student in university, he had C compiler and front-end compiler knowledge, and copied a lot of code from Java and C. https://t.co/dhPfrwnD29

But where do you write a new programming language? Interestingly enough, Eich wrote JavaScript in an HTML comment on older browsers. He embedded JavaScript in HTML so the code would get passed along to the website, similar to how a computer virus works. https://t.co/oJUMPMGVXZ

In December 1994, JavaScript was officially created. Throughout the years, it has been updated and numerous open-source frameworks like node.js and react.js have been created to push the world forward. https://t.co/JXWsr1ajfD
