Why and how to use #Python format printing ðŸ§µ

Why use format printing? Format printing makes adding variables to strings concise and easy. Whether you want to check what a variable is or print variables for a user, format printing is the best way to show results.

There are two main ways to use format printing in Python: using % and the format() method

Using %: Within the string, add a % with a character signaling what data type the variable is. Outside of the string, add a % with the variables in order of printing character choices: f -> Float s -> String using set() r -> String using repr()

%f has the syntax %x.yf, with x being the minimum total number of characters in the float (unused characters being added as spaces to the front of the string) and with y being the number of numbers after the decimal point https://t.co/YECFphvOLJ

%s and %r can convert most data types to a string, with %s using set() to create the string and %r using repr() For a standard string print, use %s For a string print with quotations, use %r https://t.co/rZQC38eU3F

Multiple variables can be added to a string by surrounding the outside variables with parentheses, separating each variable with a comma https://t.co/x5XWImR5ME

Using format(): Outside of the string, add ".format()" with the variables within the method Within the string, add curly brackets {} with the index of the variable within the format() method https://t.co/GEQdIIvuK1

Variables can also be identified by a keyword when using format() https://t.co/Ppq3cZw9u5

When to use what method (% vs. format()) Use % when you want more control of your printing and/or when adding extra 0's to numbers Use format() when you want to print a variable multiple times
