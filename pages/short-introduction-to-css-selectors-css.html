css selectors are basically used to 'select' the html elements you want to style and give them a specific design.

css selectors are divided into 5 types/categories : 1. simple selectors 2. combinator selectors 3. pseudo-class selectors 4. pseudo-elements selectors 5. attribute selectors

the element selector - selects the elements of html doc based on their names! https://t.co/ekwvcqo91j

the css id selector - selects the attribute of a specific html element, the id can be unique to a page and can be edited respectively. https://t.co/9tz3wqmimw

the css class selector - selects html elements with a specific class or class attribute! https://t.co/hwdopanjtf

the css universal selector - selects all the elements as the name justifies! https://t.co/ttqhxeshez

css grouping selector - groups similar styled elements and style them accordingly! https://t.co/tdfoym14jd
