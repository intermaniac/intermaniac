Solidity programming language influences Solidity is a programming language: - proposed in 2014 by @gavofyork - developed by the Solidity team, led by @ethchris A thread of the main languages which influenced Solidity: C++, JavaScript, Python. ↓ https://t.co/NF9D29YKiX

@gavofyork @ethchris Table of Contents - a curly language - C++ influence - JavaScript influence - Python influence - GitHub Repository ↓

@gavofyork @ethchris A curly language a Curly language, or better a Curly-bracket language, is a programming language that has a syntax that defines statement blocks using the curly brackets { }. Example: https://t.co/UxQbSltZ4E

@gavofyork @ethchris C++ influence - implicit/explicit type conversions. - variable declarations. - function overloading. - for loops.

@gavofyork @ethchris JavaScript Influence - general look and initially inspired by the ECMAScript. - functions are defined using the keyword function. - import syntax and semantics that are similar to those available in JavaScript. The JS influence has been reduced lately.

@gavofyork @ethchris Python influence - modifiers are inspired by Python decorators. - multiple inheritance. - C3 linearization. - general assignment/copy semantics of value/reference types.
