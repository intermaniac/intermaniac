What I learned today: Images in HTML ðŸ§µ #100daysofcoding #HTML #CSS #javascript

1. How do we put it? Using <img> in html, you can embed an image. The 'src' property is where you declare the path of the chosen image. It could be in a directory (eg. /folder/image-file.jpeg) or in a link (eg. https: // www .example . com / images / sample.jpg)

2. 'alt' property The 'alt' property refers to the alternative text of the image that DESCRIBES it. If the image did not load properly, the 'alt' text will be shown instead. It is important because...

It is handy for: - visual-impaired users that uses screen readers - provides alternative text to other browsers that does not support image type - search engines - user's preference (reducing data transfer)

3. Width and Height You can specify the width and height of the <img> by: - typing 'width' and 'height' to the element property - using CSS By best practice, you shouldn't change it by HTML attributes.

4. Using figure and figcaption Using these elements, it provides a semantic container for figures with caption. There is nothing wrong with using 'div' as a container and 'p' as the caption. But using the proper semantic elements will help assistive technologies.

5. CSS Backgrounds If you want to use an image just for decoration ONLY, embed it using CSS. NOTE: Only use it for decoration purposes. If there is NO semantic meaning at all. Still, use the <img> element if an image has a meaning.
