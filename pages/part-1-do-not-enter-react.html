What is React? - Open source JavaScript library for creating User Interfaces (UI). It is declarative, efficient and developers use it to create single page applications. React library only focuses on the view layer of MVC model

What are features of React? - React uses VirtualDOM, not real DOM. Which is perfect to see the updates applying to your app as you build it. - Unidirectional way data binding. Data passes from parent to child - Develop web applications as well as mobile using React Native

How to start a new project in React? - We can initialize project and do all stuff from scratch and install dependencies, but the easiest way of doing is: https://t.co/N2XUXEkb7S

What does DOM stand for ? - Document Object Model

Types of DOM? - Virtual DOM and real DOM
