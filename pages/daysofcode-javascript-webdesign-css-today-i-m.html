Step-1:- Pick a font which is popular and has some basic weights to it like Bold, Italic, Bold-Italic, and Light.

After that, use that same font for your whole website and make customizations to it. Check whether it suits your expectations.

Step-2:- Find and select a font which has many weights to it.

If you are using https://t.co/QF0IXNWyct (and you most likely will), select a font which has styles like 10 or more to it (which you can find at the right side of the font's title itself). Cuz the more the styles, the more you can customize the font.

Step-3:- Find a second font that "POPS!" This is for those fonts which requires special attention from the viewers. You can use this font for titles, some texts in a paragraph which needs 'popping'.

Remember this, that this font must be contrast from the first font. For example, if you've picked a font like 'Roboto' (sans-serif), then use "cursive" like fonts for the second one! Which will be contrast as well as grabs special attention.

Don't use the same font-family twice! For example:- don't try 'Roboto' and 'open-sans' thinking that they're different. Make sure to check their category of the font, and try to use different category of fonts and be contrast.

One more crucial thing is that some fonts has only capital letters style or customization to it. Which is okay if you're using it for titles and stuffs. But think about that for a paragraph, the viewer will be skipping that immediately because that style is not pleasing to see.

So make sure to be aware of it and be careful while picking font-styles for your website!
