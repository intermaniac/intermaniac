Events in React A ðŸ§µ Events in react are just like events in JavaScript. The difference in React is that since we are setting them up inline, we need to use camel case. so an event like onclick is written onClick. Do check the React official docs for more supported events. https://t.co/ALVilBoULv

We need two things to set up events in React: 1. Attribute 2. Event Handler Attribute is the name for the type of event to be carried out. We will be looking at the onClick event in this thread. Event Handler is the function you want executed when the event occurs.

We can set up the Event Handler as a reference or as a inline function. Passing Event handler as reference We pass in the Event handler as a reference for onClick event by setting up a separate function and writing the name of the function in the curly braces. https://t.co/8thh49YK3y

Here, we display "Hello World" in the console after clicking the button. https://t.co/SdOj8KH5pn

Passing Event handler as inline function We pass in the Event handler as an inline function for onClick event by setting up the function directly inside the curly braces. https://t.co/GeBpFL5AJG

Here, we show an alert of "I love programming" in the browser window after clicking the button. https://t.co/saZokgR2Ur

Note that both methods of setting Event Handlers will do the same thing so pick the one that work best for you.

In cases where an argument is required to execute a function, if we pass in that function directly, the function will be invoked the moment you render your application. We don't want such a situation as we only want to invoke our function after clicking the button.

For example, we want to show the "Fruits" header in the console after the click of the button. If the function is passed directly, it will be invoked immediately your application renders https://t.co/GFqnWOaJNa

Here, we can see "Fruits" header in the console immediately after refreshing the page. https://t.co/sjMKupkdFY

So to prevent invoking the function immediately we render our app, we first setup an arrow function before passing in the function. https://t.co/BieZU0EXCa

Here, we can see everything works as it should. "Fruits" is shown in the console only after clicking the button. https://t.co/EcGtn03d5n
