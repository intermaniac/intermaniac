I'm working on building a GitHub pages website and whew it's not hard, but also not very straight forward. Might as well make a thread of wtf is going on in case others find it useful! https://t.co/gC7dPu5kzu

First step is pretty straightforward. Create the repository for your page. This is pretty much all I accomplished on Day 1 because I spent the rest of my time googling other people's websites. https://t.co/gAIhjxwEkf

I decided I wanted to add a theme. So the next step is to spend way too long overthinking which theme to use. That was Day 2 for me. No choosing. Only overthinking. https://t.co/YmaDMreVNx

I decided to choose the minimal mistakes theme and that led me here: https://t.co/FO6I9OJYyD">https://t.co/FO6I9OJYyD The first step in the install instructions say to add text to my Gemfile. I don't have a Gemfile. I responded swiftly to this hurdle by wondering off and making some tea.

Spongebob narrator voice: Much googling later. I found a bunch of Jekyll websites that also pretty much assumed I already had the suite of files already (e.g. _config.yml, Gemfile, etc) but...nope. I did find this useful for install prerequisite stuff! https://t.co/Zj525FPZBs

I eventually made my way BACK to GitHub and found directions for creating the basic Jekyll/GitHub pages framework. Be sure to clone your repository so its on your local machine! I def had to remerge mine with lots of googling. https://t.co/bF3AIU59aj

I still don't really know how to move around different file locations using the terminal. Did you know you can right-click on a folder and open a terminal at that location? Hashtag life hack. https://t.co/WeGrc4Bfvj

Oh! And now I could finally complete the install for the theme. Back to here: https://t.co/FO6I9OJYyD">https://t.co/FO6I9OJYyD

Once everything was completed and I pushed that commit I was ready to check out my baby website. But the GitHub action that runs every time you sync stuff to your pages repository now was unhappy and failing. End of day 3? https://t.co/Sw6kAivN8E

Another googling session and going back to check other folk's https://t.co/FbjcYj9BtX repositories later I found a different workflow action (search for GitHub pages Jekyll and you should find it) that you can add and now it works! I have a blank website that builds! End of day 4 https://t.co/Wemk0VNdZX

Walking away before I find anything else that doesn't work. Thread to be continued... https://t.co/02FPuOJzl7
