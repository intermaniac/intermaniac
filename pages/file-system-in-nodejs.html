📍Brief introduction → The file system(fs) is a core package inbuilt so it does not require third-party installation → fs module provides tons of utilities to interact with the system’s operation.

📍Getting started → Let’s look at some common methods to use, shall we? → To import packages of fs, write the snippet down below on the top of your codes.👇 https://t.co/XbZTVBSVpy

📍Reading files → Format: fs.readFile(path, callback function) → To read files asynchronously. https://t.co/E1TQlaN1oU

📍Writing files → Format: fs.writeFile(path, data, callback function) → To write content to a file asynchronously. If a file exists, the system will override the content. https://t.co/Agl0nLYfid

📍Reading directories → Format: fs.readdir(path, callback function) → This asynchronously read the content of the directory and returns an array of file and directory names of that directory. https://t.co/wS2fgrMC2W

📍Checking the existence of a path → format: fs.existsSync(path) → It returns true if the path exists. https://t.co/kH0opneTGw

📍 Creating a new directory → Format: fs.mkdir(path, callback function) https://t.co/wtwKJotcZN

📍 Deleting files → Format: fs.unlink(path, callback function) → This deletes the file with the path asynchronously, and it will not delete the directory. https://t.co/DD3lKrSDXr
