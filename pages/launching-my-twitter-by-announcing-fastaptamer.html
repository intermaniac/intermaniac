Launching my Twitter by announcing FASTAptameR 2.0 - the latest iteration of our combinatorial selections software! Users can access it from the web, Docker, or GitHub (links @ end of thread). @BurkeLabRNA @BiochemPhD @DHBurkeAguero https://t.co/04n4BICKiw

Like the original, FASTAptameR 2.0 addresses analytical tasks that are fundamental to all combinatorial selections, such as Counting occurrences for each sequence Clustering related sequences Calculating enrichment between populations Filtering / tracking sequences by reads

FASTAptameR 2.0 features an incredible web-based user interface, interactive graphics, detailed user guide, and explicit support for any linear, sequence-encoded selection - from aptamers and ribozymes to phage display and protein evolution, using native or modified alphabets.

Analysis begins by counting individual sequences and defining species metadata using the COUNT module. Analytical pipelines are then fully customizable, allowing users to create workflows that connect intuitive modules. https://t.co/t3BtKn2CW1

The new user interface allows interactive exploration of selection data. This screenshot shows an example of the intuitive user interface for the classic 'Count' module. (While FASTAptamer 1 was a great early tool, its command-line interface limited usability for many users.) https://t.co/80AqZqvc1o

The original FA modules have been reworked to be more efficient. E.g., sequence clustering uses an optimized algorithm that made test cases run ~100x faster. Clustered outputs can be visually evaluated with an interactive PCA plot of kmer frequencies. https://t.co/PzO5upAL6j

New modules include sequence translation, motif/sequence/cluster tracking, motif discovery, mutation network discovery, and many more. FOR EXAMPLE: Looking at position-level enrichment (below) shows conservation in the aptamer motif (a pseudoknot). https://t.co/xN18q3gA83

This is @BurkeLabRNA's first collaborative paper with Mizzou's Digital Biology Lab (directed by Dong Xu) but certainly not the last! More software upgrades and innovations are currently in progress. Stay tuned.

Please check out our combinatorial selections software FASTAptameR 2.0 using the web version (https://t.co/llikOpQ6EQ) or local versions (https://t.co/bQo6aM1eY7, https://t.co/6iyCyNsJqi) and let us know what you think!
