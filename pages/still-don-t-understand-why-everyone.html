Still don't understand why everyone is using #Docker? üê≥ üßµ Here's what Docker is and why you should use it in your next project: #devops #sysadmin #server #programming #coding #developer

Docker is a tool which packages applications into "containers" that can execute reliably regardless of the environment. This is useful if you need to run an application in multiple stages of the deployment process under 1 environment: development, testing staging & production. https://t.co/IGVgacYlbM

There are 3 main components to Docker:

1. Dockerfile Dockerfiles consist of code which tells docker how to build a Docker Image. https://t.co/jSdMXFPz1q

2. Docker Image A Docker image is an immutable snapshot of your application's code and dependencies down to the operating system. Each line represents a command in the process of building the final image and is referred to as a "layer". https://t.co/TprQOUqZJw

3. Docker Container A Docker container is a Docker image running in an actual environment. You can run multiple containers from one image. https://t.co/YTC7LMfNqO
