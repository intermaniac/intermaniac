When we are new to JavaScript, Many people find it difficult to learn JavaScript. They don't know how to start, where to start, and mainly what are the important topics to learn JavaScript. Here is the best way that I used to learn JavaScript.

1. Learn the basics of HTML and CSS JavaScript is mainly used in the Front-End to perform DOM operations. (Now JavaScript is everywhere) So to start with JavaScript First learn basics of the HTML so you will know the real use of javascript, and why developers use javascript.

2. Learn the Basics of JavaScript After knowing how to design a web page, Learn the following basics things: - Variables - Data Types - Functions - Events - Method chaining - Loops - Conditions

3. Perform simple DOM Operations JavaScript was mainly invented to perform DOM operations. So to start with JavaScript first learn how to perform basic DOM operations using JavaScript. [continued...]

Here are a few operations you can try: - Select the div - Change the content of the div - Perform events on the button - Change the style of elements - Add events(clicks, mouse etc). - Perform events on the button - Change the style of elements

4. Intermediate Phase Now you know how to manipulate the document using JavaScipt. Now it's time to power the website using JavaScript. Learn the following things: - Learn about APIs - Fetch data from the backend without reading the web page using Fetch or Axios [continued...]

- Add and remove elements from the document - Learn to use third-party libraries like Charts or animation libraries - Fetch data from APIs and load it dynamically from the document

5. Advance Phase Now you have learned 80% of JavaScript, But the next 20% will take the same time to learn that took to learn the first 80%. And these last 20% of topics make top developers and many people stop learning because of this. [continued...]

Here are the topics to learn to become a complete JavaScript Developer: - Async/Await - Callbacks - Promise - Closure - Arrow functions - Objects
