Beginners Introduction to ReactJS ⚡️ A 🧵 👇 https://t.co/pMsHmveVvC

☑️ ReactJS is a JavaScript library which was created by Facebook in 2013 ☑️ReactJs allows us to create reusable UI components. ☑️ ReactJs is used for building simple, fast and scalable frontend web applications. ☑️ReactJs uses unidirectional data flow

👉 ReactJs Pre-requisites: ☑️ HTML and CSS ☑️ Fundamentals of Javascript and ES6 ☑️ Document Object Model ☑️ Package Manager (Node + Npm) ☑️ Git and CLI (Command Line Interface)

👉 Features of ReactJs ☑️ Simplicity ☑️ Performance ☑️ One-way Data Binding ☑️ Virtual DOM

☑️ Simplicity: React uses a component-based approach which makes the code reusable as per your need! ☑️ Performance React uses Virtual DOM, DOM executes in memory so we can create separate components which make the DOM run faster

☑️ One-way Data Binding Like I said earlier, ReactJs uses unidirectional data flow, i.e. the data flow from parent components to child components! https://t.co/TfinMmxXx0

☑️ Virtual DOM A virtual DOM object is a representation of the original DOM object but Manipulating the DOM is slow whereas Manipulating the virtual DOM is much faster, because nothing gets drawn onscreen.

☑️ React js Concepts for Beginners 🎯 JSX 🎯 Components 🎯 State 🎯 Props

☑️ JSX JSX allows us to write HTML in React. It makes it easier for us to write and add HTML in React. https://t.co/UYb5nOKGXR

☑️ Components In React, Components are independent and reusable bits of code. They serve the same purpose as JavaScript functions, but work in isolation and return HTML. types of components: 🎯 Class Components 🎯 Function Components https://t.co/mwEAk0q0R4

☑️ State The state object is where you store property values that belong to the component. We use the React useState Hook which allows us to track state in a function component. https://t.co/ygBNyzXjeQ

☑️ Props props are used to pass data from one component to another i.e from a parent component to a child component. https://t.co/vMIs2Gv5ib
