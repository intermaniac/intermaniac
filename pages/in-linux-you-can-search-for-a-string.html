In #linux, you can search for a string in a file with different commands like grep, sed and awk. In this thread, you'll get to know how you can use awk #command to search for strings in a file as an alternative to grep. There are some useful builtin functions comes with #awk.

awk breaks every records/line from file into fields based on whitespace. Eg: file.txt contains "Java Programming Language" Then #Java would be $1 i.e. 1st field Then #Programming would be $2 i.e. 2nd field Then #Language would be $3 i.e. 3rd field $0 represents entire line.

Pattern: awk 'pattern {action}' file.txt Some basic examples: Printing entire file - awk '{print}' file.txt Printing entire file with line numbers - awk '{print NR,$0}' file.txt Printing first word of first 5 lines - awk '{print $1}' file.txt | head -5

You can print specific columns or words using - awk '{print $1}' file.txt - awk '{print $2}' file.txt - awk '{print $1, $2}' file.txt Printing last word - awk '{print $NF}' file.txt

Printing lines that contains letter a - awk '/a/' file.txt or - awk '/a/{print}' file.txt Printing lines that starts with a - awk '/^a/' file.txt Printing lines that ends with a - awk '/a$/' file.txt You can add more #regex patterns for more efficient searching.
