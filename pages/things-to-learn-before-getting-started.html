3. slice and splice Method returns a new array with selected elements, while splice method changes the contents of an existing array. https://t.co/heS1Yvujpf

6. destructuring The destructuring assignment is a special syntax which enables unpacking (assigning) values from arrays or object properties directly into variables. https://t.co/qgL0TC1iQZ
