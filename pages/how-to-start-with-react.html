How to start with React.js? const A_THREAD = ( )=>{ };

React is nothing but bundles of javascript someone already writen for you Hence Step 1: Master vanila (HTML , CSS , JS) Build projects -Tic tac toe -Calculator -Wheather using fetch

Step 2: learn why React , what does it make easy for us , how can we reduce repetitive code by using library like React.js . Step 3: basic file folder knowledge and create-react-app or may be vite what ever you prefer..

After learning how to initiate react app Step:5 What are components and how can we reuse them... For eg:- a nav component which is made once can be used on more than one page just by typing <Nav/>

Step:6 learn about props... How props helps us to customize a perticular component by sending demanded property to it. It can be also use to share variables from parent file to child . Mostly known to us as prop drilling...

Step:7 What are hooks and how they make our life easy ,? Specially study about useState and useEffect . They help us to change the state of a particular component in a particular situation. Make some projects using hooks - counter app - calculator -dark light theme switcher Etc

Step 8: learn what is Axios and how it is different from regular fetch function .At this part you also have to revisit your async await and promises . Try building youtube , spotify , google map clones by fetching data from RAPID API

Step 9: What is context API , how it helps us to manage states. When you will start building projects with states you will realise the more states there are the more complex your app get to tranfer one states from one component to another . This is where context Api stand...

Step 10: Learn redux / redux/toolkit , there a plenty of youtube tutorials on these topics . Redux is just more advance context API but they are not the same.. Try building e- commerce websites using redux/toolkit ...

Step 11: Try creating some of your own custom hooks . Enhance your problem finding as well as solving skills . Use your own hook in your own project ....

Step 12: finally learn how to deploy your react app for free - @vercel - @Netlify -@heroku These service providers can be helpfull..

Choose any css library , i go for tailwind and start building as many project as you mentally can . Build a portfolio to show case all of them. And now you are ready to get a role of an beginner react developer ... Keep learning more and more ...
