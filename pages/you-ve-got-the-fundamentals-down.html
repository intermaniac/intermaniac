What is a JavaScript framework? Frameworks like React, Vue, and Svelte are all front-end tools that help you create interactive, data-driven layouts. They make DOM manipulation, data rendering, and state management a lot easier.

Why are JavaScript frameworks important? As projects get more complicated we can find ourselves building helper functions to make development easier. examples: - conditionally hide and show a component based on state - create x number of elements based on array length 1/3

The example list of helper functions can get huge and complicated. This is what these JavaScript frameworks were built to solve. They each have a way to handle the above examples without you writing extra code. 2/3

Here is a list of typical features of a JavaScript framework: - Conditional rendering - Looping over data for element creation - Data fetching - Event handling - Componentization - Component lifecycles - State management 3/3
