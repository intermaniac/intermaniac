A brief History of HTML - HTML (HyperText Markup Language) is a standard markup language used to structure text on the web. It was developed in the late 1980s by Tim Berners-Lee at (CERN) Why was it created?

Sir Tim Berners-Lee developed HTML as a way to create documents that could be displayed on computers connected to the internet and linked to other documents.

HTML is a way to write and format text on the internet. Imagine you're writing a letter to a friend. In the letter, you might want to make some words bold or italics to emphasize them, or you might want to create a list of items.

HTML lets you do all of these things and more cool things on the internet using symbols called tags. A tag is a special symbol that tells the web browser how to format the text.

For example, if you want to make some text bold, you can use the <b> tag to wrap around the text like this: <b>Hello everyone this text is bold!</b>

The text between the <b> and </b> tags will appear in bold when it's displayed in a web browser. Some tags need to be closed while some don't. <> is an opening tag </> is a closing tag. The symbol of the tag is called the Angle bracket.

There are lots of different tags you can use to format text and create lists, tables, and other types of content. You can even use tags to create links to other pages on the internet.

To create a link to another webpage in HTML, you can use the <a> tag, also known as the anchor tag. The <a> tag is used to create a hyperlink, which is a link that takes you to another webpage when you click on it.

One more example is a webpage about your favourite animals. You want to include a list of your top five animals, with each animal's name on its own line. In HTML, you can create a list using the <ul> tag for unordered lists, which means the list items are not numbered.

You can then use the <li> tag for each list item. Here's what the HTML code might look like:

<ul> <li>Panda</li> <li>Gorilla</li> <li>Tiger</li> <li>Lion</li> <li>Bear</li> </ul> When this code is displayed in a web browser, it will look like this: - Panda - Gorilla - Tiger - Lion - Bear

You can also create an ordered list, which is a list with numbered items, using the <ol> tag instead of the <ul> tag. The rest of the code would be the same, but the list would be numbered like this: 1. Panda 2. Gorilla 3. Tiger 4. Lion 5. Bear
