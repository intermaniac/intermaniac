Some VScode extensions that'd make your coding and debugging experience a lot smoother and easier. We all know the hassle of debugging code even with the aid of an IDE, so I've curated some extensions that have helped me with day to day coding on VScode. #techtwitter ðŸ§µ

1. Live Server: This launches a local development server with a live reload feature so you'd see changes in real time as you code. Best when VScode is on autosave. Makes it easier and faster to spot errors in static and dynamic pages.

2. Prettier: This is an opinionated code formatter that formats your whole code immediately you save. This saves time and keeps code consistent with preset rules especially when working in groups. Really love this because it fixes indentations which was a real time waster.

3. Bracket pair colorizer: I know most have had syntax errors relating to unclosed braces(normal or curly{}) especially when they are huddled together. This gives each pair a different color making it easier to spot lone unclosed ones. A real time and headache saver. Lol.

4. Auto rename tag: This is a lifesaver really. It takes away the stress of having to open and close a tag, and renaming a tag twice. So whenever you type the name of the tag it automatically generates the opening and closing pair. Also when you change one part both pairs change.

5. Git Lens: This simply helps you quickly glimpse into whom, why, and when a line or code block was changed. You jump back through history to gain further insights as to how and why the code evolved, effortlessly exploring the history and evolution of a codebase.

6. JavaScript code snippets: This is a library of helper functions that adds more import/export triggers, class helpers and method triggers. So by typing three letters you could have an auto-completed import, class or method structure.

7. Colorize: This instantly visualizes CSS colors in your code, making it easier to know the particular color associated with your hex or other type of color code.

8. Code spell checker: This just helps check spelling errors in the text editor. Not a do or die though as programs don't really care how you spell variables or constants and the like.

9. Debugger for Chrome: This simply allows you debug your js code in VScode.

10. Icon fonts: This offers snippets for a variety of icon fonts including the Font Awesome pack. These font names can be funny and a bore at times.

11. Regex previewer: Writing regex can be a pain sometimes, this gives you a side document that matches your regex. It gives a variety of examples to match so you can easily and quickly write regex with a variety of use cases.

The end for now. I hope these would help your ease your debugging and coding journey as they've helped me.
