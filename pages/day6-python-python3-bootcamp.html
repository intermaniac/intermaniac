1. What is a function? A function is a block of code or group of related statements which only run when it is called. Functions ensure modularity in our programs that is it ensures that our program is divided into small modules which ensure modularity.

2. Types of functions: In Python, there are various types of functions. They include: a) Builtin functions b) Lambda functions c) Recursion functions d) user-defined functions

a) Built-in functions These are functions that are pre-defined in a programming language. They include for example print(), abs(), len() etc.

b) Recursive functions These are functions that can call other functions or the functions themselves. NB: We'll cover the recursive functions in detail in another thread

c) Lambda functions A lambda function is a small anonymous function. A lambda function can take any number of arguments, but can only have one expression. NB: We'll cover the lambda functions in detail in another thread

d)user-defined functions These are functions that are defined by the programmer. In Python, there are various examples of user-defined functions. i) Simple function ii) Functions with outputs ii) Higher-order functions iv) Decorator functions

We'll cover the user-defined functions in a series of threads. Today will cover a simple function. i)Simple function This is a function without a return statement or input that is it has a basic structure.
