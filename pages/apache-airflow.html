Apache Airflow...The Complete Thread Apache Airflow is one of the most used Apache's Projects, we have 100+ Apache Projects including - Apache Kafka - Apache Arrow - Apache Iceberg and many more.. #apache #backend #airbnb #python

Apache Airflow was created by Airbnb in the year 2014 The "Air" in Airflow comes from Airbnb.."flow" represent workflow Apache Airflow is a workflow management system,Apache Airflow isn't an ETL tool ETL is an acronym for Extraction Transformation and Loading #python @Airbnb

Airflow is built on top of Python...So it is used heavily by Python Engineers for workflow management An example of a workflow can be Extraction of data from multiple sources going through transformation using and finally loading into a Data warehouse for further analysis

The major components of Airflow are - Schedulers - Workers - Executors - Metadata(A postgresql database) - DAGs DAG -> Directed Acyclic Graphic is the tree structure of the workflow It is called Acyclic because you can't move backwards once an operation fails it reschedules it

Airflow can be run locally or run on docker The default port of Airflow is 8080 #python #code @Airbnb #Airflow #thread #CS #computerscience
