CSS border enables you to specify the box's border area. Between a given element's margin and padding, borders are displayed. The border may either have a predetermined values: none, hidden, solid, dashed, dotted, double, inset, outset, groove, ridge or an image.

BORDER STYLE An element's border can be solid, dotted, etc. thanks to the border-style attribute. It is a shortened property that allows the line style for the border's four sides to be configured. https://t.co/4gLcMU1rDV

There is a small difference between the values none and hidden, which both display no border. If any other competing borders are specified, the none value has the lowest priority when it comes to table cell and border collapse, while the hidden value has the most importance.

The values inset, outset, groove, and ridge produces an effect that resembles 3D and is largely dependent on the border-color value. This is often accomplished by combining two colors that are just a little lighter and darker than the border color to provide a "shadow".

Since none is the default border style, you must provide one in order for the border to be displayed around an element. Contrarily, the border's default width or thickness is medium, and its default color matches that of the text.

BORDER WIDTH The width of the border region is specified by the border-width attribute. It is a shortcut property for simultaneously adjusting the border thickness on an element's four sides.i https://t.co/9ydg3kmQJG

The border width can be specified using any length value, such as px, em, rem, and so on. In addition to length units, the border width may also be specified using one of three keywords: thin, medium, and thick. Percentage values are not allowed.

BORDER COLOR The border area's color is defined by the border-color attribute. The border color of an element's four sides can also be changed using this shortcut attribute. https://t.co/WZ3XV1jYC9

When used alone, the CSS border-width or border-color properties are ineffective. Set the border's style using the border-style property.

Shorthand Property of the Border Border-width, Border-style, and Border-color can all be defined in one rule by using the border CSS property, which is a shorthand property. https://t.co/HtoAD8SoDz

The default value for that property, if any, will be used in its place if the value for a specific border property is left out or not supplied while setting the border shorthand property.

For example, the element's color property will be utilized as the value for the border color if the border-color property value is absent or not supplied when establishing the border. https://t.co/Y0Sy9ZubAx

However, because the default value for this property is none, border-style will not display at all if the value is left blank. https://t.co/XrU2oWJHpo
