What is an if/if else statement? Combined with a comparison operator if statements can be used to conditionally run code. If something is true, run this, if something else is true run that, else run something else. https://t.co/5xEAj0IiUm

What is a ternary conditional statement? A special operator that performs an if-else check using a short form. It can be used to compare and return a boolean true/false value easily. https://t.co/ZNp3cv3DtF

What is a switch case conditional statement? A comparison statement that checks an expression against a series of cases. It will execute the first case that meets the condition and continue checking until it hits a break statement. https://t.co/GmxGke9IJx

If statements should be used for complex conditional evaluation, especially when nesting if statements is required. Ternary operators are great for simple checks that automatically need a value returned. Switch statements are cleaner when evaluating a ton of stacked conditions
