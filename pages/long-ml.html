Long ML thread. Data Analysis: What is it? It is a process where the data is inspected, cleaned, transformed, and modeled with the aim of extracting actionable knowledge to draw useful inferences. Some examples of data analysis tools are Python, R Programming & MATLAB. 1/n

There are many types of data analysis, such as text, statistical, diagnostic, predictive, and prescriptive analysis. 1) Text analysis is also referred as text mining. • It is the process of extracting relevant information from unstructured textual data. 2/n

2) Statistical Analysis: • It is a process of collecting,organizing,exploring, interpreting, and presenting data using statistical techniques • It allows businesses to make informed decisions. It can be further classified into descriptive and inferential statistics. 3/n

3) Diagnostic Analysis: • This aims to identify the causes of data being the way it is. It makes an attempt to diagnose a potential problem, as characterized by the data, and determine causative factors using correlation and other statistical/analytical techniques. 4/n

4)Predictive Analysis: • It aims to predict future results by gleaning information from existing/historical data. • It is used to predict trends, behavior patterns of systems or customers, occurrence of bugs, and so on. • includes artificial intelligence and data mining. 5/n

5) Prescriptive Analysis: It takes predictive and diagnostic analysis one step further, by providing a set of actionable decisions to reach the required business goal, rather than simply providing a root cause analysis of a problem, or predicting future outcomes. 6/n
