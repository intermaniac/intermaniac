Let me tell you my quick story how I started using CSS frameworks.... Flashback: I used to write plain css in my code to design everything

When I made my first portfolio website, it took me 2 whole months for designing the website While learning React Js, I tried bootstrap. A little practice and you're good to go. After that, I also learnt tailwind css

So, How you should get started?

Basics : Framework is a predefined css which is implemented by the server to the element where class is defined To implement the property, you have to write the value in class attribute Eg. In bootstrap, class="text-center" makes object in center https://t.co/aW0O0Wqq2i

To begin : You have to add the script and style link of the framework in index.html (Go to documentation of the framework for these links)

To implement: Go to documentation and read a little bit. Learn which value gives you your results Bonus : press Ctrl+K for search inbetween the docs

Now you are good to go. Search, learn, paste in code, test, repeat üîÅ Bonus : If you're beginner, try Bootstrap first

Here are top css frameworks: Bootstrap Tailwind Bulma Materialize Pure.css
