üêß Cursor Movement ‚Üí Ctrl + A - Go to the beginning of the line you are currently typing on. ‚Üí Ctrl + E - Go to the end of the line you are currently typing on. ‚Üí Ctrl + XX - Move between the beginning of the line and the current position of the cursor.

‚Üí Alt + F - Move cursor forward one word on the current line. ‚Üí Alt + B - Move cursor backward one word on the current line. ‚Üí Ctrl + F - Move cursor forward one character on the current line. ‚Üí Ctrl + B - Move cursor backward one character on the current line.

üêß Text manipulation ‚Üí Ctrl + U - Cut the line from the current position to the beginning of the line, adding it to the clipboard. If you are at the end of the line, it cuts the entire line.

‚Üí Ctrl + K - Cut the line from the current position to the end of the line, adding it to the clipboard. If you are at the beginning of the line, cut the entire line. ‚Üí Ctrl + W - Delete the word before the cursor, adding it to the clipboard.

‚Üí Ctrl + Y - Paste the last thing from the clipboard that you cut recently (undo the last delete at the current cursor position).

üêß Access your History ‚Üí Ctrl + R - Lets you search through previously used commands. ‚Üí Ctrl + G - Leave history searching mode without running a command.

‚Üí Ctrl + J - Lets you copy current matched command to command line without running it, allowing you to make modifications before running the command. ‚Üí Alt + R - Revert any changes to a command you‚Äôve pulled from your history, if you‚Äôve edited it.

‚Üí Ctrl + P - Shows last executed command, i.e. walk back through the command history (Similar to up arrow). ‚Üí Ctrl + N - Shows next executed command, i.e. walk forward through the command history (Similar to down arrow).

‚Üí Ctrl + _ - Undo previous command ‚Üí Ctrl + O - Executes and re-enters the command found via Ctrl + R and Ctrl + S ‚Üí !! - Execute last command ‚Üí !* - Execute previous command except its first word. ‚Üí !*:p - displays arguments of the last command.

‚Üí !n - Executes the nth command in history ‚Üí !n:p - Displays the nth command and adds it as the recent command in history. ‚Üí !$ - displays the forth last argument of the previous command.

‚Üí !^ Substitutes first argument of last command in the current command. ‚Üí ^foo^bar - Replaces foo with bar. ‚Üí !n - Executes nth command from the bash history.

üêß Terminal/Process control ‚Üí Ctrl + L - Clears the screen, similar to the clear command. ‚Üí Ctrl + S - Stop all output to the screen. This is useful when running commands with lots of long output. But this doesn't stop the running command.

‚Üí Ctrl + Q - Resume output to the screen after stopping it with Ctrl+S. ‚Üí Ctrl + C - End currently running process and return the prompt.

‚Üí Ctrl + D - Log out of the current shell session, similar to the exit or logout command. In some commands, acts as End of File signal to indicate that a file end has been reached.

‚Üí Ctrl + Z - Suspends (pause) currently running foreground process, which return shell prompt. You can then use bg command allowing that process to run in the background. To again bring that process to foreground, use fg command. To view all background processes, use jobs cmd.

‚Üí Tab - Auto-complete files and directory names. ‚Üí Tab - Tab Shows all possibilities, when typed characters doesn't uniquely match to a file or directory name. ‚Üí Ctrl + S - Gets the next most recent command.

üêß Close Terminal ‚Üí Ctrl + Shift + W - To close terminal tab. ‚Üí Ctrl + Shift + Q - To close entire terminal.
