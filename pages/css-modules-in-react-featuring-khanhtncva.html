@khanhtncva By creating a CSS file with the .module.css extension, you'll have created a module where you can't access the classes without importing. https://t.co/cHTdGj0hdl

To use the class name, you can default import the "classes" object from the .module.css file, and then use the JSX expression {classes.[insertYourCSSClassHere]} to use the styling!
