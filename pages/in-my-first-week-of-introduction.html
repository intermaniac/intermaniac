Variables and Assignment operators Variables can be considered as containers used to store values. There are some rules to follow to name variables in python and they are; 1. Using ordinary letters, numbers and underscores.

2. We can't use python's reserved words as variable names and spaces are not used So the ideal and pythonic way to name a variable is to use lowercase letters and underscores in place of spaces. To assign a value to a variable, the (=) sign is used Eg: my_height= 50, total_netpay
