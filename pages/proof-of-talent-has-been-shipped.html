Proof-of-Talent has been SHIPPED to testnet 💥🚀 Presenting you: 🍯 Proof-of-Talent 🐈‍⬛ GitHub Credentials 🪙 Minting page and more... 🧵👇🏻 #kleo #klvrs #web3 #ProofOfTalent https://t.co/SAyw2h39Y2

🍯 Proof-of-Talent Work in web3 should not yield to antiquated ways of corporation work but set up its own, on-chain native standards. We have started the building of Proof-of-Talent protocol, and are now launching the early beta in Goerli testnet. https://t.co/ERVA25LRg0

🐈‍⬛ GitHub Credentials As the first concrete step, we allow open-source developers to prove their contributions on-chain. More credentials dropping every week!! 🔜 🤫 https://t.co/qU0OWR83gV

🪙 Minting page You can find all the available Proof-of-Talent credentials via the Badges page. All you need for minting is to connect your wallet and prove ownership to off-chain accounts. Check it out and mint your first badges: https://t.co/chUlS1hkEV https://t.co/YGsfI4T5ma

🗝️ GitHub OAuth Needless to say, you can now link your GitHub profile to your Kleoverse Profile. You can also use Proof-of-Talent contracts to mint your on-chain Identity Badge. This way your platform credentials move with your wallet to any decentralized app. https://t.co/IbuvyZg0RD

🚥 Brand new Skill Chips Besides Proof-of-Talent, many further improvements for frictionless Talent Discovery, too 📯 Check out the newly improved Profiles Page and its new Skill Chips to find just the right talent for the task! https://t.co/T1ix6EhKeL

📝 Bounty Create and Edit Page We just recently updated a New Job form. Now's time for the Bounty Create/Edit page. You can select payment methods from existing payment methods that's been used before. No need to remember long contract addresses or decimals. https://t.co/tGvZCx6O0u

🧿 Coinbase wallet support Last but not least, we added support for @coinbase wallet in Kleoverse. Wallet detections as a whole have been improved as well with an extra fallback support for multi-injected wallets like @brave wallet. https://t.co/orqqG1lq7P
