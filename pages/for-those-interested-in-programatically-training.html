For those interested in programatically training Stable Diffusion to recognize new concepts using Dreambooth w/o GPU's, I highly recommend @rohanarora_'s https://t.co/6nIBgoH54J Here's a quick thread on using Python to train and generate photos for $5/training in less than 30min https://t.co/ttN66iHdFh

First a disclaimer, you can do this all for free via Google colab (they even have free GPU!!) https://t.co/TQz2YTsdPf I was interested in a programmatic solution and didnâ€™t want to bother with setting up a backend service to train/run inference. Thus I settled on Meru

To get started, sign-up for Meru and load it up with at least 3 credits ($5). They have a great docs API and the devs are very responsive via email. https://t.co/KAD2LIzqUG https://t.co/gepP35HW02

Next, take 10-20 photos of your subject. These guidelines from @levelsio are useful references https://t.co/UpiIJCWlwV

If you use Python, you can git clone my repo here - https://t.co/vsfSFNYzRQ Follow the instructions in the README to train and generate your model https://t.co/5cl1jbCF5B

Some useful functions include: upload_images -> uploads images for training Train -> trains model get_images -> runs inference prompt -> generate img prompt_file -> generate img based on file download_model -> download model to specified dir open_images -> opens the image

The model takes about ~20mins to train, and inference for the images via their API is ~80 seconds but this is coming down. You can also download the model and run the inference yourself, which should speed things up. https://t.co/QIpahd1tZy

Hopefully, you found this useful, feel free to comment/DM if you have any questions about Dreambooth/Stable Diffusion/Meru.
