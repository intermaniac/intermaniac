👉 7 Advanced Natural Language Processing with Python 👇🧵

1️⃣ Word2Vec Turns a word into a vector, which is useful for finding distances between similar words. Surprising examples: word2vec('king') - word2vec('man') + word2vec('woman') =&gt; 'queen' See tutorial. 👇🧵 https://t.co/2u2ruOYr54

2️⃣ Tokenization Spitting a text into words. Python standard library does that with split(), strip(), and splitlines(). The NLTK library also does it with word_tokenize() See example step 5 in tutorial. 👇🧵 https://t.co/cUEvUfpXEM

3️⃣ Stop word removal Remove common word that don't carry much meaning. Valuable when making analysis on text. NLTK toolkit comes with stop words. Example shows how to remove them using list comprehension. 👇🧵 https://t.co/JjJFT1LXf7

4️⃣ Lemmatization The process of reducing words to the root form. Valuable when counting how common words are in a text. Example shows how to do it with NLTK toolkit. 👇🧵 https://t.co/Pih48PvVH5

5️⃣ Sentiment analysis The analysis of the sentiment of a text. Useful it determine if a review is positive or negative. See video tutorial as example. 👇🧵 https://t.co/eivc2WVee1

6️⃣ Inverse Document Frequency A measure how common or rare words are across documents. Useful to find a group words that best represents a document. Great for finding relevant documents in response to a query. 👇🧵 https://t.co/SPQ2hzzf3O

7️⃣ Markov Chains A stochastic model describing a sequence of possible events in which the probability of each event depends only on the state attained in the previous event. Example: Given any two words, what is the most probable next word. 👇🧵 https://t.co/mUH7SOCfxJ
