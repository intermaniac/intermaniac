AWS Certified DevOps Engineer Professional Exam Preparation. course code- DOP-C01 A Thread.

This exam (DOP-CO1) is advanced professional level exam. before giving this exam, you must complete any one of the below certification. AWS Cloud Practitioner or AWS Certified Cloud Developer or AWS Solution Architect- Associate 1/n

AWS DevOps exam ask questions from below topic domain. 1. SDLC Automation 2. Configuration Management and Infrastructure as a code 3. Monitoring and Logging 4. Policies and Standard Automations 5. Incident and event responses 6. HA, DR and Fault tolerance 2/n

lets start with CICD- CI stand for continuous integration CD stand for continuous Delivery and Continuous Deployment 3/n

What is Continuous Integration ? When developers push code to repository (Github, gilab etc) and for every code push, code build and test happen automatically and build results are notified to developers. This process is knows as continuous integration. 4/n

What is Continuous Delivery ? This is automated deployment, for every passing code build, code binaries are getting deployed after a manual approval to the application server environments. It ensures software releases quickly and reliably. 5/n

What is Continuous Deployment ? This is fully automated Deployment, there is no need of manual approval, once code build is done, code binaries are getting deployed all the way to production automatically. There is no intervention in code deployment. 6/n

CI tools- Bamboo Jenkins teamcity etc.. CD tools- AWS code deploy Jenkins CD Octopus deploy ANSIBLE towers GOCD etc... 7/n
