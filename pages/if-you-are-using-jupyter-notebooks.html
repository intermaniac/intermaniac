First of all What is a magic command? These are special commands that IPython offers compared to the standard Python shell

Types of Magic Commands Line Magic: Represented by % and applies just to the code that follows it on that same line. Cell Magic: Represented by %% and applies to everything that follows in that entire cell. Ok enough with the intro let's start with the commands

1. %run Running a Python file in a notebook becomes easy. If you have a python file, for example demo .py and want to run it you can do it with this command %run demo .py

2. %%time You can calculate the whole time taken to run the code in the cell using this https://t.co/4Hlidt6376

3. %%writefile Example: %%writefile app .py This will create a new file app .py with the entire content present in that cell

4. %load <file_name> This is a very useful command if you already have a python file with certain functions or code. You can use this command to load them directly into the notebook

5. %pinfo <variable> This allows you to get information about an object. https://t.co/Thaxqo81Sb

6. %history -n We all accidentally delete commands and their results in a notebook. You can use this magic command to display all the previous commands -n will help you get the last n commands

7. %lsmagic The final one, and to see all the available magic commands in the Ipython shell you can use lsmagic. https://t.co/pInyDKF4ko
