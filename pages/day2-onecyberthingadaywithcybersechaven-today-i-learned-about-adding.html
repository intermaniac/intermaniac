#Day2 #onecyberthingadaywithcybersechaven A thread 👇👇👇 Today, I learned about Adding and Removing software in Linux OS As part of growing my proficiency in Linux OS, one of the fundamental skill I need to learn is to know how to Add and Remove software from the machine.

In my today's Study, I was about to learn about the 3 methods used to add new software to the Linux machine. These are: 1. APT ( advanced packing tool) 2. GUI-based installation manager 3. GITHUB

1. USING THE ADVANCED PACKAGING TOOL (apt) Apt is the default software manager in Debian-based Linux distributions which are Kali and Ubuntu This software manager can be used to achieve the following

a) searching for packages - by using the command “apt-cache nameofthepackage” they used the word package because most software on Linux, are not standalone, they need a group of files typically libraries and other dependencies to run efficiently.

b) adding software - the apt manager can be used to add software by running the command “apt-get install nameofthepackage” https://t.co/rgbvaVq61k

C)removing software - It can be used to remove software by using the command “apt-get remove nameofthesoftware”.

However using this command does not remove everything about the software, so if you want to remove both the software and its configuration files you are to use the command “apt-get purge nameofthesoftware”.

finally, on the removing commands, if you want to delete the whole package which includes the libraries and dependencies that are required for the software to run efficiently then the auto-remove command should be used like this “apt autoremove nameofthepackage”

D)updating packages - the apt command can be used to update the packages in the Linux repository by running the command “apt-get update”. This command will update the repository with new software or a new version of the existing software.

E)Upgrade packages, the apt command is also used to upgrade existing packages on your OS by using the command “apt-get upgrade”.

-Adding repositories to the sources.list file: I also learned how to add repositories to the source.list file, this comes in handy when a particular software package is not in the repository, that is in the source.list of your Linux distribution

This can be done by opening the file in a text editor, typing in the command, and then save it. I used the leafpad text editor by running the command“leafpad /etc/apt/sources.list”.Other text editors can be used to carry out this task such as command line text editor “nano”. https://t.co/dk1M38MDbv

2. Using GUI I also learned how to use a GUI- based installer “Synaptic” to install software on the Linux machine. https://t.co/P8GvvRsnNp
