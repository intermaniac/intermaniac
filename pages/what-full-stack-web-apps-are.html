What?! Full stack web apps are easy now? All using #TypeScript?! I found the gateway 💊 drugs into full stack, let me share the tools I am using for my side project. 🧵 👇

Prisma (https://t.co/0vu2Eh07dR) is a typesafe ORM with a great API? 😲 It autogenerates types? Prisma writes types based on your schema directly to `node_modules`. Madness, but I think this trend will spread!

Not only is Prisma great by itself, run `npx prisma studio` (https://t.co/iHgeK1Hoer) and you can instantly view, edit and view your schema like a spreadsheet. 🏃 I was able to experiment and quickly prototype the right schema for my project, and verify data easily. Amazing.

I want to use Prisma as part of an opinionated framework with batteries included. I am here to build—not configure dependencies. 🌩️ https://t.co/qT3YYJnIrR is the batteries included framework that is blowing my mind—build on top of @nextjs! ✅ Developer experience ✅ User auth

I 👏 am 👏 not 👏 writing 👏 a 👏 single 👏 type 👏 definition. @blitz_js allows me to have a function that gets data from my database using my typed Prisma client—and lets me import that backend function in my front-end code?! They use https://t.co/mbQ8POaxZT btw

... I still need to host my database. I know very little about databases. 🎩 @planetscaledata is exactly what I need. ✅ Unlimited scale. ✅ Works with Prisma. ✅ Insane DX. ✅ Ex-Segment folks work there. 👋 MySQL is cool now?! They got a dope hat?!! https://t.co/FM0p1Ko23Y

At this point I got my dream stack for my side project, I can focus on building. 😔 ...oh wait, I need a stateful server for a job I can’t do with serverless alone. 😮 https://t.co/YMDloppzKB — something a front-end developer like me can use. A modern day Heroku!

I got most pieces, and I can focus on product... but I don’t want to reinvent the wheel. 🔥 https://t.co/RW8StfPP0n is the UI framework I found that is the modern day version of Evergreen UI I build 5 years ago. ✅ CSS-in-JS with theming (cx prop) ✅ Good components + docs

This is my dream stack to work on side projects! ✅ Language = @typescript + @reactjs ✅ Framework = @blitz_js on top of @nextjs ✅ ORM = @prisma ✅ Database = @planetscaledata ✅ Hosting = @Railway and @vercel ✅ UI = @mantinedev
