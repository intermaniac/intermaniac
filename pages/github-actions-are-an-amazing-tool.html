GitHub Actions are an amazing tool that lets us do things like: 👉 Lint code 👉 Run tests 👉 And much more Let's build our first one and take advantage of this great tool for free 👇 🧵

1️⃣ .github folder In the root of your project create a .github folder and then within that folder create a folder called workflows. Inside of .github/workflows is where we will put out our GitHub Actions. Now create a file inside of workflows called build.yml https://t.co/UHObGoT66b

2️⃣ Naming our action & triggering it The first part of our action is the name, this is simply what we want the Action to be called and what GitHub will label it throughout our repo. Then, use the 'on' keyword to specify when to trigger the action. E.g. 👉 PRs 👉 pushing code https://t.co/ig8mhM4p4g

In our example, we will trigger the Action on pushing code as well as any pull requests. But, there are loads more options, check them out below 👇 https://t.co/BDk9Ok24JW

3️⃣ Jobs - part 1 For our example, we will build a GitHub Action that runs a lint script and our tests. For this to work, in our repo, we need scripts defined in our package.json to do these functions. This way GitHub will install our packages and these scripts.

3️⃣ Jobs - part 2 Let's look at adding our lint action. Here we define our lint job and the steps it needs to run within it. This includes: 👉 Checking out the code 👉 Setting up node 👉 Installing our dependencies 👉 And, finally, running our lint script. https://t.co/YULUqcWl4e

3️⃣ Jobs - part 2 Now, to add our tests job, we can do the same steps as before but change the final step to run our tests instead of linting. 👇 https://t.co/saBW07Xowv

4️⃣ Running our actions After pushing our actions to our repo, when we push future code or create a PR, GitHub will automatically run our defined actions. For PRs, we can check on the progress by using the 'checks' tab on a PR. Like below 👇 https://t.co/HhHAsvdZrn

Congrats, you just created your first GitHub Action 🎉 But, this is just the tip of the iceberg, GitHub Actions are powerful tools and can do amazing things. Check out some of the amazing things you can do and find out more about Actions here 👇 https://t.co/k8QCjuGGvP
