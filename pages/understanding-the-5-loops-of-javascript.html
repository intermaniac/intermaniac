📌 understanding the 5 loops of javascript 🧵👇  1. https://t.co/gdpgsinyhm

loops offer a quick & easy way to do something repeatedly 🚀 the 5 loops in javascript essentially do the same thing: — they repeat an action a certain number of times. however, they have important differences 💡 let’s dive in! ⚡️

javascript for loop repeats a block of code as long as a certain condition is met. typically, used to loop through a block of code a specific amount of times 💡 https://t.co/ozgolwhrm3

2. javascript while loop loops through a block of code as long as the specified condition evaluates to true. as soon as the condition fails, the loop is stopped. https://t.co/yspnbwufek

3. javascript do…while loop the do…while loop is a variant of the while loop, which evaluates the condition at the end of each loop iteration. with a do…while loop the block of code is executed once, and then the condition is evaluated 💡 https://t.co/ysjdyoetfx

4. javascript for…in loop the for…in loop allows programmers to iterate over the properties of an object ⚡️ https://t.co/sq14fuwlar

5. javascript for…of loop the for…of statement creates a loop iterating over iterable objects (including array, string, set, and so on). for…in iterates over property names 💡 for…of iterates over property values 💡 https://t.co/q4cxmnqi1h
