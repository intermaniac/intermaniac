SCSS (CSS3) TUTORIAL ON RESPONSIVE WEB DESIGN WITH @MIXINS AND @INCLUDE! this thread will be focused on making RWD with SCSS very easy and simple to understand. #webdesigntips #webdeveloper #webdesign #responsivewebdesign #tutorial #css #developer #css3 #newbie #100daysofcoding

2/n If you had followed my TL in the past months, you must have observed I bumped into SCSS while on my tech journey. It's been fun so far, and I am excited to share what I have learned in an easy way to help newbies in web development.

3/n SCSS, put in total words to me, could be Super CSS. this tool comes with great functions, and we're going to use two functions, mixins and include.

4/n This tutorial will declare a variable, breakpoints, as in the image attached to this tweet. These breakpoints contain pixel screen sizes with various variables assigned to them. https://t.co/zrYftGlTVE

5/n Next, we call the mixin function. It will contain the media query we want to attach to the breakpoint variables. We'd use a map-get function in our media query. Why? Because this is Super CSS, lol. It enables the function to read from our first variable, breakpoints. https://t.co/xX0mqHEcuq

6/n Since we have about five breakpoints, we'd have to write five media queries. It looks like stress, but actually, we're getting to the interesting part soon! Stay with me. The @content in the media query represents what will be inside the mixin function whenever it's called. https://t.co/Xd1vymqww6

7/n In the HTML file, we really do not have much to do. We only provide an element so we can have something to manipulate from the styles. https://t.co/1NXwY8oyGV

8/n Now, how do we use what we've discussed so far? We call the function @include and pass the mixin variable name we declared earlier. Notice how we have called them? inside where we needed them. That's how SCSS works! https://t.co/yGfiKjQHjV

9/n These functions saved us the time of writing and repeating ourselves and made it easier just to include the media query instead of repeating @media and screen each time we needed a part of our web app to be responsive. (this image below is the compiled css from scss) https://t.co/BK8W1Cgwl8

10/n You should try it out, and this will help you in creating responsive web apps (me too! lmao). So yes, this is basically how I've understood SCSS, and I hope I helped someone! Please be kind to like and share if you found this useful. Thank you! https://t.co/2KBKl79hcq
