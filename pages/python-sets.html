Python Sets..The complete thread Set properties to be discussed - Mutability - creation - insertion - deletion Mutability: is the property of any Data structure that has to do with changing its contents #python #code #backend

Python Sets - Use case : used in storing unique data types.. - Mutability : python sets are immutable i.e you can't change it's content after creation - creation : a python set can be created using the built in set() constructor new_set = set() It can be created manually

myset = {"apple", "banana", "cherry", True , [1,2, 3], } It can store any python data type - Insertion : This might get confusing, cause I already said it was immutable (unchangeable).. To best understand this you can simply use the dir() function to check methods available.

You can add items to a set by using the add() method my_set = {'jude', 'john' } my_set.add('philip') This would add philip to the set Sets are unordered so Philip won't necessarily be the last item on the set update method adds an entire set to another set #python #code

- deletion : deleting a item of a python set can be done using the pop() method The pop method deletes and return the deleted item The item to be deleted must be specifiied between the parentheses If item doesn't exist an error is raised #python #dev #backend #basics

Deletion of sets item can also be achieved using the remove() method The remove method works exactly like the pop() method but if the item to be deleted doesn't exist the remove () method doesn't raise and error it handles it #python #programming #code #tech #coding #sets
