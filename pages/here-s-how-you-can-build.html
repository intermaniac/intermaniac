The Python Modules you need: 'sys': For fetching terminal arguments 'os': For creating and accessing files & folders 'Image' tool from 'PIL': For converting the image format

STEP 1: Import the above-mentioned modules https://t.co/e08ObLoEwO

STEP 2: Grab the folder name arguments from the terminal command The way this tool works is, the user writes the source folder name (that contains JPG files) and the destination folder name (where PNG files will be stored) in the terminal. To fetch them, we use sys.argv

sys module stores the terminal command as List items such as for "python3 JPGtoPNGconverter JPG PNG", JPGtoPNGconverter will be list item [0], JPG will be list item [1], and PNG will be list item [2]. JPG and PNG are the folder names of the source & destination we need to fetch https://t.co/nE9hEguWRI

STEP 3: Create a Destination Folder in case it does not exist There may be a chance that the user did not create a folder to save PNG files. So we will create one with the same name he wrote in the terminal.

We will check if it exists in the same folder where the command was run by using os.path.exists(<destination folder name>) which will give us a boolean and if it gives us false, we will create one by using os.makedirs(<destination folder name>) https://t.co/RoEnVQtxfZ

STEP 4: Iterate through source folder, access image files, convert and save in destination folder STEP 4.1: We cannot simply write the source folder name as the for loop will iterate through the name itself instead of its file content. We need os.listdir(<source folder name>) https://t.co/GtVdcwctLV

STEP 4.2: Now inside the loop, we access the image file by using the Image tool's "open()" method and pass the path to the image file as a string. https://t.co/azlgMlAVhB

Now we can use the "save()" method of Image tool to convert and save the image files in destination folder BUT it will show the name of the saved files as penguin.jpg.png instead of penguin.png because we are passing image names with .jpg extension

STEP 4.3: Fetch the name of the image and remove .jpg from filename For this we again need to use 'os' module. "os.path.splitext(imgFile)" method will turn the name into a list where, penguin.jpg will become ["penguin", "jpg"]. We only need the first item ( [0] ) of such a list https://t.co/F6lahi7KG9

STEP 4.4: Finally convert to desired format and save the image in destination folder For this we use the "save" method of Image tool where we pass the image path along with image name with .png extension as a string. And the second argument is "png". https://t.co/BGdY8m1JbE

It's done. Now all you need to do is pass the command and run the file in the terminal. Make sure that the name of Source Folder that consists JPG files is correct and that it exists in the same folder where you are running the converter .py file with python3 command. https://t.co/IbwUhZjEa3
