Difference b/w Class And Functional Components In ReactJS A thread👇🏻🧵

📍Syntax- Class components have the syntax of JS classes. It inherits React.Component and render() method which returns HTML. It sets states using ‘this’ and ‘super’ keywords. Functional components are JS functions. It has a return() that displays JSX expression or raw HTML code

📍Passing Props In Class components, props are passed in the constructor. It can be declared using constructor(props){ super (props) }. Functional components accept props as the only argument, so you can pass properties as props while declaring the function component.

📍Managing State– In Class comp. a constructor assigns the initial state. Once the instance is in memory, the properties are created and read. In Functional comp. useState() hook is used which consists of the state variable, function that updates it, and initial state variable

📍Life Cycle Methods Class comps. use componentDidMount(), componentDidUpdate() and componentWillUnmount() functions to maintain its life cycle. Functional comps. use useEffect() hook as it's similar to the life cycle combined and is responsible for the life cycle of components.

📍Mutability- Class comps. objects are mutable. If the state of the object change then it impacts its original state. Functional comps. objects are immutable. If the state of the object change then it never impacts its original state but makes a copy and changes the state.
