Yesterday, I had an extremely fruitful discussion about GitHub Copilot's potential license violations with the developer audience at @MSFTReactor Berlin. Here are my insights and learnings üëá üßµ

If you don't know what I'm talking about: GitHub Copilot scans all public repos and learns from it, regardless the repo's license. In very specific cases, the generated code can come very close to public code that is not meant for using in private projects.

I already had a long discussion with some folks on Twitter about that. https://t.co/bMgdqQvT4X

So what were the outcomes of yesterdays discussion? 1. Some non-permissive licenses demand that work that derives from the original work to be made publicly available. There was a consens that GitHub Copolit "derives" from that work.

2. It would be great, if generated code that is very similar to an entry in the training set would attribute the author. üëâ Alexander Wachtel said that tracing back the source is nearly impossible.

3. As Copilot is made from the community's Open Source work, it would be the right step to also Open Source Copilot. This would at least give me the option to to host an instance of it myself. üëâ Hosting that huge model myself is unlikely but having the option is what counts.

4. Actually, it's very similar to Grammarly or DeepL, which also learn from public books and blog posts that without attributing the author. Or even Google, which also crawls all websites. üëâ They are not Open Source either. Why don't we have the same discussions there?

5. It's hard to say if generating a few lines of code that are (for an obvious reason) very similar to some lines in a public repo is already violating the license. üëâ Developers had different opinions here. That's what lawsuits will decide.

6. Repo owners would like to have the option to opt-out of being scanned by Copilot or want to have a share of the earnings. üëâ Interestingly, they don't mind humans reading their code and don't want to participate in hosting the service.

üôè I would love have a lot more discussions about that in the future and am super thankful for yesterdays open mindset discussions, where nearly everyone had an opinion on that topic. What's yours?
