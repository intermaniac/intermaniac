For Newbies, especially those in @cybersechaven community and those following @ireteeh If you are having challenges installing kali Linux on Virtual box or Vmware...Kindly follow this thread. I will be writing about some of these challenges and how they can be tackled.

If you are getting this type of error message, This is what you should do: When building the virtual machine do not allocate physical storage hard disk dynamically, Choose the fixed option and allocate between 16gb-20gb based on the amount of storage you have on your computer. https://t.co/jBCOAXzOG5

For this type of error message: Boot your computer into the startup menu and choose BIOS setup,then go to Advanced and select system options Activate VTX and VTd, Save changes and restart your computer.. The error message will be cleared. https://t.co/gfQRSFQx8P

For this type of error: When building the Kali Linux VM, make sure the bit version you choose is the same with the bit version of the ISO file you have downloaded. If your ISO file is 64-bit , the version you select must be 64-bit.. A 32-bit CPU can not run a 64-bit Machine https://t.co/VjRuFlcgF1

For Mac user: If you encounter this error when installing virtual box, I will post videos of what you should do in the thread.. https://t.co/cZq5feTDbd
