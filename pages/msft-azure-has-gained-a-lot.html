$MSFT Azure has gained a lot of attention over the past 5 years, now IC making up to 37% of revenue. Thread sharing everything about this incredible component of Microsoft's thesis: https://t.co/RvFXtoaYpV

Azure is a comprehensive set of cloud services. It offer developers, IT professionals, and enterprises, freedom to build, deploy, and manage applications on any platform or device.

Microsoft has its own datacenters distributed around the world, making possible for Azure to offer Public Cloud services and also counts with Hybrid & Private ones. Solutions cover the whole spectrum of cloud usage, allowing Azure to offer IaaS, PaaS & SaaS. https://t.co/BRdMgSTlCe

Pricing varies from product to product. Azure's mainly sells through a Consumption-based Model, meaning you pay for the resources you use. Secondarily, some products count with fixed subscriptions for predictable workloads and Spot subscriptions, for interruptible ones. https://t.co/hPwnzg8IYm

Azure counts with more than 200 products, with more than 40 solutions/use cases, and covering a wide variety of verticals. https://t.co/CEss2j8Ive

Azure VM is computing power as a service. Azure Virtual Desktop runs on the cloud and on any modern browser. Azure SQL is a database management system. Azure Cosmos DB is for modern app development. Azure Arc helps building apps and services. Azure ML is for making ML models https://t.co/3Qpqpgv62b

The final component is Azure Marketplace. This is an authentic marketplace for software apps. In it, you can find, try, buy and deploy software apps needed to build and manage cloud infrastructure. The catalog also includes consulting services from Microsoft Partners. https://t.co/WcC2bCc7Ap
