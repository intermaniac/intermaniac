What are media queries in CSS? Here is a thread crash course 🙂 A thread🧵 https://t.co/2CFKSk4KzQ

A media query is mostly composed of an optional media type and a number of expressions called media query feature expressions. These expressions can be combined via logical operators. I introduced two things above 👆🏾 ✅ Media type ✅ Media features Let’s break them down

Media type 👉🏾 you use this to define the category of devices for which the media query you are authoring will apply. The types are mostly all, print, and screen. You can omit the media type because it will default to all.

Media features 👉🏾 This is used to describe a specific characteristic of the user agent e.g the browser. Common examples of media features are device-height, device-width, prefers-color-scheme(think dark and light mode).

For example, the width feature allows a media query to test against whether the user-agent is of a particular width and apply some styles at that constraint.

Let's touch on Logical operators. Logical operators are used to composing complex media queries. These operators are: not, and, and only. P.S: You can combine multiple media queries in a single rule by using commas to separate them.

How does a media query evaluate? So when you set a media query and the media features expressions, the styles in the media query will only apply if the conditions set in the media features evaluate to true i.e a device is matched.
