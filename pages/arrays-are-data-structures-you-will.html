.concat() Use this method to merge two or more arrays. https://t.co/G7M45sPQOa

.at() This method takes an integer and it lets you get an item in an array via its index. It also takes in negative integers that let you count back from the last item in the array. https://t.co/8AjZJGngdY

.filter() This method filters down elements in a given array that passes a test provided by a function. https://t.co/IZu8vjizhH

.find() Returns the first element in an array that satisfies a given testing function. P.S: If no values satisfy the testing function, undefined is returned. https://t.co/49tbF0DKYk

.findIndex() Returns the index of the first element in an array that satisfies a given testing function. If no elements satisfy the testing function, -1 is returned. https://t.co/JYHiygFD2s

.join() This method creates and returns a string by concatenating all of the elements in an array, separated by commas or a specified separator string. P.S: If the array has only one item, then that item will be returned without using the separator. https://t.co/lGoDRTe9fB

.reverse() Reverses an array in place and returns the reference to the same array where the first element now becomes the last and the last now becomes the first. P.S: .reverse() is destructive because it changes the original array. https://t.co/98CivDiWhi

.every() This method tests whether all elements in an array pass the test implemented by a provided function. It returns a Boolean value i.e true or false. https://t.co/0yCgIywObL

.some() This method tests whether at least one element in an array passes the test implemented by the provided function. https://t.co/w90IOadBCJ

.map() Creates a new array populated with the results of calling a provided function on every element in the calling array. https://t.co/6SO9NecRw2

.includes() This method determines whether an array includes a certain value among its entries, returning true or false as appropriate. https://t.co/hoLYuOsYTs

unshift() Adds one or more elements to the beginning of an array and returns the new length of the array. https://t.co/uiBCEWLJA6

shift() The shift() method removes the first element from an array and returns that removed element. This method changes the length of the array. https://t.co/koAcuiqj3X

.push() Adds one or more elements to the end of an array and returns the new length of the array. https://t.co/b6eVnnY37C

pop() Removes the last element from an array and returns that element. This method changes the length of the array. https://t.co/4zIeZGN08z
