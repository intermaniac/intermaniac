⑤ justify-content ⇝ defines the flex-items alignment along the main axis. {justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly} https://t.co/qjdSc24bKf

⑥ align-items ⇝ defines how flex items are laid out along the cross-axis on the current line. {align-items: stretch | flex-start | flex-end | center | baseline} https://t.co/mMHZBL7cu8

⑦ align-content ⇝ aligns a flex container’s lines when there is extra space in the cross-axis, similar to how justify-content aligns individual items within the main axis. https://t.co/vFQe7pAABy

⑧ gap ⇝ explicitly controls the space between flex items. It applies that spacing only between items not on the outer edges. {gap: row-gap, column-gap} 𝗡𝗼𝘁𝗲: It is not for flexbox only, gap also works in the grid and multi-column layout

📌 [𝗖𝗵𝗶𝗹𝗱𝗿𝗲𝗻 - 𝗙𝗹𝗲𝘅 𝗜𝘁𝗲𝗺𝘀] ① flex-grow ⇝ defines the ability for a flex item to grow if necessary. It dictates what amount of the available space inside the flex container the item should take up. {flex-grow: 2} ⇝ The value is unitless & the default is 0 https://t.co/yBXRFVWWS2

② flex-shrink ⇝ defines the ability of a flex item to shrink if necessary. {flex-shrink: 3} ⇝ Negative values are invalid & the default value is 1 https://t.co/dMde9DSaGl

③ flex-basis ⇝ specifies the initial size of the flex item before any available space is distributed according to the flex factors If set to 0, the extra space around content isn’t factored in. {flex-basis: % or absolute value | auto} ⇝ default is auto https://t.co/XGuhua16zO

④ flex ⇝ It is the shorthand of [flex- grow | shrink | basis] The default value is [0 1 auto] as per the default values of all the 3 properties. {flex: none | <'flex-grow'>, <'flex-shrink'>, <'flex-basis'>}

⑤ order ⇝ flex-items are laid out in the source order. However, we can control the order of each flex item with the help of this property. .flex-item1 { order: 5 ⇝ will change the order of the first flex-item to the 5th item } Default value is 0;

⑥ align-self ⇝ set the alignment for an individual item {align-self: auto | flex-start | flex-end | center | baseline | stretch} The default value is auto;
