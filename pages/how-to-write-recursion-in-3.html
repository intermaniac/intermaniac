Step 1: Recursive case - the flow Step 2: Base case - the stopping criterion Step 3: Unintentional case - the constraint

Let's look into it with an example. Problem: Find the factorial of the given number. Factorial is the product of all positive integers less than or equal to a given positive integer. i.e. 4! = 4 * 3 * 2 * 1 = 24

Step 1: Recursive case - the flow To identify the recursive case we need to find how the factorial is defined using the factorial function. General expression of factorial is n! = n * (n-1) * (n-2) * ...2 * 1 Can be written as n! = n * (n-1)! Now we got the recursive flow! Code: https://t.co/g7ZhlcCkVk

Step 2: Base case - the stopping criterion In the above code, we can clearly see that the function keeps calling itself without stopping. To prevent the infinite loop of recursion we need a base case. In our case, 0! = 1 1! = 1 So as a base case, we can use these cases. Code: https://t.co/NpuU3dUHGB
