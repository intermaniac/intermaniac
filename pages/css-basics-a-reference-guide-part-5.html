🔶 CSS BASICS ( Part 5️⃣ ) 🔸 Margin 🔸 Padding 🔸 Outline 🔸 Height/Width

🔸Margin: The margin property defines the outermost portion of the box model, creating space around an element, outside of any defined borders. Margins are set using lengths, percentages, or the keyword auto and can have negative values.

CSS Margin cheatsheet here 👇 https://t.co/bQRaJdpwyy

🔸 Padding: The padding property in CSS defines the innermost portion of the box model, creating space around an element’s content, inside of any defined margins and/or borders. Padding values are set using lengths or percentages, and cannot accept negative values. https://t.co/SQDhst26bJ

🔸 Outline The outline property in CSS draws a line around the outside of an element. https://t.co/CFzhaDcQE6

It’s similar to border except that: 1. It always goes around all the sides, you can’t specify particular sides 2. It’s not a part of the box model, so it won’t affect the position of the element or adjacent elements (nice for debugging!) https://t.co/58ewcV1kEw

🔸 Height/Width The height and width properties are used to set the height and width of an element. The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element. https://t.co/HxDcW3DXn9
