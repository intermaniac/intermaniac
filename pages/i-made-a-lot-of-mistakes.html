I made a lot of mistakes when I started with Wordpress... My sites were getting hacked, they looked terrible and I really had no idea what I was doing... Here's 7 Things every WordPress developer should know, so you don't end up like me. ðŸ§µðŸ‘‡

1. WordPress Core: Every WordPress Developer should have a good understanding of the WordPress Core - from the structure of the CMS to the specific of core functionality.

2. Themes & Plugins: Knowing how to work with WordPress themes and plugins - from building and customizing them to troubleshooting any issues - is a must for any WordPress Developer.

3. Content Management: Understanding how to effectively manage and update content in WordPress is crucial for any WordPress Developer - Custom fields and post types as well as backend data management are crucial.

4. Security: Keeping your WordPress site and its data secure should be a a top priority, as well as implementing any security measures to ensure the site is safe and secure. Knowing how to use security plugins is a good start, but backend security is vital too.

5. Web Standards: Working to create websites that follow web standards and best practices goes a long way to ensure visitors have the best possible user experience and your site stands a better chance of ranking well in search engines.

6. Search Engine Optimization: As a WordPress Developer, you should have a good understanding of SEO and associated techniques for maximizing visibility for WordPress sites in search engine results. Not just how to use an SEO plugin, but the goals of SEO

7. Mobile-Friendly Design: More and more users are accessing the web via their mobile and making sure your sites are mobile friendly should be one of the top priorities. Managing mobile / desktop design is important for all WordPress sites.

BONUS: You also need to know how SSL works and how to manage a hosting server. Things like MySQL management and Crons are important as you become more advanced with WordPress too.
