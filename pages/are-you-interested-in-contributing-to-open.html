Are you interested in contributing to open source projects but confused how to find out projects? Here’s a detailed thread on the different ways in which you can search for open source projects on GitHub (even for beginners): https://t.co/WmmiaINam5

Different ways: 1.Explore tab https://t.co/E9d722muDW box on GitHub 3.Trending tab 4.GitHub Topics 5.Projects from your social network https://t.co/6jQV4gTdVe

1.Explore tab: -Go to https://t.co/SsrJgUyrvM, here you will find many open source projects based on your interests. https://t.co/0V5e3xKQmi

2. Search bar on GitHub You can directly search for any technology/ skill that you want to practice in the search box. EG: If you want to practice React, just search for React & you will get plenty of projects related to React but you can’t just select any project. https://t.co/Lr2TnbBO3D

-Criteria used to select a project out of a list of projects: a.The most important criteria is the project should be active & it’s community should be interactive with the contributors. https://t.co/bhYLNeyitt

EG: The project in the screenshot has recent changes (i.e. 5 hours ago, 21 hours ago, yesterday) which means it is active. https://t.co/llJgc06amI

To check whether a project’s community is interactive or not: Just check their closed Pull requests to analyze their conversations with the contributors.

Because if you are a beginner, you need guidance from the community & if no one is helping you out or not responding or not accepting your PRs, then what’s the use of contributing to that project, right? https://t.co/kKESbZDixh

b.The 2nd criteria is optional but can be considered i.e., the number of stars in a project: So, now what is star? Star is used to bookmark a project for future, so if people are interested in a project, they book mark that project. https://t.co/7mBbc6cCpR

That means, more the no. Of stars a project has, the more no. Of people are interested in that project. The red circles in the screenshot depicts the no. Of stars a project has, so you can go for a project that has more no. Of stars. https://t.co/sNP5YPhfJl

c.More no. Of issues can also be considered as a criteria but it is also optional. Every project has some defects/issues, which can be found in the Open issues tab of that project. So, the more no. Of open issues, the more you can contribute to that project. https://t.co/Slj7bA8eAb

3.Trending tab Just go to https://t.co/1pYOrKiJBu on GitHub & here you will find some projects/repositories & developers who are trending/most popular. https://t.co/R8IrtbCnyq

4.GitHub Topics Just go to https://t.co/Db6sNSEf5O & here you can find projects based on subject areas. EG: Python, C++, React, CSS, Hacktoberfest etc. Are all topics You can either search any topic from “All featured topics” or popular topics from “Popular topics”. https://t.co/aZQIjKGWvU

5.Projects from your social networks You can find open source projects from social media platforms like Tech Twitter & LinkedIn. You can just search & go through the profiles of the people in your network to find out on which projects they are working on. https://t.co/GHs8bfw7cB

That's a wrap for this thread. Thanks for reading !!
