A Short Introduction to TypeScript 💙 🧵👇

TypeScript (TS) is a superset of JavaScript (JS) - meaning: any JS code you write is already valid TS code (not necessarily error-free, though 😄). The TypeScript compiler turns TypeScript into JavaScript. So what's the point if it's turned into JavaScript anyway? 🤔 https://t.co/x0QUqqT83A

The TypeScript compiler analyses your code and creates a map describing how all your code is related - based mainly on the types you provide. This analysis is called Static type-checking. If you look carefully at the code below, you'll notice that it's 🤏 close to 100% JS. https://t.co/jIYopDqPyi

Yes, the extra "name: string" does all this magic - and the TS compiler, of course. "string" is one of the Primitive TS types. There there are three in total: 🔹string 🔹number 🔹boolean There's also an Array to combine all these and a particular type called "any". https://t.co/BJ0bt9yo7J
