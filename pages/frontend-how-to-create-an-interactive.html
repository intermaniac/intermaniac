This relatively easy-to-achieve effect can make your #frontend project just that bit fancier. There's a reason it's being used a lot on award-nominated sites üòâ. https://t.co/mV9kU3fCrA

Let's start with the #HTML/#CSS. An element with two div's that represent the circle and the dot. The position of these mouse elements is set using CSS variables. Notice the var(--mouse-x). https://t.co/eVtRnwdf3n

Then a mousemove listener stores the mouse position as #CSS Variables. Making the mouse move. Be sure to set the variables only on the mouse element since changing them will force a repaint. If you set them on the body the whole page will be repainted causing performance issues. https://t.co/hNbrH46dB9

I made a demo on #CodePen to give an impression of how this can be used. I used the mix-blend-mode exclusion to be sure the cursor is always visible. https://t.co/qFPczzMZHL
