📌 Daily #Roblox scripting thread 6! ✨ 🔁 FOR-LOOPS #️⃣ 👇 A #RobloxDev thread!

🔁 The for-loop is used to ITERATE a certain number of times. 🤔 But what does "iterating" mean? ♻️ It means for the loop to "run" or "refresh"

2️⃣ There are two use cases for the for-loop 🔁 The first one is to loop, or iterate, a set number of times. 👇 This is how:

⌨️ Type "for i =" followed by three numbers separated by a comma. 1️⃣ The first one is the START VALUE - the variable i will be equal to this number when the loop starts.

2️⃣ The second one is the GOAL VALUE - the loop will repeat until the variable i is equal to this number.

3️⃣ And the third one is the INCREMENT - Every iteration, the variable i will be increased by this value. ❓This value is optional and defaults to 1. Here's how it all looks 👇 https://t.co/XKNA0PlCyn

📝 Please note that "i" is a variable, meaning you can call it whatever you want. It's also only readable INSIDE the loop!

️⃣ The second use case of the for-loop is to ITERATE over a TABLE. Here's how that looks 👇

⌨️ Type "for i, v in Table do", replacing "Table" by the array or dictionary that you want to loop through. Like this 👇 https://t.co/BysqGn7vPk

What this does is that it runs ONCE for every index/value paid in the provided table. 🤔 The variables "i" and "v" are equivalent to the index and value respectively. 👍

💂 A really good way to remember this is to translate it into English: 👉 "For each index and value pair in the table, do this:"
