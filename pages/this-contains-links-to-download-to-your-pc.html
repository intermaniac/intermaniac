This thread contains links to download to your PC or Mac the software you need as a Data Analyst. Microsoft Excel, SQL, Power BI, Tableau, Anaconda. Kindly like and retweet

Microsoft Excel: This should come pre-installed if you use a Windows laptop, but if it doesn't or you'd like the latest version, download via this link:https://t.co/yg38rrOwfs Watch this video to see how to install it: https://t.co/webyJrIDZZ

SQL: There are different RDBMS' you can download to use SQL. You can install the MySQL Workbench for PC or Mac here: https://t.co/7H8ukzZS7N or download PostgreSQL here: https://t.co/8NbtWDMMmQ Watch how to install them here: https://t.co/q7JewRZ11C

Microsoft Power BI: Unfortunately this is only available in the Microsoft Store for Windows PC. For Mac users, you can use the online web version or stick to Tableau. To download on Windows, visit https://t.co/zEkSkwYyz1 it should redirect you to the Microsoft store to download

Tableau: Tableau comes in two packages. Desktop and Public. Tableau Desktop isn't free. To download the free Tableau Public visit: https://t.co/CKlOtKb0VG

Anaconda: This is a package that makes it easy to use Python and other Data Analytics and Science tools like Jupyter Notebook, Visual Studio code etc. Download it here: https://t.co/0pwm7Ppw0g Learn how to install it here: https://t.co/5w9eRWV15A
