Filter by Most Recent git log -n 3 : This command returns a list of the three most recent commits made to a repository. Filter by Author or Committer git log --author="John smith" : It will shows commit which done by John smith . git log --committer="John smith"

Filter by Date git log --before="2022-11-05" : It will retrieve a list of commits from before 2022-11-05 git log --after="2022-11-02" --before="22022-11-08" : It will retrieve a list of commits that were published after 2022-11-02 and before 22022-11-08.

Filter by Message git log --grep="update" : It will return a list of all commits whose name contain "update". Filter by File git log -- https://t.co/qV1wNAzoRu">https://t.co/qV1wNAzoRu : It will shows the changes made to the "https://t.co/qV1wNAzoRu">https://t.co/qV1wNAzoRu" file in our code

Filter by modify File git log --patch : The git log patch command displays the files that have been modified. It also shows the location of the added, removed, and updated lines.
