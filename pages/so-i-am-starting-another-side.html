So I am starting another side project where I will be benchmarking different Wordpress hosting providers. Mainly shared ones for now. A short thread on what tests I plan to run and why.

Basically I am looking for a good hosting provider for a few projects. In my past experience I have always been unhappy with them. I usually prefer to host everything myself since I have the experience and time. Well time is not always available anymore, so I need alternatives.

List of tests I plan to run so far are:

Create post/delete post using a simple php generated page with code from God knows where. I created it like 5 years ago. It will be measuring time it takes for the server to create and then delete the post.

A more advanced load testing using a collection of bash scripts I developed. This will determine if the host has some fancy protections that do more harm than good. It depends mostly on my network speed, but in general it will catch different cases with problems.

Load test with a 3rd party tool that simulates traffic. For now I have chosen https://t.co/J1Xdl54Df9 for this job. The free plan is limited to just 1 minute durations, so I will be looking for a replacement or will just buy the thing to use it properly.

Load test with some Wordpress plugin just for the fun of it. Not sure which one yet.

Since I will be paying for these accounts anyway I plan to keep them going for at least a month and monitor their uptime with https://t.co/GLRmBfP5N6

End goal is to make a list of reliable hosts. I know there are plenty of services who report stats like this, but let's be honest - most are flawed or skewed one way or another.

I think it will be very useful for people like me who dont have the time to research all hosts.
