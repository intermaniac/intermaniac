5. forEach: This method executes a callback function on each element of an array. The callback function takes three arguments: the current element, the index of the element, and the array itself. #learning

For example: let arr = [1, 2, 3, 4, 5]; arr.forEach((element, index, array) => { console.log(`Element at index ${index}: ${element}`); }); // Output: // Element at index 0: 1 // Element at index 1: 2 // Element at index 2: 3 // Element at index 3: 4 // Element at index 4: 5

6. reverse: This method reverses the order of the elements in an array. For example: let arr = [1, 2, 3, 4, 5]; arr.reverse(); // [5, 4, 3, 2, 1] #NodeJS

7. join: This method concatenates the elements of an array into a single string, with a specified separator between each element. For example: let arr = [1, 2, 3, 4, 5]; let str = arr.join(', '); // "1, 2, 3, 4, 5" #LearnToCode

8. map: This method creates a new array by calling a callback function on each element of an existing array. The callback function takes three arguments: the current element, the index of the element, and the array itself. #CodeNewbie

9. filter: This method creates a new array that contains only the elements of an existing array that pass a test implemented by a callback function. The callback function takes 3 arguments: the current element, the index of the element, and the array itself. #technology

The filter method returns a new array that contains only the elements for which the callback function returns true. For example: let arr = [1, 2, 3, 4, 5]; let even = arr.filter((element, index, array) => { return element % 2 === 0; }); // [2, 4] #React

10. reduce: This method reduces an array to a single value by iterating over its elements and applying a callback function. The callback function takes 4 arguments: the accumulated value, the current element, the index of the element, and the array itself. #programmingmemes

The reduce method returns a single value that is the result of applying the callback function to all the elements of the array. For example: let arr = [1, 2, 3, 4, 5]; let sum = arr.reduce((accumulator, element, index, array) => { return accumulator + element; }, 0); // 15

11. find: This method returns the first element in an array that satisfies a given condition implemented by a callback function. The callback function takes 3 arguments: the current element, the index of the element, and the array itself. #javascript30

If no element satisfies the condition, the find method returns undefined. For example: let arr = [1, 2, 3, 4, 5]; let even = arr.find((element, index, array) => { return element % 2 === 0; }); // 2 #JS100

12. findIndex: This method returns the index of the first element in an array that satisfies a given condition implemented by a callback function. The callback function takes three arguments: the current element, the index of the element, and the array itself. #nodejs

If no element satisfies the condition, the findIndex method returns -1. For example: let arr = [1, 2, 3, 4, 5]; let evenIndex = arr.findIndex((element, index, array) => { return element % 2 === 0; }); // 1 #learning

13. includes: This method determines whether an array includes a certain element, and returns true if it does, or false if it does not. The includes method takes 2 arguments: the element to search for, and an optional starting index. #programming

For example: let arr = [1, 2, 3, 4, 5]; console.log(arr.includes(3)); // true console.log(arr.includes(6)); // false console.log(arr.includes(4, 2)); // true console.log(arr.includes(4, 3)); // false #techtwitter

14. flat: This method creates a new array with all the elements of an existing array, and any nested array is recursively flattened. The flat method takes an optional argument that specifies the maximum depth to flatten. #learncode

If no argument is provided, the array is flattened to a depth of 1. For example: let arr = [1, [2, 3], [[4, 5], 6]]; console.log(arr.flat()); // [1, 2, 3, [4, 5], 6] console.log(arr.flat(2)); // [1, 2, 3, 4, 5, 6] #programmers
