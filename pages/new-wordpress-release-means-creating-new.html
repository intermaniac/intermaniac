A new WordPress release means creating a new course! This will be version 3.0 of my Building Block Themes course. Here are my plans. ðŸ§µ

With all the new features, there's too much to cover in one continuous set of lessons. In the current course, students build out the Twenty Twenty-Two theme from scratch. The current course has 65 videos (3-5 minutes), and that's the upper limit for such an approach.

So the new course will start with a set of Quick Start lessons. These will teach the essential skills to build a theme with Full-Site Editing. Then after that there will be sections, each focussing on one aspect of block themes.

In my view this allows new students to have a gradual onramp, while still covering all the features in WordPress 6.1. And students that already went through the course can pick and choose the features they want to learn.

This will also give me the opportunity to showcase more themes. A theme like Wei by @richard_tabor is a good example for how to use the Duotone feature. And Frost by @bgardner has a good implementation of fluid typography and spacing sizes.

"What sites can I build now?" is a common question. Full-on block themes are not always a good fit. But you can mix Full-Site Editing with proven development techniques. So to round things off, I'll be redoing my Building Hybrid Themes course.

So these are my plans! Lots of work ahead. I'm interested in hearing your thoughts on this outline. And of course if there's anything you struggle with when it comes to Full-Site Editing, I want to hear about it!
