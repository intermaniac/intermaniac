Git commands, which I often use. Thread ðŸ§µ

1) git init To initialize a new git repository(repo). 2) git clone To create a copy of an existing git repo. This way you can easily copy the repo, which is in the github repo to git repo in your local machine.

3) git status It shows you all the files, which you have to commit. 4) git add To add a file to the staging area. 5) git add . To add all the files to the staging area in one go.

6) git commit -m "message" To record the file permanently in the version history. You can add a message here means what changes you have done. 7) git branch To check all the local branches, you have made in the current repo.

8) git branch branch1 To create a branch named branch1 in the current repo.

9) git checkout branch1 To switch from one branch to other, say 'branch1'. It firstly, checks whether the branch named 'branch1' exists in the current repo or not. If the 'branch1' exists then, it goes there and if not then, it will create a new branch called 'branch1'.

10) git push origin branch1 To send the commited changes made in the branch branch1 to your remote repo.

11) git branch -D branch1 To delete the branch 'branch1'. But first you need to switch from 'branch1' to 'main' branch, you need to use 'git checkout main' and then, you can delete the branch 'branch1'.

12) git pull To fetch and merge the changes you did in the remote repo to the local machine. And that's it guys, hope you like it and please follow.
