There are many things you should consider when choosing free WordPress plugins. Here are 5 important steps you must follow. Thread ü•Å

1. Choose from WordPress ORG If it comes from the repository, it's safe to use. The plugin directory of WordPress has a tough screening process which does not allow any random plugin to enter and make sure the plugin doesn't contain any virus. Link: https://t.co/laRvlHk7GB

2. Read the reviews Don't just go for the plugin with the most downloads or the highest rating. Read the reviews and see what other users are saying about it. Check WordPress repository reviews. Also, search on Google using keyword like: "Plugin Name" + review

3. Check the compatibility Make sure the plugin is compatible with your version of WordPress. Plugin which is not updated to match the latest versions can raise many issues on your website.

4. Check the support If something goes wrong, you'll want to be able to get help from the plugin author. Make sure they offer support for free plugins. Typically, free plugins have a support forum. However, there are plugin developers that will help you directly.

5. Test site speed In order to ensure that the free plugin you are using on your website is truly lightweight, measure you site's speed and performance before using it fully. Simply install & active the plugin and test your site speed. If the speed is bad, go for another one.
