Ever since I first started developing with WordPress in 2015, I've been looking for the perfect local development environment. My requirements are pretty straightforward but fairly opinionated. ðŸ§µ

It should use some form of Linux server virtualization, giving me the same environment as a live web server, as well as the ability to quickly change anything on the server instance.

It should be uncomplicated to set up and configure, with minimal initial software requirements, easy enough for the beginner, but with scope to be as advanced as the more experienced developer requires it to be.

It should have a low overhead and low system resource usage. It should support multiple PHP versions and run on any of the big three operating systems (Mac/Windows/Linux).

Last, and most importantly, it should be fully open source and use open source software, that's not going to have its license changed whenever the company that owns it decides to.

Today, I am finally one step closer to realizing that dream, thanks to @Canonical's Multipass, @FiloSottile's mkcert, and a bunch of shell scripting. https://t.co/qzB6rXvv81

It currently only supports macOS, but I plan to have Windows and Linux support tested very soon.

I have a bunch of documentation to add to the readme, and then I'd love to have folks on macOS help me test it out.

I'm on a roll, so bugger it, I just updated it with PHP 7.4 support, added a bunch more readme instructions, and tagged the 1.0.1 release. It's ready to test on macOS machines if anyone wants to give it a try... https://t.co/xBnuIjtwlO
