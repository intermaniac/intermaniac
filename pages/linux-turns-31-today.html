1. Developer Friendly Easy to install software such as, python, nodejs, git, visual studio code, vim, and all other useful applications. Linux supports almost all of the major programming languages. Web Development, Blockchain, ML are way more friendly

2. Secure Linux tends to be more secure than other OS like Windows, which is filled with vulnerabilities. Linux is Open Source. The code is reviewed by a huge number of developers constantly. So there are very less chances of security flaws.

3. Package Management While using Linux Desktop, you get the best package managers possible on any Desktop OS. Overall, the process of package management, the concept of repositories, and a couple more features makes it possible for Linux to be more secure than Windows.

4. Free As Linux is Open Source, majority of its distributions are also open-source. You can easily install Ubuntu, Fedora, Manjaro, ShastraOS like distributions absolutely free of cost. Linux also have good verities of software and apps totally free and open-source.

5. You don't need high specs computers to run linux. Linux distros are very lightweight including its software. So it consumes very less resources. Because Linux is blessed by Open Source developers all around the world. So you can now easily revive older computers.

6. Full control over the system This is why I personally use Linux OS. I have to decide when to update. I have to decide what to install. I have full control over all my files. No tech giant like Microsoft is there, to take my data, update when they want.

7. Privacy Google, Microsoft always collect user data. Almost everything. You cant trust them. Even if they claim any kind of privacy and security. Linux distributions do not collect any user data. You have right to see everything that is transmitting from your system.

8. Customization One more reason to Love Linux. I can customize my desktop in any form I want There are lot of Desktop Environments with lots of customization ways. Like Gnome has powerful extensions, KDE has good range of Widgets. You can find tons and tons of Icons and Themes
