These days I find myself using @Docker more and more to package software environments with #Python. You can find good resources on this by @itamarst at https://t.co/zQJvmnjy5b.

This introduction https://t.co/yJaEixkvm2 is designed for first-year #STEM students who have not had calculus yet, but need to do basic linear and nonlinear algebra, basic calculations and visualization. It is always free!

The intermediate booklet https://t.co/h5RdUGDwGC adds topics in calculus like integrals, derivatives and differential equations, with new topics in linear and nonlinear algebra and statistics. These are topics that are common in a second-year STEM curriculum.

The lab book https://t.co/xXwETZkKSM covers reading data files, plotting data, and introductory topics in model fitting and statistical analysis.

Differential equations are so important they get their own book https://t.co/RTdicrTszU. You will learn how to get numerical solutions to ODEs, and how to use these tools for root finding and optimization tasks.

Optimization also gets its own book https://t.co/ZoQvQR78ka. You will learn about minimization of functions, linear programming with constraints, mixed integer programming and global optimization.

Data science and machine learning are taking #STEM by storm, but before that we had regular regression (https://t.co/Pb3cXfWQJq). This book covers linear and nonlinear regression in depth, including uncertainty quantification.

Sometimes you just don't have a good model, and then we resort to data-driven modeling (https://t.co/q211qzQNn5). This book covers interpolation, machine-learning and symbolic regression of data.

Finally, machine learning has brought automatic differentiation into #Python! Learn how to use this powerful tool in https://t.co/r9dh3pk95y. There are applications in uncertainty quantification, sensitivity analysis, constraints, optimization and more!
