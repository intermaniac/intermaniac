How To Add New User In Linux 🐧 🧵👇🏾

There are various ways to add user in Linux but in this thread we'll look at how to do so using the useradd command

To add a user invoke the useradd command as root followed by the username useradd given

To create a home directory for the user add the option m useradd given -m

To immediately create a password for the user add option p followed by the password useradd given -m -p Password123

To immediately add user to a group, sudoers perhaps, add the G option followed by group name useradd -m -p Password123 -G sudo

To change the user password invoke the passwd command followed by the username passwd given
