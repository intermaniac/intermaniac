Tutorial for beginners in Open source for free: Thread-3: Basics: Installation of Git on Windows, How to resolve "git is not recognized as an internal or external command, operable program or batch file" error & setting up environment variables

1. Go to https://t.co/muc52Y35e1 & click on Download for Windows. https://t.co/1YJWEzmPl6

2. Then choose whether your Windows configuration is 32 bit or 64 bit. Download accordingly. 3. After download is complete, install Git. Note: Please make sure that Git should be installed in Program files (on your system). https://t.co/jnPby5dzCu

• To check whether Git has been installed or not: Open command prompt on your system & type “git”. If it runs successfully (shows same as screenshot), then it means Git has been installed. https://t.co/2xQr0gwJnT

• If it shows some error, then please check if you have Git in your Program files (This PC-> Local Disk-> Program Files-> Git) or not. Also, you need to add environment variables. https://t.co/MCCR5SN4WQ

• What are Environment variables ? Environment variables help programs know what directory to install files in, where to store temporary files, and where to find user profile settings.

• You need to set environment variables for Git-> bin & Git-> cmd. • To set up the environment variables, go on search & type “environment variables”, open it up. https://t.co/BOkRUqxKmR

• Click on Environment variables & then go to Path. https://t.co/ZGFO1jLM8L

• Open Path & Click on New/ edit to add the path of Git/cmd & Git/bin from your local machine. Hit OK. • Then, again check if Git is installed or not by running “git” on Command Prompt. https://t.co/KMjefOIWQJ

That's a wrap for this thread. Thanks for reading !!
