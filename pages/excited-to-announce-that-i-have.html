Excited to announce that I have led @Gitpod's $25M Series A, and have joined their board! https://t.co/bc47NHG1x3 I believe that Cloud Development Environments (CDEs) will unlock a new era of productivity and joy for developers on the same scale that GitHub did. ðŸ§µ

Fundamentally, the digital universe has two superpowers: to enhance communication & reduce scarcity. At GitHub we leveraged the former to create a collaboration platform around code. Gitpod leverages the latter to create infinite custom-tailored dev environments in the cloud.

CDEs are like perfectly configured, high powered developer laptops that you can use and discard as easily as sticky notes. One perfect laptop for every project you work on, so you can say goodbye to dependency collision issues between unrelated projects.

Never again stress over updating your operating system, only to spend the next three days figuring out why iconv won't link properly anymore or spend hours tracking down why that one header file can't be found. Less time on StackOverflow means more time creating awesome stuff.

Need to work on that project from five years ago? What do you think the probability is that your dev environment is still compatible? Exactly. CDEs make working on old stuff a breeze.

With CDEs, the code you run, and the libraries you use, run isolated from your local machine, which means enhanced security, and a much safer execution environment for testing out untrusted libraries. Not to mention installing things via bash scripts! ðŸ˜¬

Want to test out a PR that introduces new dependencies? Not too keen on having that permanently change your local dev environment? CDEs let you start fresh every time.

CDEs are especially powerful in complex environments at large companies. At GitHub we built our own complex setup script to provision developer laptops, always aiming to get new employees committing to main on their first day. It still didn't work 100%. CDEs make it easy.

There is still work to be done to make the full dream of CDEs come true, but as the industry pioneer, the Gitpod team is uniquely positioned to execute on this vision. I'll do everything I can to help!
