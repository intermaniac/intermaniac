Excited to launch this side project: @reportAIge üöÄ #AI generated reportage art for various cities based on top #twitter trends using #openai + #dalle, updated daily Built w/ #typescript, #nextjs, hosted @vercel üòôüëå https://t.co/QbY1iNKUuP Read on to learn how I built this üßµüëá https://t.co/A7xXDf4Za6

TL;DR heres the #opensource repo: https://t.co/JGSGQKY4wt (contributions welcome!) 1. We make a stop by the #twitter trends api to get the top trending hashtags for each city along with their respective total volumes

2. The tweet volumes are used for the heatmap intensity on the globe and the hashtags are fed into the #openai text completion generator to create a single sentence story with a specified max character count https://t.co/WETDm3DEXp

3. Once this description has been created, we then let #dalle work it's magic to create a piece of reportage artwork of a randomly selected style. https://t.co/X4dcfUIL1j

4. This artwork is then stored on #google cloud storage and a corrsponding timestamped #mongodb atlas entry is added to the database. #NextJS serves the most recent images to the frontend application for your viewing pleasure üòÅ

5. Every time a new piece of artwork is created, a twitter status update is released @reportaige with the image and description. We also utilize @vercel depoly hooks to redeploy the site and update the artwork https://t.co/VshuGI1mBU

6. Autonomously rinse & repeat on a daily basis at 12:00pm local time for each city through multiple #github action #cron jobs https://t.co/FLiObZCnd2

If you're still here thanks a ton for following alongüôè There's plenty of ideas in the pipeline for future features for the project! And again, this is an #opensource project and contributions are welcome and encouraged! Here's a good place to start: https://t.co/pdE6e3Mbk9
