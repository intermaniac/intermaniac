💡 Overview of sort( ) ❍ The sort method allows us to sort the elements of the array. ❍ It changes the position of the elements in the original array, in other words, the sort method mutates the original array.

❍ By default, it sorts the array items in ascending order, which means the smallest first and largest value will be the last one. ❍ The sort method "typecasts" the elements into strings and compares the strings to determine the order of the elements.

💡 Solution: "compareFunction" ❍ Yes, to resolve this problem we need to pass a "compareFunction" to the sort() method. ❍ The sort( ) method accepts an optional argument which is a "compareFunction" that compares 2 elements of the array.

❍ But, if we omit the "compareFunction" the sort() method sorts the elements on the basis of Unicode values. ❍ The sort( ) method sorts elements on the basis of the returned value of the compare function.

💡 Syntax of compareFunction: ❍ The compare function also accepts 2 arguments, which are 2 continuous elements of the array. ❍ Here, "a" and "b" are the 2 continuous elements of the array. https://t.co/Vw1W5TOcu4
