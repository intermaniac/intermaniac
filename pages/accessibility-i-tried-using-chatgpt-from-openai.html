Accessibility: I tried using ChatGPT from @OpenAI to build an accessible code for a modal in JavaScript and HTML. Here is my experimentation step by step in a short thread. Don't hesitate to share your feedback or suggestion with me, when you finished reading this thread. ⬇️

I first ask "Can you help me write an accessible JavaScript and HTML code for an accessible modal?" You not the accessible 2 times in the prompt. Also, I didn't guide much the AI as I didn't know what kind of answer to expect. I also wanted to have a simple conversation. https://t.co/OeyA77xFOl

What we can notice on the previous code example, is that in addition of the code, there is an explanation of the code, with some references to it. Interesting. Let's continue.

As the code didn't include any reference to an effort for accessibility, I pointed it out and ask a new iteration with a care for ARIA attributes. The code is kept alike, but the AI introduces changes, but only in the HTML code. https://t.co/EA3XtKKqYo

Still not enough for me, I add a small precision: can you improve the JS with ARIA attributes changes? It does it. And a colleague told me: can you ask the AI to take keyboard navigation into account? https://t.co/EFxZd8Xskj

So I did, I asked the AI: "Can you add support in JavaScript for keyboard navigation?" Interesting thing is that it took care of the ESC support, but also of the fact the user in keyboard is supposed to stay inside the modal while "tabbing". https://t.co/8cSmbycuHo

It was really fun to spot the missing parts in the code and ask the AI to improve it. Fun fact is that it's better that way than to have the direct perfect prompt.

I've tried the prompt "Can you help me write a vanilla JS and HTML code to create an accessible modal including ARIA attributes in HTML and in JS, and supporting keyboard navigation." that groups what I asked step by step, and the result wasn't the same.

The result was better than my first prompt, but it has missing parts compared to the result of the entire conversation. End of the thread. For the moment :D

If you are curious about the result of the direct complex prompt, here it is. https://t.co/wTN9u81XLQ
