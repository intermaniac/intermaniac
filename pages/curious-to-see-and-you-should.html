[2/10] before we start, what is process mining? process mining is a data driven approach that helps us analyse, monitor, and optimise our processes.

[3/10] process mining combines data mining techniques with business process management (bpm) to assist in the study of operational processes using event logs.

[4/10] okey, so why use python for process mining? 🐍 python can be used to create algorithms that help to discover hidden patterns and relationships in data. let's look at how python can be used for process mining a bit closer

[6/10] 2️⃣ optimizations there are many different ways to optimize a process, and python can be used to find and create custom solutions that are tailored to the specific needs of the process.
