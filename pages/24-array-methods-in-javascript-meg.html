7) findindex(): 🔹 returns the the index (position) of the first element that passes a test. returns -1 if no match is found. 🔹 executes a function for each array element. 🔹 does not execute the function for empty elements. 🔹 does not change the original array. https://t.co/ovzsljfnnj

8) from(): 🔹 returns an array from any object with a length property. https://t.co/mlrulf766t

9) includes(): 🔹 returns `true` if an array contains a specified value, otherwise returns `false`. 🔹 this method is case sensitive. https://t.co/g1irybnwxq

10) indexof(): 🔹 returns the first index (position) of a specified value. returns -1 if the value is not found. 🔹 starts at a specified index and searches from left to right. 🔹 negative start values counts from the last element (but still searches from left to right). https://t.co/uzzybhhdmo

11) isarray(): 🔹 returns `true` if an object is an array, otherwise `false`. https://t.co/wyet7jy8ig

12) join(): 🔹 returns an array as a string. 🔹 does not change the original array. https://t.co/cnibvjibmc

13) lastindexof(): 🔹 returns the last index (position) of a specified value. returns -1 if the value is not found. 🔹 starts at a specified index and searches from right to left. 🔹 negative start values counts from the last element (but still searches from right to left). https://t.co/g8qtv4mqpg

14) map(): 🔹 creates a new array from calling a function for every array element. 🔹 calls a function once for each element in an array. 🔹 does not execute the function for empty elements. 🔹 does not change the original array. https://t.co/s7caxfbwuf

15) pop(): 🔹 removes (pops) the last element of an array. 🔹 changes the original array. 🔹 returns the removed element. https://t.co/aalbl0l5yl

16) push(): 🔹 adds new items to the end of an array. 🔹 changes the length of the array. 🔹 returns the new length. https://t.co/9nbaytkqvd

17) reduce(): 🔹 executes a reducer function for each array element. 🔹 returns a single value: the function's accumulated result. 🔹 does not execute the function for empty array elements. 🔹 does not change the original array. https://t.co/5ydwwtnqc2

18) reverse(): 🔹 reverses the order of the elements in an array. 🔹 overwrites the original array. https://t.co/vm5weho8w6

19) shift(): 🔹 removes the first item of an array. 🔹 changes the original array. 🔹 returns the shifted element. https://t.co/w0mj2jnd28

20) slice(): 🔹 returns selected elements in an array, as a new array. 🔹 selects from a given start, up to a (not inclusive) given end. 🔹 does not change the original array. https://t.co/zecqawacia

21) some(): 🔹checks if any array elements pass a test. 🔹 executes the callback function once for each array element. 🔹 returns true (and stops) if the function returns true for one of the array elements. otherwise returns false. https://t.co/fi3orwzwyx

22) sort(): 🔹 sorts the elements of an array. 🔹 overwrites the original array. 🔹 sorts the elements as strings in alphabetical and ascending order. https://t.co/1ymobzmkng

23) splice(): 🔹 adds and/or removes array elements. 🔹 overwrites the original array. https://t.co/0xgyldts3e

24) unshift(): 🔹 adds new elements to the beginning of an array. 🔹 overwrites the original array. https://t.co/g14kcfryr4
