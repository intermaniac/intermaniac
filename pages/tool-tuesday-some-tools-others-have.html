Next: Microsoft PowerBi. I spent the longest time trying to build out a trading performance dashboard in Excel only to realize it made more sense in PowerBi. There are many custom visuals available for interactive charts. https://t.co/8lyNjE6pHq /5

Why build a dashboard in PowerBi over Excel? - juicy charting - flexibilty with data/streaming - easy to publish or share dashboard - automations including Zappier integration - great for applying filters and isolating data - highly visual by default - connect to custom app /6
