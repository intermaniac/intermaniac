8 Solidity Projects on GitHub. A thread ↓ https://t.co/0pTvLEk3A8

Solidity (!) Yes, the Solidity language itself it's an Open Source project. Solidity is a statically typed, contract-oriented, high-level language for implementing smart contracts on the Ethereum platform. https://t.co/bRZcF6XgOk

OpenZeppelin Contracts An interesting library for secure smart contract development. Build on a solid foundation of community-vetted code. https://t.co/EmhFHV5z5O

Awesome Solidity Gas-Optimization A resource for gas Optimization. It contains many links and some examples. https://t.co/jaLVHANoDM

Smartcontractkit The Repository to support the "Solidity Blockchain Smart Contract. Beginner to expert" video on FreeCodeCamp. By Patrick Collins https://t.co/w7995xFZoG

Awesome Solidity This is a curated list of Solidity Resources. I suggest checking it out because it contains many good ones that maybe you missed. https://t.co/K9mNZgx0Wq

Blockchain Solidity Smart Contracts. Beginner to Expert (JavaScript Edition) Patrick Collins Strikes again, with a 32 hours video on FreeCodeCamp and this repository as support. https://t.co/HeHFDILFhc

Solidity Patterns A compilation of patterns and best practices for the smart contract programming language Solidity. https://t.co/CyYor6W9K2

Ethernaut The famous Web3/Solidity based wargame by OpenZeppelin. https://t.co/Yb8Nhzz3e1
