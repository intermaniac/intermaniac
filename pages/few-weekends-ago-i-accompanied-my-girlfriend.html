A few weekends ago, I accompanied my girlfriend to do KiteSurfing. We got lost from each other, and she lost her phone. Everything would've been easier if we had known each other's phone numbers. Here's how I built a python solution using #ChatGPT without writing code ðŸ§µ https://t.co/Pz9dq32m9a

I began by trying out different prompts. I think that we'll start to see a lot of prompt engineering for software programs. Just as https://t.co/eRbbsF3b27 from @sharifshameem or PAI from @Suhail, but for snippets of code and scripts https://t.co/pU7GqbOjYX

After many iterations (maybe 2 hours of trial and error), I got a fully working script precisely as I thought it should. Again, my restriction was that I couldn't type any part of the code. https://t.co/2nzkrtXcvb

Another restriction I put myself in was that I couldn't use Google. I now understand why some people are claiming that these LLMs could potentially transform how we do searches. I learned how to find my Twilio keys, how spaced repetition worked, and how to deploy in @Replit

The future of software development is an intersection between NoCode and AI. I call it #NoCodeAI. Seeing NoCode experts like @bentossell going deep into AI reassures me of this idea.
