1/10 üßµ I made a Python script to calculate how many of the first N transactions on Etherscan are indeed Potatoz FCFS tx for the @memeland mint.

2/10 Please take my results with grain of salt. I haven't slept well last night. My results may contain bugs and may be wrong. I take no responsibility for them. https://t.co/c1yIuSXC3f

3/10 I make the following assumptions: - As the honorable gentleman @9gagceo said, the transactions confirmed before mint time are internal team tx. So they should be discarded. These are the first 162 lines from the excel with tx you download from Etherscan.

4/10 - any address with zero potatoz and 1 captain mint is NOT a Potatoz FCFS, because it is either: * a memelist * a mint address registered on behalf of 3 potatoz minting only 1 guaranteed mint * a waitlist So it should be discarded and not counted as Potatoz FCFS.

5/10 - any address with >=3 potatoz minting one Captain is NOT a Potatoz FCFS, because: It has the right to mint at least one guaranteed and is minting only one, thus, this one guaranteed and not a single FCFS. So it should be discarded and not counted as Potatoz FCFS.

6/10 - Any address minting: (the number of potatoz it holds) / 3 is NOT a Potatoz FCFS, because: * It is minting exactly and only the guaranteed mints it has and nothing else. So it should be discarded and not counted as Potatoz FCFS.

7/10 What is NOT taken into account by my script: * memelist AND 3+ potatoz minting 2 spots (could be still 2 guaranteed, same applies to memelist + all triples combinations) * anything else I don't realize now as I haven't slept well last night.

8/10 Again, kindly feel free to criticize my logic assumptions in the comment section below. I might be wrong. But assuming that I am not, and assuming there are no bugs and the script results are correct, here are some numbers:

9/10 If you download the oldest 5000 tx from etherscan in an excel sheet from the mint contract: * By line 2177 there are 1500 minted Potatoz FCFS spots. * By line 2670 there are 1800 minted Potatoz FCFS spots. * By line 3036 there are 2000 minted Potatoz FCFS spots.

10/10 Thank you very much for taking the time to look into my humble work üôè https://t.co/f1LHWl5cA9

UPDATE: One honorable community member spotted a big flaw in the assumptions. It's unknown whether a wallet with 0 potatoz is assigned on behalf of single potato or not. I assumed it's only for triple potatoz. So the results are far from that optimistic.
