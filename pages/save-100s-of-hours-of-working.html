Save 100s of hours of working with big datasets with this simple Python package and your pc will love you for do it. Read this thread and start using Polars

Keep in mind that Polars is not a replacement for Pandas. It's more focused on performance and you must understand the syntax to get the most out of it and its power. How to get started with polars?

Selecting data Polars doesn't have index, so Pandas' .loc and iloc functions do not exist. To select a column in Polars you have to use .select, and if you want to select rows based on a criteria, use .filter. https://t.co/0CglL85uDv

Be lazy An exciting feature of the polars is that you can make lazy evaluation. This means that the polars do not read the entire database and only look for the column we are going to use https://t.co/YHOmuiX4SP

Column assignment To create new columns with polars you must use the function '.with_column' and assign it the name on '.alias'. https://t.co/emRN7e70wU

Column assignment based on predicate Suppose our database has 3 columns, and we want to reassign values according to a criteria. In polars a mask can be created on the original base and the original data will not be modified. Saving a lot of memory. https://t.co/lINGC9qIPz

Filtering In polars the filtering signature is very similar to pandas. Using boolean expressions, but instead of using the 'loc' method we use 'filter' like the following code. https://t.co/qvR6Kysyiq
