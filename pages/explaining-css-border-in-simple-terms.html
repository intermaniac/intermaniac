Css border is a shorthand. Before that let me show you the individual border properties. Following are the border properties. - Border width - Border color - Border Style - Border radius - Border sides - Border shorthand Let me take you through each of them in detail :

CSS border width : Helps us set the thickness of the border

CSS border color : Helps us color the border.

CSS border-style : This is used to give the following styles to the borders: - solid - double - groove - dashed - dotted - ridge - inset - outset - none - hidden There is also a shorthand for styles. You can use 4 different styles on 4 sides. Let me show you... https://t.co/FX872FF9ut

CSS border radius : Helps us curve the border of the box on which the border property has been applied.

CSS border sides : The best thing about the border is that we can set different properties on each side, unlike the CSS outline property.
