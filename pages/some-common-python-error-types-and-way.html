SyntaxError: This Error is raised by the parser when a syntax error is encountered. - leaving out a keyword - putting a keyword in the wrong place - leaving out a symbol, such as a colon, comma or - - - - brackets - misspelling a keyword - incorrect indentation - empty block https://t.co/Yx7LQ0bm8S

To minimize syntax errors ensure you get conversant with Python syntax and usage. Using good IDEs could also come in really handy.

TypeError: This occurs when the data type of an object in an operation is inappropriate. This can happen when an operation is performed on an object of an incorrect type, or it is not supported for the object.

- Performing an operation between two incompatible data types e.g. adding a string and an integer. - Incorrect list index type e.g. using a string as a list index value instead of an integer. - Iterating on a non-iterative value e.g. trying to iterate on an integer. https://t.co/QFJuwdrQ7K

To minimize: Do try to verify the types of objects. You can do that by hovering the cursor on the object and it shows a little documentation on it depending on your IDE. You can also use the - print(type(the_object))
