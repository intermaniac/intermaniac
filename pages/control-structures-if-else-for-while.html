When to use each of the control structures that we looked at: if statements: Use 'if ' statements when you want to execute a block of code only if a certain condition is met.

if...else statements: Use if...else statements when you want to execute one block of code if a certain condition is met, and a different block of code if the condition is not met.

while loops: Use while loops when you want to repeat a block of code an unknown number of times, as long as a certain condition is met.

for loops: Use for loops when you want to repeat a block of code a specific number of times, or when you want to iterate over the elements of an array or object.

other control structures that I didn't mention above. for...of loops: Use for...of loops when you want to iterate over the elements of an iterable object (such as an array or a string) and perform an action on each element.

for...in loops: Use for...in loops when you want to iterate over the properties of an object and perform an action on each property.
