The cascade algorithm determines how CSS properties are combined and prioritized via properties defined by the browser or external stylesheets (.css files)

There are 3 levels of specificity: 1. Element Selectors (like our h1 example) 2. Class Selectors (.my-heading) 3. ID Selectors (unique element) If 2 stylings conflict, the most recent one is chosen read from top to bottom https://t.co/BONfe0gNcg
