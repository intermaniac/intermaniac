The useState hook is a function that accepts a single argument, which is the initial state for the component. In the example above, the initial state is 0.

useState returns 2 values: an array with 2 elements, and we can use destructuring to assign each values. The first one is the state and the second is a function updating the state. Convention is choosing a name for the state variable, and add "set" in front of that function name.

State can be pretty much thought of data that changes with time. In our example, when the button is clicked, the add function is called at a time where the seconds are updated by calling the setSeconds function ‚è≤Ô∏è

Calling setSeconds tells React to re-render the component. In that moment, the entire Timer is re-loaded, meaning when useState is called, the value we get is the value that we called setSeconds with. This goes on until the user quits, or Timer is removed/unmounted from the app.
