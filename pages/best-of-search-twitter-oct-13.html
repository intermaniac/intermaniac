BEST of SEARCH Twitter (Oct 13–Nov 13): Thread! - Github Code Search! - Metaphor: Generative LLM Search - eBook Search w/ OpenAI embeddings - Rewind: Search through your screen history - "Google Search is a dying mall" - Academia + Cybersec search master list …and more: 1/10

Github Code Search public beta launch! New and improved Github Code Search, written in Rust, emerged from the preview 11mos ago. @sourcegraph has competition! Reminds me of cs/ https://t.co/gIAkodDmpC Engineer: @timothyclem Free and open for all: https://t.co/tedhTH8a8A 2/10 https://t.co/8XdfiOY94X

Metaphor: Generative LLM Search Most search systems use a combination of information retrieval and vector search. The novel approach is predictive search which uses LLMs to generate links based on a prompt. The website has cherrypicked examples. Try it on Discord. 3/10 https://t.co/KJeGaC3aVt

eBook Search w/ OpenAI embeddings Very useful vector (or semantic) search demo. Colab get embeddings on paragraphs and on query of a book and does a brute force kNN dot product. Model: https://t.co/BmNvXx8sh4 Colab uses SBERT sentence embeddings. You can use OpenAI API. 4/10 https://t.co/M5i7ZpoCUs

Rewind: Search through your screen history One of the most WOW search demos I've seen! Technically combines low-latency high volume data processing, vision, compression to be able to search through anything on your screen. @RewindAI @dsiroker Sign up for early access. 5/10 https://t.co/BiGUjAv01q

"Google Search is a dying mall" The rhetoric of loss of trust with Google Search goes mainstream. Using "reddit" in queries, ads, SEO spam, bad for tail queries. Some notable precursor was @mwseibel's tweet https://t.co/qpitShRBkP and @dkb868's post https://t.co/pdcIyohEzQ 6/10 https://t.co/3aR3rsCWoc

Academic literature search compilation (@DrasmaJabeen1) One of the biggest issues of domain-specific vertical search is discovering and remembering what they are. This is a fantastic compilation of resources to search through academic literature. 7/10 https://t.co/sfM4ak1bCj

Cybersecurity Search Engines for Researchers (@danielmakelley) A list of 30 search engines to scan public records and feeds for information leaks, email addresses, credentials, etc. 8/10 https://t.co/gfYsOo9k3i https://t.co/8j2GsTMo1e

Bonus: Elon saying search at Twitter will be a focus. Social media / content companies typically end up focussing on the freshest content for views and ends up wasting a goldmine of data that's valuable to the world. Let's see if Twitter can leverage this. 9/10 https://t.co/bNDpV4pKc3

I'm trying something new with this Search rewind! I nerd out about Search anyway having worked on it at @facebook, @google and @glean. It is HOT right now. I wanted to share my learnings with everyone! If you find this useful, RT the 1st tweet and I'll do it more often! 10/10

11/10 Link to the top in case you want to support! https://t.co/sY3CArW0IK
