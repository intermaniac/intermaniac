I started my career in Data Science back in 2016 ⏳ Here's a List of Python/ Machine Learning libraries I would urge you to master!! 🚀 Links to official docs & YouTube Tutorials in the thread 🧵👇

1️⃣ NumPy The most popular Python library for scientific computing. Docs: 👇 https://t.co/ZQGjuqJY65 Video Tutorial: 👇 https://t.co/1RKmepGLCg

2️⃣ Pandas: Go to Python library for data manipulation and analysis. Docs: 👇 https://t.co/kNP3XnEmNL Video Tutorial: 👇 https://t.co/dw0aPHMCxe

3️⃣ Scikit-Learn: Python library for machine learning. Provides tools and algorithms for working with data and a wide range of ML tasks. Docs:👇 https://t.co/8WIOC2ObNI Video Tutorial: 👇 https://t.co/5bJyaWk01l

4️⃣ PyTorch: The most loved Deep Learning frame work. A must have tool in your Arsenal!! Docs: 👇 https://t.co/4Yais7Z6zB Video Tutorial:👇 https://t.co/CBJMWqdtXY

5️⃣ TensorFlow: The most popular Deep Learning frame work! Yo can master one of PyTorch or TensorFlow but must have a read-write capability for both. ✅ Tf docs are the best place to learn. Docs: 👇 https://t.co/J12n7P5FvP

6️⃣ PyTorch Geometric For those interested in ML for Graphs. - Geometric Deep Learning - Graph Neural Nets Docs: 👇 https://t.co/BcRlfrALf5 Video Tutorial (by Stanford):👇 https://t.co/C0Sbc50KOd

7️⃣ Matplotlib The best tool for Data Visualisation!! 📊 Docs: 👇 https://t.co/ZFB6WCImeW Video Tutorial: 👇 https://t.co/IH0rBqcHuy

8️⃣ OpenCV Working in computer vision❓ You must master this library to stand out!! Docs: 👇 https://t.co/PcnR6v35TW Video Tutorial: 👇 https://t.co/sFLk2J0sUw

9️⃣ HuggingFace 🤗 97,436 Models 🔥 15,690 Datasets 🔥 Everything open-sourced 🙌 If not already using go & start using HF 🤗 Docs: 👇 https://t.co/hpNTiVZunB Video Tutorial:👇 https://t.co/Bg46ddKyyg

🔟 JAX Not just another PyTorch/TensorFlow❗️ A library that offers unique capabilities for high-performance machine learning. I've recently started learning this & loving it as I progress!! Docs: 👇 https://t.co/qXLkPuL6rP Video Tutorial: 👇 https://t.co/cipI3refKe
