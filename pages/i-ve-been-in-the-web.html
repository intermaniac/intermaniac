📌 I've been in the web dev space for over 2 years and these are 10 Github repos I think every Javascript developer should check out: 🧵 A Thread 👇🏽 https://t.co/G4DeNlJ7OU

1. WTFJS: Despite the myriad of differences, the one thing on which almost the entire JS community agrees is that it has various tricky nuances. If you've ever wondered about why [] = ![] evaluates to true, this is going to be a goldmine for you. 🔗 https://t.co/ibAAokqavy https://t.co/U21z5rYMQV

2. Awesome Javascript: If you're anything like me, you love checking stuff out built on Javascript. This repo then, my friend will turn you into a 'kid in a candy shop!' Just take a look and see the awesome, shiny things built with JS . 🔗 https://t.co/wdGZjqbZ4G https://t.co/EU3O32wzYq

3. Modern Javascript cheatsheet: You love cheatsheets, right? Who doesn't? They're amazing at providing 'at a glance' info and this repo is full to the brim with JS cheatsheet for just about every concept you can think of. Check it out here: 🔗 https://t.co/2dvDtgnSGp https://t.co/a08RLJhiu5

4. Must watch JS: This is a simple list of talks at various conferences, one must watch. What is not simple (and totally awesome) is the content quality. Just go check it out and you'll know what I'm talking about: 🔗 https://t.co/ZOGvrHoG5l https://t.co/YOnlmGQG1l

5. Javascript from Scratch: If you wanna build your Javascript stack from scratch, look no further. It does require some programming experience but provided you fulfil that criteria, this is well worth checking out. 🔗 https://t.co/M995Za7xRO https://t.co/GD4UlQ0aUz

6. You don't know JS: This is worth checking out for beginners and pros alike. Specially if you feel you know JS, trust me you'll feel different after going through this. 🔗 https://t.co/Z5op0oVrnM https://t.co/B57i8kNERM

7. Javascript Questions: Self-explanatory! This repo contains a lot of questions from the very basics to advanced ones. Use it to refresh your knowledge or prepare for a coding interview: 🔗 https://t.co/pOzb1Zj6vF https://t.co/GCjCH8rbKF

8. JS the right way: If you wanna know all the best practices and upskill, this is simply one of the best things you'll lay your eyes on. Take a look and I promise you'll find something to play with. 🔗 https://t.co/r2jxd91pLL https://t.co/nXtAyT8sSu

9. Airbnb Javascript Style guide: Simply the best Javascript style guide I've found. It has over 129k stars and is well worth checking out: 🔗 https://t.co/ztAP1qJj6v https://t.co/29UYelqikw

10. Javascript Algorithms: This repo contains a lot of examples of data structures and algorithms with excellent explanations for each one. 🔗 https://t.co/zpLoWTPGol https://t.co/cV4XRXLZMI

BONUS: Tech interview handbook As the name suggests, this repo is super handy for preparing for an interview. It covers all phases of tech interviews and is quite information dense. 🔗 https://t.co/UXLMQCVkWC https://t.co/MG6DLS3R0O
