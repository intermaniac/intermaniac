📌 Top 14+ JavaScript Animation Libraries. Thread 🧵

1. GreenSock JS ➖ Professional-grade JavaScript animation for the modern web. 🔗 https://t.co/KofuGf4CY1

2. Velocity.js➖ You can use Velocity.js to scroll browser windows. 🔗 https://t.co/1xVjY0zLN8

3. Popmotion➖ A functional, flexible JavaScript animation library. 🔗 https://t.co/5hd1nD361l

4. Three.js➖ JavaScript library used to create and display animated 3D computer graphics in a web browser using WebGL 🔗 https://t.co/RKyJhF2vbP

5. Anime.js ➖ one of the best animation libraries out there, Help you to add some really cool animation to your project. 🔗 https://t.co/fEJRD6HTkL

6. AniJS ➖ A Library to Raise your Web Design without Coding. 🔗 https://t.co/5CchLO9RJH

7. Mo.js ➖ Motion graphics has a big part to play in animations, and Mo.js is one option with which you can make an impact. 🔗 https://t.co/4gONUSwZQe

8. Vivus.js ➖ Vivus is a lightweight JavaScript class (with no dependencies) that allows you to animate SVGs, giving them the appearance of being drawn. 🔗 https://t.co/kX5JJoltEV

9. ScrollReveal JS➖ If you wish to animate your web elements as they scroll into view, ScrollReveal won’t disappoint. 🔗 https://t.co/piHgfgPvhE

10. Typed.js ➖ Typed.js is a simple library (more of a plugin, really) to animate typing on your screen. Once you enter any string, a visitor can watch it being typed at a set speed. 🔗 https://t.co/fa0HtnTheh

11. Lottie by AirBnB➖ Lottie is a library for Android, iOS, Web, and render them natively on mobile and on the web! 🔗 https://t.co/PC0voTr34A

12. screenfull.js ➖ If you want to have a full-screen feature,use this library 🔗 https://t.co/OgxE48NbFw

13. swiper.js ➖ Fully responsive, swipe-enabled, infinite looping, and more,the carousel you'll ever need. 🔗 https://t.co/9UUDIiH2EA

14. owl.carousel.js ➖ Touch-enabled jQuery plugin that lets you create a beautiful responsive carousel slider. 🔗 https://t.co/SVZUEFvDX7

15. popper.js ➖ Js library to place the tooltip. 🔗 https://t.co/sKTobWcZyc
