Proper management of accounts in Linux enables the system admin to monitor or control users on the system In this thread, I will walk you through how to create and manage basic users and groups in Linux https://t.co/WCCGMErRDt

Creating users on Linux To create a user, we use the adduser or useradd command. For this example, we will use the adduser command. https://t.co/qZvqHbEJXl

To check if the user was added successfully, we use the cat command to read the /etc/passwd file At the bottom of the file, you can see the newbie user was added successfully. https://t.co/RnQJVp35FN

Adding a user to the sudo group Edit the sudoers file to add the new user to the group For our case, we will give the user full permission. https://t.co/DeXs0ekRBE

Delete user To delete a user we use the userdel command https://t.co/9sN6VpwRYB

Creating a group in Linux To create a group, we use sudo addgroup command https://t.co/l8XmfuTmE2

To check if the group was added successfully, we use the cat command to read the /etc/group file You will notice that in the file, we also have the newbie group; it was created automatically after the user was created. https://t.co/m3tS01FheF

Adding group to the sudo group It is the same process as adding the user to the sudoers file. https://t.co/dwYyPrQzJK

Adding a user to a group The benefit of adding users to a group/creating a group is to simplify permissions management. To add a user to a group, we use "sudo usermod -aG name of the group name of the user" https://t.co/hO4LrBcjnb

In the above example, we created a new user and added a password to the user then we added the user to the testgroup Adding a user to a group. The user inherits all the privileges of the group. For example, permission to add a new user, etc. https://t.co/lQV3RCkHF5

Deleting a user from the group To delete a user, we use sudo gpasswd user-name group-name https://t.co/x3Vddo7h8M

Deleting a group To delete a group, we use sudo groupdel group https://t.co/Z0DYWqTQvb

To read more about managing users and groups on Linux, Check out these resources by RedHat https://t.co/cKryY8MfE6 Happy Coding!
