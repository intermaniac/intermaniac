What is JSON? JSON is a lightweight data-interchange format that uses human-readable text to store and transmit data objects.

A bit of a History: It was introduced in the early 2000s as part of JavaScript for communicating between JavaScript clients and back-end servers. Gradually, it expanded to become the most common medium for describing and exchanging text-based data.

Using REST as the convention for APIs and JSON as the medium for exchange proved a potent combination for balancing simplicity, flexibility, and consistency.
