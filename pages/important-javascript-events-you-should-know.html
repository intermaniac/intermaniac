1. The Click Event - onclick The click event happens when a user clicks on a web page's element. It can be handled with an onclick event handler. https://t.co/Bl41XWSNDm

2. The Contextmenu Event - oncontextmenu When a user presses the right mouse button on an element to display a context menu, the contextmenu event is triggered. It can be handled with an oncontextmenu event handler. https://t.co/IiXasRPMOh

3. The Mouseover Event - onmouseover The mouseover event happens when a user moves the mouse pointer over a particular element. It can be handled with an onmouseoever event handler. https://t.co/N5dDZQwmem

4. The Mouseout Event - onmouseout When a user drags the mouse outside of an element, the onmouseout event is triggered. It can be handled with an onmouseout event handler. https://t.co/IRa9aGev8m

5. The Keydown Event - onkeydown When the user presses down a key, the keydown event is triggered. It can be handled with an onkeydown event handler. https://t.co/oZcSO3LhAH

6. The Keyup Event - onkeyup When the user releases a key after pressing it, the keyup event is triggered. It can be handled with an onkeydown event handler. https://t.co/ibfJfm1CU8

7. The Keypress Event - onkeypress Whenever the user presses a key that has a character value assigned to it, the keypress event takes place. For instance, keys like Ctrl, Shift, Alt, Esc, Arrow keys, etc. will produce a keydown and keyup event instead of a keypress event. https://t.co/vQzArjEA1a

8. The Change Event - onchange When a user changes the value of form input, the change event takes place. It can be handled with an onchange event handler. https://t.co/VF3GTKjBw9

9. The Submit Event - onsubmit When the user submits a form on the website, submit event is triggered. It can be handled with an onsubmit event handler. https://t.co/km4bW9morl
