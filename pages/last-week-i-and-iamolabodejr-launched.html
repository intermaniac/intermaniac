Last week, I and @iamolabodejr launched my first-ever client-owned project. It took us 6 months of hard work, and I acted as a full-stack javascript developer. Here is how we did it. A thread. ðŸ§µ #fullstack #expo #MobileAppDevelopment #Nextjs #adonisjs #opensource #framework

1. The Project The project was about building a gift card exchange platform where users exchange their gift cards for cash. After a lot of consideration and project planning, we settled on the following :

Frontend 1. @infinite_red Ignite React Native Boilerplate (Mobile App) With more than 14k stars on GitHub, this is one of the best react native boilerplates. It works well for building both small and enterprise-scale mobile applications using React Native.

It allows you to set things up using @expo or a bare react native project. It also comes packed with a lot of CLI commands that can be used to create application features on the fly. And many more exciting features that make your mobile app development experience better.

2. Next Js with Tailwind CSS I and @iamolabodejr built the web app with this duo. He worked on the landing page while I focused on the admin panel using @EnochNdika Salvia-kit Next Admin Boilerplate.

Backend Development @AdonisJs As a developer with a solid PHP and Laravel background, AdonisJs was an easy tool to pick up. It has been one of my favourite backend frameworks since I started using Javascript for my project's backend service.

It is also very easy to learn with good documentation. Follow @ammezie for interesting tutorials on @AdonisJs.

@expo. Expo is a great tool that easily brings web development deployment experience into the mobile application deployment process. With the new EAS update service that makes it easy to ship in updates, mobile development and deployment is getting even better with Expo.

Yeah Bringing all these tools together is what I will love to talk about some other time, but this is my first #thread and I will be sharing my experience and ideas on how to become a better software developer I also talk about supply chain technology.
