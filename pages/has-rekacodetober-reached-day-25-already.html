Has #REKACodetober reached day 25 already? Wow! For today's post, we have @siti_nanabilah with her continuation of her graph plotting in Python, but more interactive! To see how she does it? Read more in the thread! #REKACodeTober @rekastudios https://t.co/jZxnPYdRAW

"Helloo, from last year's #Codetober participation, I tried out plotting graphs with python. This year I wanted to try out making it interactive and I finally managed to do it! For this project, I used Jupyter Notebook in visual studio code which eases my way of coding a lot!"

"I tried out with Multi-class Weather Dataset from Kaggle and played around with the dataset using Pandas DataFrame. Out of the 12 columns, I only took 7 of them for visualization." https://t.co/qQ3mljgZp7

"For the interactive graph, there is a Python library of HTML interactive widgets for Jupyter notebook called IPyWidgets. Install this and you are ready to interact with the graphs with specific value points set by the user." https://t.co/HMkHXZ7TWY

"I defined a function to manipulate the graph's visualization and another function to pass the values set."
