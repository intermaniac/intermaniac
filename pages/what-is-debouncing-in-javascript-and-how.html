1. Create a function that accepts a function to debounce and the timeout delay as arguments. 2. The debounce function returns a new function, which when executed, creates a timer to execute the original function after a delay and clear the previous timer. But there's more...

ðŸ’­ What if we sometimes need to immediately call action? We don't want it to execute again after the timeout in that case... Solution: flushing. We can attach an extra method to the debounce function that runs the pending action instantly and cancels the timer. Here's how: https://t.co/v9n1L9lKnS

1. Store the arguments of the last action into an array when calling the debounced function. 2. Create a new "debounce.flush" function - it will run the action with the last arguments and clear the timer and stored arguments. 3. Call debounce.flush() to run the action instantly.
