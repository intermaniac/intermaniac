Intro thread to values in JavaScript: In computer science and software programming, a value is the representation of some entity that can be manipulated by a program. Though all values are made of bits, they play different roles.

Every value has a type that determines its role. In JavaScript, there are 5 types of primitive or basic values. Numbers, strings, booleans, undefined and null (These last 2 can be grouped as undefined values).

The other two types of values (that are essentially objects), are objects and arrays. An object or array are collections, a way to group together multiple or one type of data. Today we will focus on numbers.

Values of the number type are, unsurprisingly, numeric values. Let’s say we use the number 13. Use that in a program, and it will cause the bit pattern for the number 13 to come into existence inside the computer’s memory.

JavaScript uses a fixed number of bits, 64 of them, to store a single number value. JavaScript has only one type of number. Numbers can be written with or without decimals.
