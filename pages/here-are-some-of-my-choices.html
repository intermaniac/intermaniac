PyCharm It is a dedicated Python Integrated Development Environment (IDE), providing a wide range of essential tools for Python developers, tightly integrated to create a convenient environment for productive Python, web, & data science development. https://t.co/uesBrBA8Ry

Atom It is a free & open-source IDE, specially designed for Python developers & built using web technologies . It is a source code editor for macOS, Linux, and Microsoft Windows with support for plug-ins & embedded Git Control. It was developed by GitHub. https://t.co/MAvaMmQtEs

Python IDLE It is an Integrated Development & Learning Environment. It is used to create, modify, execute Python scripts & can also execute a single statement. It provides a debugger, & a text editor to create Python script including syntax highlighting, autocompletion etc. https://t.co/8K7pzhdSuj

Thonny It is an IDE for Python that is designed for beginners and support different ways of stepping through the code, step-by-step expression evaluation, detailed visualization of the call stack and a mode for explaining the concepts of references & heap. https://t.co/8Yp4CL2IFg

Eric Python IDE It is a full featured Python editor and IDE, written in Python. It is based on the cross platform Qt UI toolkit, integrating the highly flexible Scintilla editor control. It allows easy extension of the IDE functionality with plug-ins downloadable from the net. https://t.co/yTflH4vGCg

PyDev It is an IDE for Eclipse, which may be used in Python, Jython & IronPython development. It comes with Django integration, code completion, code completion with auto import, type hinting, code analysis, go to definition, refactoring, debugger, remote debugger & many more. https://t.co/TfWV7DMSHP

Spyder It is a free-open source scientific environment written in Python, for Python. It features a unique combination of editing, analysis, debugging, profiling functionality, data exploration, interactive execution, deep inspection, & beautiful visualization. https://t.co/QqPuKryBkn

Sublime text It is a shareware cross-platform source code editor. It supports many programming & markup languages. Users can expand its functionality with plugins, typically community-built and maintained under free-software licenses. https://t.co/tZOREZ6QqK

Jupyter It is an open-source IDE, used to create Jupyter documents that can be created & shared with live codes. It is a web-based interactive computational environment & can support various languages that are popular in data science such as Python, Scala, R, etc. https://t.co/VMLbHx9UMk
