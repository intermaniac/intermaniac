Long Rage Thread, 1/14: I’ve been playing on my familys realms server using Minecraft Java edition using my MacBook for some 3 years, and have had no issues - even the forced migration of account to Microsoft went smoothly. @MojangSupport @MicrosoftNorge

2/14 So now my nephews living on the other side of the country have now come of age and would like to join the family realm. We all think it would be a great way to keep in contact while playing a creative and social game. Should be quite easy, right?

3/14 Since they are minors we where instructed to make a Microsoft Parents account, and through that parent account make childrens accounts. Should be quite easy, right?

4/14: We have now come to a point where both parent and childrens accounts are set up, Xbox- profiles edited with the required parental settings for joining clubs and play on servers. All that’s left is downloading and installing the game, should be quite easy, right?

5/14: Using their parents MacBook, logged into the child account we followed instructions and payed for the new bundle of bedrock and java from the Microsoft store, but after paying we come to a download screen where we need Windows or the Xbox app to actually download the game…

6/14: So we navigate a jungle of FAQs and find a fix downloading through https://t.co/5n2N2gcvPF On the Microsoft profile we can see that we own the game, but when opening the launcher while logged on to the same profile we only get acces to the demo.

7/14: In the launcher title screen there is a button for buying the game. We click the button, is redirected to the Microsoft store where it clearly says on the child’s profile that it has bought the game.

8/14: Another round of FAQing gets us nowhere. Next logical step is checking forums and Reddit. It’s reassuring to see we are not the only ones experiencing problems getting the game installed on Mac, but no known fixes.

9/14: So is it a bug? Waited in line for almost an hour on the national Microsoft help line number, ended up in Indian call center where the employee hung up the phone after hearing the problem. @MicrosoftNorge Dette er ikke greit på noen måte!

10/14: OK, not really surprised we didn’t get help from corporate. FAQed around some more and found a post saying to buy the game directly from https://t.co/m14etk8nAi could be a fix. Ok, so applied for refund from the Microsoft store, which was no problem.

11/14: Bought the game again from the store at https://t.co/m14etk8nAi, but get the same issue: On the profile it says we own the game, but on the launcher we only get acces to the demo.

12/14: The online help and FAQs just send us on an eternal loop between the same pages on the Microsoft Account, the Xbox profile and FAQs and download pages of https://t.co/m14etk8nAi. It is kind of Kafkaesque and would have been a funny SNL-skit… but this is IRL^^

13/14: All we want is to play Minecraft on our 3 generation Realms server, but after spending some 12 hours we are still not any closer to achieving that goal… I’m really close to rage quitting right now…

14/14 Like what the FAQ is the fix here? @mojang @MicrosoftNorge #Minecraft #TalesFromTheLoop #CorporateHell #Xbox #Microsoft #JavaEdition #Sabotage #SMH
