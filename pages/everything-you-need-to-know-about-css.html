1. What are Variables in CSS? A Variable is just a name that refers to a value, variables in CSS helps in reducing repetition and inconsistencies in your CSS code.

One of the coolest thing is that you can access and change the value of a CSS Variable programmatically using JavaScript. We'll understand this later in this thread.

Mostly these variables are declared in :root. It is a pseudo-class that identifies the root elements of tree. Once you declared any variable in the :root, you can use that variable anywhere in your CSS.

But It is not mandatory to use :root for variable declaration, you can declare variable inside any element but there will be a change in it's scope.

Well, As I said above, if you declares variables inside :root then you can use that variable anywhere in your CSS. Whereas, if you declares it inside any other element e.g., inside `.container` selector, then the variable will available to all it's children only.
