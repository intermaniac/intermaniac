All you need to know about JavaScript Math Object. A thread ðŸ§µ #javascript #100daysofcoding #jsbasics

This object provides lots of methods to play with numbers. Some of these methods are:

1. PI you can get the value of PI by console.log(Math.PI)

2. Rounding off here you will use Math.round() function like let x = 1.5; console.log(Math.round(x)) if the decimal is below 0.5 it rounds off it to 0 and if the decimal is equal to or greater than 0.5 then it will round off to 1

3. Another rounding off methods 'floor' and 'ceil' let a = Math.PI; // here a = 3.141592653589793 if I use the floor method, console.log(Math.floor(a)) // output = 3 it will round off 'a' into its lowest decimal form like here a = 3;

now if I use ceil, console.log(Math.ceil(a)) // output = 4 // it will round off 'a' into its highest decimal form like here a = 4;

4. Finding Minimum and Maximum we can easily find minimum and maximum using Math.min & Math.max methods of Math object let's see, find minimum console.log(Math.min(1233,3412,3,-324,-23,2)) output: -324

// find maximum console.log(Math.max(1233,3412,3,-324,-23,2)) output: 3412

5. Generating Random values let randomNumber = Math.random() // this method will generate a random number between 0 - 0.9999... console.log(randomNumber) to get the number to a fixed range we can use the round-off methods like,

let randomNumber = Math.random() *100 // multiply with the maximum number. console.log(Math.ceil(randomNumber)) // add Math.ceil method to get exact range.

6. Getting absolute value (It will return always a positive value) let a = -32; console.log(Math.abs(a)) // Output 32

7. Square Root let a =16; console.log(Math.sqrt(a)) // Output 4

8. Power method console.log(Math.pow(2,3)) // Output 8 End of the Thread.
