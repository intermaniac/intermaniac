#100DaysOfCode 1. Old Java Here is why you should learn Go back in the days, Java used to be the new kid in the block, now it is that old dinosaur that can still deliver and won't kick the bucket easily. #thread #SoftwareEngineering

2. JVM Java's main advantages that it can run on any OS, as long as the OS has a JVM or Java Virtual Machine, a fancy name for a runtime environment. Also, it is a fully OOP Language, has an automatic Garbage Collector, and no pointers to deal with.

3. Performance Issues All that immediately stole the hearts of C & C++ engineers, specially that Java came in C like syntax. On the other hand, Java suffered from performance issues, & it needs at least 10 times memory when compared to a C++ app doing exactly the same function.

4. Time to Market These shortcomings where mitigated as bare-metal hardware became relatively cheaper. Java has a faster time-to-market, since the developers base & community becoming larger by the day & frameworks starts to emerge, like Spring, which was a major stepping stone.

5. Containerization When Docker came to life, it changed many concepts & made Java irrelevant, since the whole idea of Containerization is to be resource efficient and scalable, and Java's VM alone needs loads of memory just to start.

6. Enter Go Google saw an opportunity to fill the gap, and take Java's advantages into a new language that is oriented towards Containerization, Go.

7. Go Creators Go was built by three guys: Robert Griesemer: the same guy who built Google's Chrome JavaScript V8 engine which basically runs the JavaScript in the browser. Ken Thompson: the very same guy who created the original Unix OS. Rob Pike: also a member in Unix OS team.

8. Go & #Kubernetes Go is a modern, not fully OOP, blazing fast language. Google was proud enough of Go that Google used it to create Kubernetes, the famous Container Orchestrator.

9. Advantages of Go - Way shorter startup times - Way smaller resources footprint. - Blazing fast perfomance. - Simple and easy to learn syntax. - built with containerization and version control in mind. - Excellent concurrency management. - produces native executable

10. Start Learning Go Being backed by Google, Cloudflare, and even IBM, Go is gaining more and more community acceptance. You need to start learning Go, soon enough it will become the industry de facto. Follow for more on programming #go #golang #java #kubernetes #learning
