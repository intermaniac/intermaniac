Let's push to GitHub directly using GIT commands and VS Code. It's so easy! What is GIT? Git is a version control system. If you don't know what a version control system is then refer to this link: https://t.co/jbyEOf1tGO

To install GIT: https://t.co/QDS3aeOo45 To install VScode: https://t.co/3dwrzJ59gz Also, download the GitHub extension using VS Code. https://t.co/r0u75gQP7D

1) Choose or make the repository and then copy the GitHub repository link. https://t.co/43jhuFLd0j

2) Open the command palette(SHIFT+CMD+P) in VS Code, type clone, and paste the link. https://t.co/ihl7U5Gsal

3) After you have pasted the link, you need to select the repository location and then you can directly open it using VS code. https://t.co/wPEeUgs9L7

4) Let's code something! I have written a simple Hello World JavaScript program now let us push it to GitHub directly from VS Code. Open the terminal (ctrl + `) in VS Code and use the command `git add <file-name>` and hit enter. https://t.co/X0CB5tvaIr

The `git add` command adds a change in the working directory to the staging area. It tells Git that you want to include updates to a particular file in the next commit. You can read more about `git add` from here: https://t.co/jBSDsgMpzX.

5) Now we have staged the files and now let us commit them. We will use the command `git commit -m "<message>"` Within the " " we need to write a commit message and it is a good practice, then hit enter. https://t.co/XrEj6FQehi

The git commit command captures a snapshot of the project's currently staged changes. You can read more about it here: https://t.co/osn1wtJUj5.

6) The end game!! `git push` is the command which will push all the changes to the repository. Just do a `git push` and hit enter. https://t.co/NV5khSk6Sz

Now, you can go to the repository which you cloned and you will notice that the changes are being reflected. I also made a quick youtube video to understand better. Hope it helps: https://t.co/RFjn9Nb3eR
