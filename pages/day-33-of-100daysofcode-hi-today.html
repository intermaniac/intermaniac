1 - Object defintion In JavaScript, objects are used to model real-world things like cars, ducks, etc. They can stock property/value pairs to qualify them. https://t.co/YRk9g7vNJH

2 - Properties access In order to access to the properties of an object, you can use dot notation. You only need to write the object name, a dot, the property name and it returns the value. https://t.co/hre46No6FK

3 - Object methods In an object, a method is property that is a function. It can add more features than a simple object. https://t.co/tkfmIFcXMU

4 - This keyword When you access to a property with dot notation inside an object, you can use the `this` keyword to replace the object name. It is a simple way to avoid silly name errors. https://t.co/rxH5mEFOh1

5 - Constructor function You can build an object by using a function. This function is called a constructor. It needs to use the keyword `this` to define values. Note : Their name are capitalized to distinguinsh them from other functions https://t.co/SjHdzxv18P

6 - new operator You can assign a constructor to a variable to create a new object by putting the `new` operator before the constructor name when assigning. https://t.co/27opuQd4cl

7 - Constructor arguments Simple constructors like above always return the same value, and that's a lot of work to change hundreds of properties. A simple way to avoid this is to use arguments with a constructor. https://t.co/fXEQDXeHKN
