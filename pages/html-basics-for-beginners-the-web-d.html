Html basics for beginners the web development journey starts with html. in this thread, i've explained some of the basics required before you begin coding. let's begin!ðŸ§µ https://t.co/rzil1zkp0n

1. the essentials - html stands for hypertext markup language - it is used to structure a web page note: - it is not a programming language

- html comprises many "elements" to help make content look a certain way. note: - html is built for the organization of content rather than its styling. css on the other hand fulfills our styling requirements.

2. decomposing html elements consider the following example: https://t.co/tryzolzvpz

it has got 3 parts: - an opening tag: <p> - the content: welcome to the html basics..... - a closing tag: </p> tags are keywords that help in formatting and displaying content.

2.1 opening tag - denotes the starting point of an element - begins with an angle bracket: < - have a tag name (such as: p, h1, div, etc) - end with '/' followed by the tag name and >

2.2 content - information that has to be displayed on the web page

2.3 closing tag - denotes the end of an element - begin with '/' followed by the tag name and >

3. more about elements consider: https://t.co/fpeimtsacj

- tags can have one or more name-value pairs - they are called attribute names and attribute values - they are present after the tag with: > a space > attribute name: id, class, etc > equal to: = > attribute value enclosed within quotation marks: "msg"

attributes provide extra information about the element. note: - not all elements have all the 3 parts to them. learn more about them below.

3.1 empty elements consider: https://t.co/jrhja79usn

- some tags do not require content and a closing tag. the <img> is one such tag that only needs a 'src' attribute that specifies the path of an image to embed it on our web page.
