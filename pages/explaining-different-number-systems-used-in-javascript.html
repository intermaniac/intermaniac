ðŸ“„1/5 Decimal Numbers Decimal numbers are the numbers we commonly use in our code, such as 1, 1234, and 4214. Pro Tip: You can use an underscore to make your numbers more readable, such as 1_000_000, 1_2_34_567, etc.

ðŸ“„2/5 Binary Numbers Binary numbers are numbers represented in base 2. They start with the prefix "0b" or "0B". For example, 0b101, 0B1010, and 0b1001. Note: A SyntaxError will be thrown if the digits after the "0b" are not 0 or 1.

ðŸ“„3/5 Octal Numbers Octal numbers are numbers represented in base 8. They start with the prefix "0o" or "0O". For example, 0o123, 0O1234, and 0o12345. Note: A SyntaxError will be thrown if the digits after the "0o" are not between 0 and 7.

ðŸ“„4/5 Hexadecimal Numbers These are numbers represented in base 16. They start with the prefix "0x" or "0X". For example, 0x1ab, 0X1c4d, and 0xAABB. Another widely used notation is using the prefix "#". You can just replace "#" with "0x" to make it a valid hexadecimal number.

ðŸ“„5/5 Exponential Notation Exponential notation is used to represent very large or very small numbers. It is in the format of "number"e"exponent". For example, 1e3, 1.2e-3, and 1.2e+3.
