# Definition of class: A python class is like a **blueprint to create a new object**. The objects having a similar set of attributes and methods go to the same class. e.g. All the new object of a car class shares some similar attributes and methods.

# Reason for creating a class: 1. To accumulate the data of similar attributes and methods together. By creating a new object we add this new instance, or any future instance, to a particular bundle. That bundle is called class.

2. If we have the same set of objects with similar attributes and functionalities, create a class. 3. Class supports inheritance, code reusability, and code-readability. We can reuse our codes or the codes written by other programmers. (Like: Packages, modules and frameworks)

# Class keyword and __init__ method: Everything in Python is a class. Like list, int, str, tuple, functions, sys etc. To create a user-defined object we use the **class** keyword in Python. The special method **__init__** is used to initialize an object of that class.

# Attributes and methods: From the above example of a rectangle class we can deduce: - Attributes: length, width, num_of_sides, num_of_diagonals - Methods: Area, Perimeter, instantiate_square

# Instance of a class An object of a class (constructed or instantiated from the class) is called the instance of that class. Instantiate the Rectangle class: https://t.co/CumJdZaXsX

# Instance variables: The variables associated with a particular instance of a class. Or, the variables created when an object of a class is instantiated. These variables are not shared by the objects. They are specific to a particular object.

We can declare them in the special method of a class named **__init__**. In our example of the rectangle class, every instance of the rectangle class has their own set of lengths and widths. So, length and width are the instance variables (attributes) here.

## Class variables: These are the variables associated with all the instances of a class. They are shared by all the instances. We declare them within a class but outside of any method.

In our example of the rectangle class, the number of sides and the number of diagonals are the same for all the rectangles. We have declared them **outside of the __init__** method.
