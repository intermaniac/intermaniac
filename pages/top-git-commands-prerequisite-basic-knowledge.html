Top git commands Prerequisite : Basic knowledge about git. A thread!! . #git #gitcommands #Developers #DevOps #freecodecamp #100daysofcodewithGFG

git config: sets author name and email address usage: git config --global https://t.co/rLyeuO0DMs "adityarao" git config --global https://t.co/0AMwEtZva4 "adityarao@example.com"

git init: start a new repository usage: git init "path where you want to initialize this".

git clone : obtain a clone of repository from the given url usage: git clone "url".

git add [file] : adds the file to staging area usage: git add "filename"

git add . : adds all the file to staging area usage : git add .

git commit : records the snapshots the file permanently to version history usage: git commit -m "message for the commit"

git commit -a : commits any files you have added with git add command and also commits any files you have changed. usage: git commit -a

git diff : shows the file differences which are not stage area. usage: git diff

git diff --staged : shows the difference between the files in stage area and latest version. usage: git diff --staged

git diff [first branch] [second branch] shows the differences between the two branches mentioned usage : git diff test_branch_1 test_branch_2

git reset [file] : unstages the file, but preserves the file contents. usage: git reset [file]

git reset [commit] : command undoes all the commits after the commit and stores changes locally. usage: git reset "commit-hash"

git reset hard [commit]: discards all the history and goes back to commit usage: git reset --hard "commit"

git stash : command stores the most recent changes usage : git stash

git stash list : list all stashed changes usage : git stash list

git stash pop : restores the most recently stashed files. usage : git stash pop:

git stash drop : discards the top most stashed change set usage : git stash drop

git pull : fetches and merges changes on the remote server to your working directory. usage : git pull [repository link]

git push ["variable"] branch sends the committed changes of branch to remote repository usage: git push origin master

git push --all [variable] pushes all the branches to remote repository usage git push --all origin
