📌  9 Great web development resources to learn & revise CSS 🧵👇 https://t.co/luT88H41VA

1. W3Schools Arguably the largest web development site on the internet. 3 billion pages displayed each year. 🤯 60M visitors each month. Their library for learning CSS is simple, comprehensive, & effective 🌟 Great resource to treat as CSS docs 💡 👉 https://t.co/pnNvY5tVEQ https://t.co/XrLdjZ3i8F

2. CSS Reference W3Schools feels like it’s been around since the dawn of time. CSS Reference is the opposite. It’s the new kid on the block, and it’s fantastic! 🙌 CSS Refernece provides free visual guides to the most common CSS properties 🤩🔥 👉 https://t.co/vlKPlGHvM0 https://t.co/QBNQmkzCNK

3. CSS-TRICKS Been popular and around for a long time. CSS-Tricks is the Disneyland of CSS 🎡 There are endless CSS “tricks”, techniques, guides, articles, and so on here. Whatever you want to know or learn to do with CSS, you’ll find it here 💯✨ 👉 https://t.co/YxFrDcqQ6e https://t.co/xs0Vcko1DV

4. YouTube It’s the world’s largest university. And it’s free! 🤩 There are endless courses and tutorials on CSS here. CSS is very visual, and YT is a visual platform. No doubt, YT is a fantastic web development resource for learning CSS 🙌🌟 👉 https://t.co/CbfgBGWyMP https://t.co/gxFn3YsxLA

5. FreeCodeCamp FCC has over 8,000 tutorials 😲 You can bet that of those 8,000 tutorials, there are at least a few hundred CSS tutorials 💃 FCC is great for testing CSS knowledge by trying to do or create something with a resource to guide you. 👉 https://t.co/P0n2phlwTc https://t.co/Sn7OUxRO63

6. MDN Docs MDN Docs provides in-depth explanations of code principles and exceptional documentation of web technologies 🙌💯 The CSS section is no different. 👉 https://t.co/SELfFzu6yu https://t.co/PW6jdak0FL

📌 The last 3 resources are platforms to put your CSS knowledge to work by building CSS projects . Learning through building is the most effective way to learn software / web development 💡

7. Frontend Mentor Solve real-world HTML, CSS, and JS challenges whilst working to professional designs. Build projects, review code & help each other grow💡 They provide the style guide, static assets & everything else needed to focus on building 🙌 👉 https://t.co/NbPLcA6uXZ https://t.co/NJSz9LBGOV

8. Frontend Practice Take your frontend skills to the next level by recreating real websites from real companies. This platform provides everything you need to get going. They also provide resources on potentially tricky aspects of the build 🙌🌟 👉 https://t.co/Jhb30k7xSe https://t.co/0KmIzXtljM

9. Codewell Codewell is another great platform to work on your HTML and CSS skills by building projects 🏗 Improve your HTML and CSS skills by practicing on real design templates. 🎨 They provide access the Figma designs for free 👏 👉 https://t.co/nFcCXrAhz2 https://t.co/JKz2rkNVV1
