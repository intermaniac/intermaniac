All the one you need to know (at the top of my head) : - <button></button> - <p></p> - <div></div> - <img src="" alt=""> - <a href=""></a>. - <input type="text">

As I explained last time, the <div></div> is for creating a box that you need to put your HTML elements into it. It's actually a must that you understand this if you want to create a website!

The <p></p> is for writing text and being able to focus them with your CSS styling more easily. We will see this on the next lesson. The <button></button> is for, you probably guess it, creating a button to interact on your website.

Now the <img src="" alt="">. This is use for displaying a image. In the src (source) you put the path to get your image. We will need to write the name of the folder if it was into one. Like so : The alt is the description of the image. https://t.co/pqnY4qJTHO

The <a href=""></a>. This is for adding a link. It can be inside a word, or even a picture. You can add any type of links, it just need to be in-between a HTML element for more efficiently. https://t.co/uLBpZpRzqW

The last one <input type="text"> This one is made to add a input field. Remember when it was the application meta? Yeah it was all made with this HTML tag. The type can be different. It can be "text" (like a search bar) or a "checkbox" (like... a checkbox)!
