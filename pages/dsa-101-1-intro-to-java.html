DSA 101 THREAD 1 INTRO TO JAVA I will be making threads on DSA from now on and Java will be the language in which I share code samples.

Java is a popular programming language, created in 1995. It is owned by Oracle, and more than 3 billion devices run Java. It is used for: •Mobile applications (specially Android apps) •Desktop applications •Web applications •Web servers …..

Why Java ? Although there are multiple benefits of coding in Java I believe the best one is garbage collection. Unlike C++, you don’t need to manually deallocate memory in Java. This results in optimised memory usage and less amount of code required.

In a nutshell Java is the middle ground between C++ and Python as it offers both a short compile and runtime and high level syntax.

First Java program : - Every line of code that runs in Java must be inside a class. Any code inside the main() method will be executed. We can use the println() method to print a line of text to the screen. https://t.co/1g1PkSymp0

Declaring (Creating) Variables In Java, there are different types of variables, for example: •String - stores text, such as "Hello". String values are surrounded by double quotes •int - stores integers (whole numbers), without decimals, such as 123 or -123 (1/2) https://t.co/sVc9WqVus0

•float - stores floating point numbers, with decimals, such as 19.99 or -19.99 •char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes •boolean - stores values with two states: true or false

Data types are divided into two groups: •Primitive data types - includes byte, short, int, long, float, double, boolean and char •Non-primitive data types - such as String, Arrays and Classes (you will learn more about these in a later chapter)

Type casting is assigning a value of one primitive data type to another type. Widening Casting - converting smaller to larger byte -> short -> char -> int -> long .. Narrowing Casting - converting larger to smaller double -> float -> long -> int -> char -> short -> byte

The Java Math class has many methods that allows you to perform mathematical tasks on numbers. https://t.co/YD3nUcpAiy
