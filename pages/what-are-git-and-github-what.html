What are Git and GitHub? What's the difference? Let me simplify it for you A thread⬇️

First, let's go back to the scenario when there was no GIT. 🔸 Devs didn't have copies of their code, they used to submit it directly to the central server. 🔸 No way to track the history of changes in the source code.

After GIT was introduced, here's how things changed: 🔸 Everyone on the team has a copy of the code on their local system. 🔸 Everyone can make changes and anyone can track those changes. 🔸 They can review each other's code before making those changes in the source code.

Let's have a look at its actual definition! 🔸 GIT Git is an open-source and distributed version control system (VCS) that helps you keep track of changes in the source code of a project. Basically, GIT is used to manage an individual/team project in the best way possible.

🔸 Features of GIT: 1. Free and open source 2. Tracks history 3. Supports non-linear development: Can have parallel branches. 4. Provides backup 5. Supports collaboration for every size of project.

🔸 GitHub GitHub is a cloud-based hosting service that offers all of the distributed revision control and source code management functionalities of Git as well as adding its own features.

In short, GitHub makes it a lot easier for individuals and teams to use Git for version control and collaboration. Popular GitHub alternatives: 🔸 Bitbucket 🔸 GitBucket 🔸 GitLab

Why GitHub is so important? Apart from managing projects, it has a lot to offer! 🔸 You can contribute to open-sourced projects. 🔸 Your GitHub profile can get you jobs. 🔸 It provides fantastic programs for students such as: GitHub campus expert GitHub Student Developer pack

GIT Vs GitHub 🔸 Git is a VCS that lets you manage & track source code history. 🔸GitHub is a cloud-based hosting service that lets you manage Git repos. 🔸 Git is installed locally, you can manage your source code history using local machines as repos.

🔸 Unlike GIT, GitHub is a for-profit service, although all basic features are free to use. 🔸 GitHub lives in the cloud, Internet access is required. 🔸 Git can be used without GitHub. 🔸GitHub cannot be used without Git.
