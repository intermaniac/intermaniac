We moved our projects from #Java 11 to #Java 17 almost right after the LTS release. Here is a list of the Top 5 features that I have used and enjoyed for more than a year that may convince you to upgrade your projects. üßµüëá

` record Point(int x, int y) {} ` - this all you need to create POJO. No, constructors, getters, setters, etc. Just removed verbosity, which was disliked and criticized by many community members. üìì

Text blocks: `String text = """ Long formated text. """; ` My SQL queries and JSON blocks in tests don't look like graveyards with all those `+.` ‚ûï‚ùå

Switch exceptions: ` boolean isWeekend = switch (day) { case SATURDAY, SUNDAY -> true; default -> throw IllegalStateException("I's not weekend."); }; ` More readable and less error-prone, and you won't forget to write a `break` at the end of the block. üíº

Pattern Matching for instanceof: `if (x instanceOf String s) { int length = s.length(); } ` No more (casting) inside the block. The code is less messy.üöØ

Helpful NullPointerExceptions: ` https://t.co/XO271YO0WS = "Lidia"; ` throws: Cannot assign field "name" because "person.wife" is null java.lang.NPE: Cannot assign field "name" because "person.wife" is null Finally, NPE describes exactly which variable was null.ü•≥
