GNN-QE is a neural-symbolic model. It decomposes multi-hop logical queries into relation projections and logic operations over fuzzy sets (symbolic). We parameterize the relation projections and the logic operations with GNNs and fuzzy logic respectively (neural). 2/11 https://t.co/ALwaUIWc2J

Fuzzy sets are relaxation of sets, with each element having a probability in [0, 1]. Fuzzy sets provide interpretability for each step (cf. embeddings: Q2B, BetaE). Then the problem boils down to how to design relation projections and logic operations over fuzzy sets. 3/11

Relation projection -> GNN: The relation projection is a generalization of knowledge graph completion. So we can borrow GNNs from knowledge graph completion (e.g. NBFNet) to parameterize the relation projection. In fact, *any* relational GNN can be easily plugged here! 4/11

Logic operations -> fuzzy logic: Fuzzy logic is a relaxation of Boolean logic for fuzzy sets. Fuzzy logic satisfies a bunch of logic laws. Here we use product fuzzy logic, where x‚àßy = x*y, x‚à®y = x + y - xy and Ôø¢x = 1-x, while other fuzzy logic systems may also fit. 5/11

Experimentally, GNN-QE outperforms all existing baselines in all query types, particularly on negation queries. 6/11 https://t.co/kggsDDNE6t

We know that multi-hop models always benefit from better single-hop models. So we perform a breakdown of the performance. GNN-QE achieves one of the best generalization from single-hop to multi-hop reasoning, suggesting a good inductive bias for multi-hop reasoning. 7/11 https://t.co/mWbTOo2UhJ

Besides, GNN-QE can accurately predict the number of answers, even without explicit supervision at training time. 8/11 https://t.co/MH8L6j2951

GNN-QE can provide visualization for intermediate variables, which may help us better understand the reasoning process. This feature is also included in the code release! 9/11 https://t.co/6imKJGcxk5
