What are Java Streams? Streams in Java are used to perform chained operations on collections of items. Streams are not used to store data; rather, they surround a data source to make it easier to conduct bulk operations on its elements. https://t.co/LywRchpKRk

What are the operations? General stream operations like 1. Filtration 2. Transformation 3. reduce and many more Stream allow function-style operation on stream. Stream joins multiple operations to be performed on elements together in a simplest form

What can be data sources on which we can create streams? Data sources can be anything which is collection of elements i.e : Arrays, Collections, I/o stream and generator function as well

Stream operation and pipelines? A stream pipeline is consists of stream followed by stream operations. Stream operations. - Filter - Map - FlaMap - Reduce - foreach - collect
