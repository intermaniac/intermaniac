Web Developers spend daily 8-10 hours on Visual Studio Code. You can save 100+ hours yearly by using these VS @code Shortcuts (I bet) ðŸ§µ #webdevelopment #coding #programming #javascript #html #css #vscode

1. Formatting a Code Formatting takes a lot of time, When we format our code becomes more readable and easy to debug. Use this shortcut to format your code: Mac: Shift + Option + f Windows: Shift + Alt + f

2. Toggle Sidebar While editing, sometimes we need to scroll the window to see the right-hand side code. and at the same to we need other files to work on. You can use the below shortcut to toggle the sidebar: Mac: Cmd + b Windows: Ctrl + b

3. Open Editor Integrated Terminal or Command Prompt To run the code we need a terminal or cmd prompt. Use this shortcut to open it: Mac: Cmd + ` Windows: Ctrl + `

4. Move Line Up or Down While editing the code, sometimes we need to move some code up or down to adjust the logic. Use this shortcut to more code line up or down: Mac: Option + Up/Down Windows: Alt + Up/Down

5. Search and open any file in the project While working on a project, we need to switch over multiple files. Use this command to switch the files Mac: Cmd + p Windows: Ctrl + p

6. Edit Multiple Text/Variables at same time using Multi-cursor When we decide to change the value or variable name, We need to replace it at all places. You can do it in a few seconds using VS code multi cursor feature. Use this command Mac: Cmd + d Windows: Ctrl + d

7. Create a new HTML element with Id or Class Attribute While designing a web page, we need to write a lot of HTML code. You can use these shortcuts to create a long code using a few keywords. div.class-name => create element with class-name div#id => create element with id

8. Autosave VS code needs explicit action to save the code. You can enable the auto-save feature by toggling File => Auto save

9. Search across files To find a function or constant or another thing, sometimes we need to search across all the files. Use this shortcut to do it Mac: Shift + Cmd + f Windows: Shift + Ctrl + f

10. Code folding While working on a large files, sometimes we need to fold/unfold some code to make it readable. You can do it in a few seconds using this shortcut: Mac: Cmd K Cmd L Windows: Ctrl K Ctrl L
