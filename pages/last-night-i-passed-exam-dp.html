1. Last night I passed Exam DP-420 to earn the Microsoft Azure Cosmos DB Developer certification. In this thread I'll share my thoughts. https://t.co/LilPCmCOI2 https://t.co/IpMReRwZdk

2. If you read Microsoft's DP-420 objective domain, you'll notice the exam was heavily modified on August 4, 2022. Sadly, no indication of what changed specifically. GRRR - that ticks me off. https://t.co/yHhOMySMoP

3. The changes to DP-420's focus are enormous! What used to be essentially a C# skills exam is now much more balanced. Much, much less code and more focus on concepts and language-agnostic implementation. I applaud this change.

4. Similarly, the AZ-204 Azure Developer exam underwent a dramatic sea change over the past couple years. The exam was originally a C# skills test, but now focuses exclusively on REST API development in Azure.

5. Anyhow, I'd say the Cosmos DB Developer cert isn't named that well. The concepts therein focus more on database administration and architecture, and the core development stuff is more or less a side item.

6. If I had a specific complaint about Exam DP-420, it's (a) there was a bit too much trivial fact recall included; and (b) I saw some embarrassing typographical errors that definitely should have been caught in editing.

7. Overall, the Cosmos DB Developer cert is worth studying for, especially if you're an Azure architect. You need to understand precisely where and how Cosmos DB fits into the Azure product portfolio, and how you can implement the service in your designs.

8. Let me know what questions, comments, curiosities, or complaints you have about Exam DP-420 and the Cosmos DB Developer Specialty cert, and I'll give you candid, NDA-friendly answers. Thanks! /end
