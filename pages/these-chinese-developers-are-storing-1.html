Last week, Socket's automated npm analysis engine flagged a package called "yingwen-lianmeng-erlingyiqilingjiu-erlingereryiling" among a list of those missing license information. On taking a closer look, @MikolaLysenko noticed something interesting about the npm package...

The unpacked npm package is 79 MB, as it contains an .EPUB (eBook) file. The EPUB iteself contains different editions of "The Economist", a website that is banned in China. Some editions contained in the package are as recent as October 2022: https://t.co/6wrwQ23W0C

The package maintainer ('wizardforcel') had published upwards of 2,900 identical packages. As of today, it's 3,764 packages! https://t.co/Im87RTUUqR The content is quite diverse: - Chinese translations of developer docs - Translated eBooks censored in China - Weird content https://t.co/F8FdLwDfpz

Nearly all of these packages credit 'ApacheCN,' an organization that claims to be a "non-profit project document and tutorial translation project". On its website, ApacheCN makes it clear that the group "have no relationship with [Apache Software Foundation]!"

Previously, 'ApacheCN' is also known to send unsolicited (spam) emails that appeared to have originated from domains of Chinese universities. https://t.co/wJkPfsoj7h

While we were scratching our heads at this mystery, we came across another uncanny finding: a 'male-wakeup-wiki' repo maintained by the same group riddled with sexist connotations. Original: https://t.co/wg0ZecTL0j Translation: https://t.co/nphykvXiLp

Although these packages may not be outright malware, they do put software distribution and version control tools to novel use, in ways deviating from the traditional use case of GitHub and npm. Please reach out if you learn anything else about these packages - we're curious!
