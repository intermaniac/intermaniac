Before starting to code, you need to set up your environment. From @github to Visual Studio @code, here are a few steps to get ready to start coding. N.B. this setup applies to macOS machines. Comment "Hi!" to this thread and I'll send you details for Windows machines too.

💻 GitHub account Have you signed up for GitHub? If not, do it right away. https://t.co/dRJ5ywES5U GitHub is a Git repository web hosting service that allows you to share your code with the world. We'll get back to the details of the version control with Git later. ✌️ https://t.co/hifKjyyDt6

💾 Open your Terminal! Open a new terminal, copy-paste the following command, and hit Enter. It will open a window, click on "Install" and wait. https://t.co/4DONEwzpvc

🍺 Homebrew Homebrew is a package manager: it's software used to install other software from the command line. Let's install it and then proceed with some updates. Copy-paste the following commands: https://t.co/Ov8Ve6pDLZ

👨‍💻 Visual Studio Code Let's install Visual Studio @code text editor! Copy (Cmd + C) the command from the picture 1, then paste it into your terminal (Cmd + V). You can also install some useful extensions (picture 2) https://t.co/ab5F8dtbRF

🙈 Oh-my-zsh Let's install now the zsh plugin @ohmyzsh In a terminal execute the following command: https://t.co/OuMHvmAxh6

🗒️ GitHub CLI Execute the following commands (in this order): https://t.co/N8SoJtyKNK

📁 Dotfiles Why would you want your dotfiles on GitHub? • Backup, restore and sync the prefs and settings. They might be the most important files on your machine. • Learn from the community. Discover new tools for your toolbox and new tricks. • Share what you’ve learned.

As your configuration is personal, you need your own repository to store it. If you are just starting out, you can fork out one of these: https://t.co/QDd6iOjKoC Forking means that you will create a new repo in your own GitHub account identical to the original one. https://t.co/eDiZR1FaRT

Time to fork the repo and clone it to your computer. Of course, change "your-inspiration-GitHub-account" by the actual GitHub account you're forking the repo from. https://t.co/t4R71rec3X

📂 Dotfiles installer & Git installer Run the dotfiles installer (picture 1) Run the Git installer (picture 2) https://t.co/LqgCtnY1y3

💎 rbenv Let's install rbenv, software to install and manage ruby environments. https://t.co/I4pEE55ppQ

💎 Ruby Now, you are ready to install the latest ruby version and set it as the default version. Run the first command. It will take a while (5-10 minutes). Then run the following ones, one by one. https://t.co/xvz32cEd7i

Install some gems In the ruby world, we call external libraries gems: they are pieces of ruby code that you can download and execute on your computer. Let's install some! https://t.co/eHLkDLvQym

🪢 Node.js Node.js is a JavaScript runtime to execute JavaScript code in the terminal. Let's install it with nvm, a version manager for Node.js. https://t.co/ZaQgvpU2Kv

🧶Yarn yarn is a package manager to install JavaScript libraries. Let's install it: https://t.co/491YhBS4dH

🗃️SQLite & PostgreSQL In a few weeks, we'll talk about databases and SQL. SQLite is a database engine used to execute SQL queries on single-file databases. Sometimes, SQLite is not enough and we will need a more advanced tool called PostgreSQL. Let's install them: https://t.co/LpmPyjoNRO

Congrats! 👏🥳🍾 You should now be all set up! Thanks for reading this completely. I hope this helped you get ready to start coding! 🔥 Please hit the 'like' button if you enjoyed this thread. You can also hit the 'follow' button if you'd like to follow my learning journey! 🥰
