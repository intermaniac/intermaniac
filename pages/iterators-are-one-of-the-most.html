- What are iterators? These are objects that can be iterated upon. It is an object which will return data, one element at a time.

An iterator in python is an object that is used to iterate over iterable objects like lists, tuples, dicts, and sets. The iterator object is initialized using the iter() method. It uses the next() method for iteration.

1. __iter__ It s called for the initialization of an iterator. This returns an iterator object

2. __next__ The next method returns the next value for the iterable. When we use a for loop to traverse any iterable object, internally it uses the iter() method to get an iterator object which further uses next() method to iterate over.
