Do you know Web APIs? The most popular one is DOM (Document Object Model), which allows us to interact with HTML, and CSS and use JavaScript on documents. But, our browsers can do a lot more. Learn about these 8 interesting Web APIs and supercharge your projects. ðŸ§µ Thread ðŸ§µ

Intersection Observer API One of my favorites. Allows for invoking events/callbacks, when a specific element is visible in the viewport. The most common use case is to run animations on scrolled elements. Support: all major browsers (except IE) https://t.co/OFhAiJzslM

Geolocation API Very common API used for getting the current location of a device. Not a lot of people know this, but you can watch and invoke a callback whenever the device location changes. Support: all major browsers https://t.co/DpIc0UX4nC

Web Storage API Small database inside the browser? Yes, this is possible with this API. Meet localStorage and sessionStorage. They have the same interface, but sessionStorage data lives until you close the browser. Great for storing user preferences. Support: all major browsers https://t.co/b1IM6nEdFc

Clipboard API This API provides support for copy, paste and cut actions and asynchronous write/read to/from the system clipboard. Exposes events related to clipboard actions, so that we can display nice messages to users. Support: all major browsers (except IE)

Battery Status API Not a wide support, but worth to mention. This API allows us to read battery charge status, as well as listen to events such as level charge, discharging or charging. I can think of many cool features for prolonging battery life. Support: Chrome, Edge, Opera https://t.co/JEgBFamQC8

Page Visibility API Cool API, that detects if the application is currently interacted with by the user. We can use this API to pause and play video, depending if a user is in the correct tab or on a different one. Support: all major browsers https://t.co/uLMJyxfir6

Notifications API This API allows web applications to send system notifications, even if the application is idle (or works in the background). Support: all major browsers (except IE) https://t.co/m9BM8YAWaq

Screen WakeLock API Not fully supported, but still a cool API. Allows web applications to stay awake for a prolonged time. Remember to turn it off, because it will drain the battery! Support: Chrome, Edge, Opera https://t.co/1nSciP2uVS

There are more notable APIs: - Web/Service Workers - WebGL - WebAudio - WebRTC - PWA related: Badging, Shortcuts, Web Share, Contact But these are bigger and deserve their own thread - stay tuned!
