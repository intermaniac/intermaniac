There are two type of operations can be performed on streams 1. Intermediate 2. Terminals Intermediate are the operations that does not shut or update the stream, instead create new stream if updation happened

like : if we apply filter it will create new stream with item filtered example : filter(), map(), distinct() Terminal operation are the operations which ends the stream and perform the result like : foreach(),reduce(), Sum() https://t.co/7rd9MkSNsj

There are some operations which are called as short-circuiting operations such as - Skip() - limit() - findfirst() - FindAny() - AnyMatch() - AllMatch() - NoneMatch() These are short-circuiting because these operations will reduce the stream size and process infinite stream https://t.co/iykrxvqxwZ

Stream further divided into 1. Serial streams 2. parallel streams By default all streams are serial unless specified parallel. Difference between serial and parallel stream are, parallel streams work asynchronously in multithreaded model and have thread safety issues.
