How to use R and Python in one file in VS code? Let's make a thread about it #pythonkachilla2 #pythonkachilla #pakdatasci #DataScience #MachineLearning #DataVisualization

First all u need to have some knowledge about Rmarkdown. It is a markdown specifically designed to run different code from code languages. U can run C++ code in one chunk and Python in next code chunk. Code chunk is just line a code line in between markdown text

Secondly we need to learn how to use R in VS code. For that Go to this link and install Radian terminal for VS code. I am assuming u have installed R already on ur system. https://t.co/isHYDfRUwb

For MacOS users install XQuartz as well. u can install it from homebrew or from this link https://t.co/bOq9B7crRO

Go to extensions in VS code and install this extension https://t.co/d7zYMPVJ1U

Go to ur terminal and type 'which radian' and copy the output This can different for windows users. This Command locates the radian path

press Command+shift+P in vs code and write this RTerm:Mac Paste the link given fro radian in the previous tweet under R path for MacOS https://t.co/x8HzNRfZ7u

Make an Rmarkdown file in a folder. Remember u can make use a Random txt editor and edit it's extension to .Rmd or u can use VS code and open it . it will be read by VS code and R file

In the first 3 lines of Rmd file write this --- output: pdf_document --- this will give u a pdf output when u run the file. Before that u have do 2 package installation.

press command+shift+P search for toggle terminal. Click on it and add a new R terminal under terminals by clicking plus sign. Another way around is to open R without any IDE which u have installed and run following commands

install.packages("languageserver") install.packages("httpgd") It will install these 2 packages in R home directory.

Try to run the file by pressing Cmd+shift+K it will render the .Rmd file to a pdf output. if it works it means u have succesfully set the VS code to run Rmarkdown files. May be u will need to install a latex distribution. if u don't have latex u can edit output: html_document

Now comes the part where we can use python and R in 2 different code chunks in 1 Rmarkdown file in VS code

Go to R terminal in VS code and run these command install.packages("reticulate") library(reticulate) wait for the process to end

Now we can go back to our .Rmd file. step 1: write some text step2: write ```{r} library(reticulate) summary(pressure) ``` this is some text ```{python} import pandas as pd ``` Step3: run by pressing Cmd+shift+K

What if we don't the output and just render the code chunks? For that go to R / Radian terminal in VS code click on lines after ```{r} and press shift+enter it will give u a graph for base R. Now write ~ in the terminal . the signs will change from > to >>>

it means now u have shifted to python from R. to go back to R from python write in same terminal exit and that's it now u can use both R and Python in 1 Rmarkdown file without using R studio. Remember when u have >>> it means u can run python commands only
