A list of the most useful #Python libraries you can use for #SEO right now. üêç This updated thread (again) will tell you the main libraries for #DataScience and #NLP that you should consider. Use them in your workflow! üßµ

Scrapy: the absolute peak of scraping. Nothing is better than this, even though the setup may be hard. You can carry out any scraping task with this library. https://t.co/nJWjfCdzcl https://t.co/HQ8bmCj9hb

Matplotlib/Seaborn/Plotly: you need some sort of visualization and these libs are here to help you. You can start with Seaborn which is easier to use. DataViz is an important topic and you should value it. Plotly is my current favorite due to its interactive plots. https://t.co/XLKbJx0fjy

NLTK/spaCy: work with human language to analyze text data and get insights into the nuances of our language. This is necessary to get your hands dirty with text data. The latter can be used to recognize entities and parts of speech. https://t.co/bVJfIn2bfS

Querycat: few functions but good quality thanks to association rule mining and BERT. It's one of my favorite libraries, but the installation may not be immediate. It's useful for visualizing losses in impressions over time. https://t.co/0t8l6gPQsV

Sklearn: A staple for Machine Learning. I don't think you really need it, but it's one of the first libs you will encounter. https://t.co/GBjKlXon5U

Transformers: Pretrained models to handle a wide range of tasks. Essential for NLP! This library is crucial for the most advanced tasks and quite reliable too. I highly suggest you check my other thread: https://t.co/1qNYrm6LmJ

sentence_transformers: Python framework for state-of-the-art sentence, text, and image embeddings. Use it for keyword clustering and other text-related tasks. It's one of my most used libraries right now. https://t.co/44CQUj8WKZ

Trafilatura: download, parse and scrape web pages. If you work with content, look no further. Cleaning the HTML elements of a page is overrated, don't waste your life on it! https://t.co/srx4RtjG95 https://t.co/wpZOUcJD2n

Streamlit/Dash: interactive web applications. Useful for prototyping and communicating. Streamlit is one of the most favorite solutions for the SEO community.

Typer: create apps that you can run from your command line. Extremely powerful for personal uses and for running local scripts. A game-changer for automating your workflow. https://t.co/PzYZ5ub3Cp

networkx: the must-have graph theory library. I recommend you learn it once you have mastered the basics. Graph Theory is of great importance for analysts who want to level up their game. I prefer using Gephi but this library is still powerful. https://t.co/VHGXu3IO5n https://t.co/YGrCzQ6nw8

searchconsole: Use this library to import your data from the GSC API. It's easy to set up and it's one of the most used libraries in my workflow. This is a must have because GSC data are invaluable. https://t.co/I7PjoE0BPe https://t.co/S2HNq89Hvh

BERTopic: one of my most used NLP libraries and for good reasons. I dedicated an entire thread on the topic: https://t.co/ABbuRHbVf8

scattertext: library for finding distinguishing terms in corpora and displaying them in an interactive HTML scatter plot. A short example from the official docs: (https://t.co/QVgZrdc7Fd).

openpyxl: if you have to work with Excel data and create spreadsheets. There are other libraries but I prefer to use this one. It's quite nice and it works well for most of the tasks. https://t.co/gtULBwQIog https://t.co/PvavXPUkg8

Datapane is the peak of reporting. You can create instant data apps from notebooks. How comfortable is that? It integrates with pretty much any plotting library. Recommended by @theDrewDag. https://t.co/rsSepvazq1 https://t.co/pCmFbwUbuO

Start with scraping and data analysis. Then, you can move to NLP libraries and study topics like NER and Clustering. Get the basics right first. You don't need Machine Learning in most cases, pivot tables will get you farther. https://t.co/PGlOV9aSmw

Sticking to the mainstream libraries is necessary to get access to "better" documentation. My suggestion is to try alternatives and always look for new opportunities across the web. Be sure to always do your research, you could find the perfect library for your needs. https://t.co/LphoBh3jdo
