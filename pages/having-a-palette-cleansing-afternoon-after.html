Having a palette cleansing afternoon after some tough dev this week so I'm going to spend an hour or so playing with the new Full Site Editing tool to build up a micro site on a domain I've had for a year. May turn it into a blog post so - ongoing ðŸ§µ #wordpress #webdev

So the goal is to build a fairly simple one-page WordPress site to act as a lead gen. I normally think WordPress is a bit overkill, but I'm a stickler for flat HTML/CSS (I made https://t.co/cp34BBWj7t during #coronapub times when my second screen was always dark)

Right, so jumped into Editor > Beta, which is present if your theme has it activated (THANK CHRIST YOU HAVE TO ACTIVATE IT). I'm using Twenty Twenty-three https://t.co/P8jYIebV33

I'm greeted with this. It's on Siteground, so I've removed a blog post and I _think_ a page. I can't remember. Will see. https://t.co/egaXIu108U

Okay, backing out. This is going to be a one pager so create a lipsum page and using it as the home page Created a one page, set it as the home page and - there's a number of changes. Big changes. Bit confused, but we'll figure it out. https://t.co/qS2GL169gU

Okay. It's because the default page changes from "Home" (the front page of the blog in the template hierarchy https://t.co/nh8XthjUyZ) when you have posts per page to "Page" Not sure why front-page isn't used. Probably a good reason. But I guess a bit surprising. https://t.co/p3fXTNBImC

Anyway clicking on "Templates" at the side produces the template parts. All fairly standard. Like the 404 is included (so often missed!) & the template parts contains header, footer & other elements FWIW comments are a ballache to change so welcome that. https://t.co/ZrnjPB75MG

I'm sure this is all documented somewhere, but I'm going in blind. Anyway. LET'S CREATE THE SITE!

Right let's clean up the main page to begin with. As this is a one pager, I'm going to remove the H1 (that will be the logo), the featured image (don't need) and comments (definitely don't need). Sure enough, you just select the boxes, and press delete, simple enough. https://t.co/VtojPVMFGw

Now let's look at the header, clicking on the header produces the toolbar, allowing you to switch between editors quickly by clicking "Edit". And it is quick. It's a Siteground hosting and it's instantaneous. I've WPBakery/Elementor on this server and it a lot slower. https://t.co/vVxEbrV7q7

Okay, first issue. I want to change colour of my background but there is a bit of a limited palette. From a bit of Googling I found out that there are new things called styles. Alas, nothing is really grabbing me. I can probably play around with the colours but for now pick 1. https://t.co/KyXXJuGbxq

Next issue, and this is a WordPress terminology thing. As the site title is an element that exists as the header, and not sure how to change it to a H1. Then I realised - that paragraph symbol is what I need. Let's change the header so it's different background colour too https://t.co/nBAqY6soZI

Now to centrally align and we have another small issue, clicking the button should have centrally aligned it, but it didn't. WordPress thing, click the row and change item justification https://t.co/yPnFVCx5NW

Let's do the footer in a similar style By selecting the group block within the footer, we can add and edit a few things. Seems relatively simple to do, though removing WordPress' link and sticking one to @DwinRhys was a bit of a faff https://t.co/qYRL6BZAcv

@DwinRhys Okay, this is where we're up to after 50 minutes, give or take tweeting Goals:- 1. Remove white spacing and get the header flush to the top of the screen. 2. Add social media icons. 3. Maybe change the colours a bit Fact it doesn't take the whole page yet isn't a concern. https://t.co/xfe8dab1xS

Okay, after an hour I'm diving into code, as I cannot figure it out. You'll notice the header isn't flush to the top of the screen, there's a sizable white border and a pale blue margin on top of it. It'd be nice to edit that. I'm not sure if that's in the FSE's remit https://t.co/0FSF0jatT6

My general thoughts on this experiment. I could see me start playing around with ideas in the FSE, and certainly for one pagers (which I still need to write content etc for), then it'd be useful to get things on. I'm not sure if I'd quite yet ready to switch clients to it.

I did deliberately go in blind (as an expert user, so familiar with Gutenberg but not with the FSE), and of course tweeting out my thoughts, so there is that. So yeah, be interested to build a few more and get this site done in it, at least.

Anyway, will let you know the site when it goes live :)
