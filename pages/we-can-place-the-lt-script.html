This means that placing the <script> tag in the head tag may delay the rendering of the HTML especially if the script has many lines of code. In our case, we will see <h1>Hello, World</h1> after the script.js is downloaded and executed. Here is a diagrammatic representation: https://t.co/9LHji9kSEt

One more issue possible with placing the script in the head tag is that if you are trying to operate on items that are on the page you will get errors, and/or your code will not appear to work because the script has been executed before the rest of the page has finished loading.
