📢 filter( ) 🍭 The filter() method creates a new array packed with elements that pass a certain condition provided by a callback function. 🍭 It doesn't execute the callback function for empty elements and doesn't change the original array.

🍭 filter() method returns an array that passes the condition. If no elements pass the condition, it returns an empty array. 💡 In short: It is used to filter the elements of the array. 🚩 Syntax: https://t.co/PuWQCCu62a

💡 In the above snippet: 🍭 I've passed a function to the filter() and it takes 2 parameters. 🍭 item: It's a required parameter, that is the value of the current item. 🍭 index: Optional, it is the current index of the array element that is being processed.

Example 1: 🤔 Suppose, if we want to filter all names from the array whose names start with "A" 🔔 Resultant array will be "nameStartsWithA" https://t.co/L7UkK8WKWt
