Why learn JavaScript ? 🤔 A Thread 🧵👇🏻 https://t.co/cl9hdInCyf

- JavaScript is one of the most prevalent programming languages in the world, especially in building and developing websites.

- A large percentage of websites use JavaScript in building them, large companies such as Facebook, Google, Amazon, Netflix and PayPal use JavaScript in building special applications with it.

📌 And as we know that: ☑️ The HTML we use to build the website structure or structure. ☑️ And the CSS we use to make the site look better because it coordinates the HTML elements and takes care of presentation.

☑️ The JavaScript makes our website dynamic, because it allows us to interactivity between the user and the site, and we can update part of the page based on a specific action that the user does on our website.

🤔 Can we use JavaScript in what? - We can use it: 📌 In the Client Side Interface, this is what appears to the user (the user) through the Browser. 📌 In the Server Side through Node.js

📌 In the mobile application through React Native. 📌 In developing electronic games using HTML Game Engine, the most famous of which is phaser.js

📌 In developing IOT APPs by using one of the IOT frameworks or IOT platforms, the most famous of which are IOT.js and Cylon.js

😁 Of course we can think of learning a ready-made library or framework in order to develop your website, But the truth is that you are familiar with JavaScript and be able to solve the problems that you face and be able to move from one framework to another.

📌 You need to learn the basics of JavaScript, which is called Native JavaScript or Vanilla JavaScript.
