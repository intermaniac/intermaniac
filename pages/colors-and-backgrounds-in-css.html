Colors and Backgrounds in CSS . . . A thread 🧵:

🔑 CSS rules are simple key - value pairs with an selector. -> We can write CSS rules to change color and set backgrounds.

🔹The color property : -> Used to set the text color inside an element. Here, the text color will be changed to red. Similarly we can set color for different elements.... https://t.co/YemtNfkj94

♦ Types of Color Values : The most commonly used ones are :- 1⃣ RGB -> Specify color using Red, Green, Blue values Example : rgb(200, 98,70)

2⃣ HEX Code -> Specify color using hex code Example : #ff7180 3⃣ HSL -> Specify the color using HSL(Hue, Saturation,Lightness) values Example : hsl(8, 90%, 63%)

⏩The value of the color or background color is provided as any one of these values

📌Note : We also have an RGBA and HSLA values for color but they are rarely used by beginners. -> 'A' stands for 'Alpha'.

🔺 The background-color property : -> It specifies the background color of a container Eg : Here we can also use other color instead of brown as well... https://t.co/xARCUXnsHt

🔺 The background-image property : -> Used to set an image as the background Eg : The image is by default repeated in x and y directions https://t.co/TjJG5pEfOd

🔺 The background-repeat property : Can be any of : -> repeat - x ==> repeat in horizontal direction -> repeat - y ==> repeat in vertical direction -> no-repeat ==> image not repeat See more possible values at MDN docs

🔺 The background-size property : Can be following : -> cover ==> fits, and no empty space remains -> contain==> fits , and image is fully visible -> auto ==> display in original size

-> {{width}} ==> set width, and the height will be set automatically -> {{width}} {{height}} ==> set width and height

📍Note : Always check the MDN docs to dissect a given CSS property. Remember, Practice will make you perfect💯👍

🔺Background - position property : Sets the starting position of a background image. Eg : https://t.co/platyyxSaL

🔺Background - attachment property : Defines a scrollable/non scrollable character of a background image. Eg : https://t.co/t2sqbZoYmx

🔺Background shorthand : A single property to set multiple background properties -> One of the properties can be missing given, the others are in order Eg : https://t.co/Qp1I8JTvAD
