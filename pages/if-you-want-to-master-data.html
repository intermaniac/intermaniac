If you want to Master Data Science, Here are 10 insanely useful Python libraries to simplify your career. -- Thread --

1 - Numpy : ▪ Numpy handles all the fancy math you learned in university in a very efficient manner, ▪ It is the foundation of all data science and machine learning packages. https://t.co/6cxig9F1fj

2 - Pandas : ▪ “import pandas as pd” is the 1st line of code you will write, ▪ pandas is a data manipulation and analysis software library. https://t.co/ILQ0ZEqnKi

3 - Scikit-Learn : ▪ If ML is your passion, then the Scikit-Learn has got you covered, ▪ where you can find any algorithm you could wish to employ for your predictions. https://t.co/qCMJLdmepR

4 - Scipy : ▪ Scipy provides the fundamental mathematical methods for performing complex machine learning processes, ▪ Like Numpy, it's likely that you won't use Scipy directly, But Scikit-learn heavily relies on it. https://t.co/kilCGs0JZ9

5 - Matplotlib : ▪ Matplolib is the basis of image visualization in Python, ▪ Matplolib is your partner for visualizing your algorithms and examining distributions in your data. https://t.co/QnTXD7tS92

6 - Seaborn : ▪ Seaborn is a library that uses Matplotlib underneath to plot graphs, ▪ It's Mostly like Matplolib on Steroids. https://t.co/rGX8EOWZQJ

7 - TensorFlow : ▪ Tensorflow is an entire end-to-end open-source Machine Learning platform, ▪ several of the top businesses in the world have chosen it for their deep learning requirements. https://t.co/TWk4gujdsy

8 - Pytorch : ▪ TensorFlow’s main competitor in the deep learning space, ▪ Used for applications such as computer vision and NLP. https://t.co/mS5HMQtuJj

9 - KERAS : ▪ It reduced the entry barrier for starting to program neural networks by so much, ▪ It is based on top of TensorFlow. https://t.co/o4mGv4SnYW

10 - NLTK : ▪ Natural Language ToolKit in abbreviation, ▪ It's a leading platform for building Python programs to work with human language data. https://t.co/Lc4sxILwHo

These are the tools I found to be most helpful, ✔ Even though such a list is never complete, it does provide you with a few tools for each use scenario, ✔ If I missed your favorite, please add it in the comments section.
