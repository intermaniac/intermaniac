In the previous thread about the #Ape framework, I talked about how to install the framework and the Chainstack plugin; check it out if you didn't yet! https://t.co/BMUZj8vKU3

The Ape console allows you to interact with a blockchain, create/import accounts, deploy & interact with smart contracts, etc.

Let's start by creating an environment variable with our endpoint URL; we'll use Goerli in this example. Get your RPC endpoint on @ChainstackHQ if you don't have one yet. https://t.co/JwynJ32sNC

Note that the URL in this case is only an example https://t.co/8PdHp0y0AC

To test the Ape console, we can create a new account, fund it with some #Goerli $ETH and then query its balance from the console. To create a new account we can use the command ape accounts generate "ACCOUNT_NAME" https://t.co/N8DNTdkc9w

We'll be asked to input a value for entropy and a passphrase, keep your passphrase handy since you'll need it to sign transactions through the console! https://t.co/oOJY3pVeL1

Once we fund it, we are ready to use our Goerli endpoint to connect to the Ape console and query our account! Connect to the Ape console: https://t.co/9b8nvZHtf3

Then initialize the account that we just created, and query its balance! https://t.co/vHkZtgS88u
