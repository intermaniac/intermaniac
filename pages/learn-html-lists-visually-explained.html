Learn HTML Lists Visually Explained📝 Thread 🧵 https://t.co/ifvleEirxR

HTML Lists HTML lists allow web developers to group a set of related items in lists. 👉 Unordered HTML List - An unordered list starts with the `<ul>` tag. Each list item starts with the `<li>` tag.

- The list items will be marked with bullets (small black circles) by default: Example - https://t.co/nMoN84gTHV

👉Ordered HTML List - An ordered list starts with the `<ol>` tag. Each list item starts with the `<li>` tag. - The list items will be marked with numbers by default: Example - https://t.co/2oYT3Xqg4H

👉 HTML Description Lists - HTML also supports description lists. - A description list is a list of terms, with a description of each term. - The `<dl>` tag defines the description list, the `<dt>` tag defines the term (name), and the `<dd>` tag describes each term: Example https://t.co/Tafb0M716J
