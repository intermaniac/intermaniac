Methods of Strings of Java explained in 1 THREAD üßµüëá -

1. xyz.length(); = Returns length of the string named xyz. 2. xyz.toLowerCase(); = changes case to lower case of all characters in the string. 3. xyz.toUpperCase(); = changes case to upper case of all characters in the string.

4. xyz.trim(); = Returns a string with no begining or ending spaces. ex- if String xyz = " Methods of String " then... xyz.trim(); = Methods of String

5. xyz.subString(int start); = Reurns a cropped string starting from the number input to end. ex- if String xyz = "Aryan Karma" then... xyz.subString(4); = n Karma (characters from 4 to end)

6. xyz.subString(int start, int end); = returns a string starting from int start to int end. ex- if String xyz = "Aryan Karma" then... xyz.subString(2,8); = yan Ka [characters starting from 2 to 8]

7. xyz.replace("ry","xyz") = Returns a new string after replacing "ry" to "xyz". 8. xyz.startsWith("Ary") = returns true if the string xyz starts with "Ary" otherwise returns false. 9. xyz.endsWith("ma") = returns true if the string xyz ends with "ma" otherwise returns false.

10. xyz.charat(int) = returns the character at the given index position. note- the index of a string's first character is 0 , and the index of a string's last character is the length of the string minus 1.

11. xyz.indexOf("Ar")= returns the integer index of the given string in this case it is 0 cause in "Aryan Karma" "Ar" starts from index "0". and if the input string in not in xyz then it will return -1.

12. xyz.indexOf("a",5)= returns the integer index of the input character (a), (starts scanning from left to right after the given integer input) ex- String xyz = "aryan karma"; System.out.println(xyz.indexOf("a",4));

(it will start scanning for a in 'n karma' and it will return the int index of first a which is at '7') 13. xyz.lastIndexOf("a") = returns the int index of the character input by user. ex- String xyz = "aryan karma"; System.out.println(xyz.indexOf("a"));

here it will return -10 Note- it scans string from right to left. 14. xyz.lastIndexOf("a",5) = returns the int index of the last string before index 5. in the above example it'll be 3

15. xyz.equals("Ary")= returns true if string xyz equals the given input otherwise false. in this case its false 16. xyz.equalsIgnoreCase("aRyan karRmA")= returns true if string xyz equals the given input (case doesn't matter) otherwise false. in this case its true.
